### 第三篇：理论与收敛（Lecture 7-12）——"为什么梯度下降有效？有多有效？"


## 五、易混淆概念速查表

| 容易混淆的概念 | 区别要点 |
|---|---|
| 凸函数 vs 严格凸 | 凸：≤（允许等号，如线性函数）；严格凸：<（排除线性） |
| 全局最小值 vs 局部最小值 | 全局：对所有x；局部：对ε邻域内的x |
| 惩罚函数法 vs 障碍函数法 | 惩罚：从外逼近，θ↑，允许不可行；障碍：从内逼近，μ↓，始终可行 |
| 一般凸 vs 光滑凸 | 一般凸：只有一阶刻画下界；光滑凸：额外有二次上界 |
| O(1/√K) vs O(1/K) | 一般凸+有界梯度→1/√K（慢）；光滑凸→1/K（快10倍） |
| 常步长γ=R/(B√K) vs γ=1/L | 前者需要知道K（迭代总数）；后者只需知道L |
| 二分搜索 vs 黄金分割 | 二分：每步2次求值，ε参数；黄金：每步1次，β=φ≈0.618 |
| Taylor定理 vs Taylor推论 | 定理：中间点γx+(1−γ)y的精确等式；推论：在x处展开+o(·)误差 |
