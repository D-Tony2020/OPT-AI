### 第 ① 页：Two Properties of Projection

**原文翻译**：

**性质 1**：对凸集 $\mathbb{X}$，$y \in \mathbb{R}^n$，$x \in \mathbb{X}$：

$$(x - \Pi_\mathbb{X}(y))^\top (y - \Pi_\mathbb{X}(y)) \leq 0$$

> **[图示]** 凸集 $\mathbb{X}$，点 $y$ 在集合外部，$\Pi_\mathbb{X}(y)$ 在集合边界上，$x$ 在集合内部。从 $\Pi_\mathbb{X}(y)$ 分别画向量指向 $x$ 和 $y$，两个向量之间的夹角为钝角（内积 $\leq 0$）。

**性质 2**：对凸集 $\mathbb{X}$，$y \in \mathbb{R}^n$，$x \in \mathbb{X}$：

$$\|x - y\|^2 \geq \|y - \Pi_\mathbb{X}(y)\|^2 + \|x - \Pi_\mathbb{X}(y)\|^2$$

**精讲内容**：

本页是对 Lecture 13 建立的两个关键投影性质的回顾，为接下来的收敛分析提供即插即用的工具。

**性质 1（钝角性质）** 的含义：从投影点 $\Pi_\mathbb{X}(y)$ 分别看向可行域内的任意点 $x$ 和域外的被投影点 $y$，两个方向之间的夹角 $\geq 90°$。这是 Lecture 13 第 ③-⑤ 页用凸组合扰动法严格证明的。

**性质 2（勾股定理型不等式）** 的含义：$x$ 到 $y$ 的距离的平方 $\geq$ $x$ 到投影点的距离平方 + $y$ 到投影点的距离平方。这是 Lecture 13 第 ⑥ 页由性质 1 直接推导的。

这两个性质之间的逻辑关系是：性质 1 是基础（投影的几何本质），性质 2 是推论（性质 1 的代数推论）。在收敛分析中，我们直接使用性质 2——它告诉我们投影操作不会增加到可行域内任意点的距离。

**为什么要重新列出这两个性质**？因为 Lecture 14 是新的一堂课（距 Lecture 13 可能过了几天），学生需要"冷启动"回忆起关键工具。此外，本讲的核心证明（第 ②-⑤ 页）将在关键步骤直接引用性质 2，所以提前列出便于对照。

在整个课程的知识图谱中，这两个性质占据了 **"凸集理论"和"算法收敛分析"之间的桥梁位置**。它们是凸集的纯粹几何/代数性质（不涉及任何目标函数 $f$），但它们为涉及 $f$ 的优化算法收敛分析提供了不可或缺的技术支撑。

**教授口吻讲解**：

好，上节课我们证明了投影的两个关键性质。今天我先把它们写在黑板上，因为接下来十分钟我们就要用到它们。性质 1 是钝角——从投影点看，可行域内的点和被投影的点"方向相反"。性质 2 是勾股——投影只会缩短距离。记住这两条，我们就可以完成收敛定理了。

### 第 ② 页：Projected Gradient Descent

**原文翻译**：

$$y^{k+1} = x^k - \gamma \, \nabla f(x^k)$$

$$x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$$

我们将使用以下预备分析来给出投影梯度下降的收敛结果。

令 $x^* = \arg\min_{x \in \mathbb{X}} f(x)$。每次迭代的误差为 $f(x^k) - f(x^*)$。我们想要上界误差。

由凸性的一阶特征化：

$$f(x^*) \geq f(x^k) + \nabla f(x^k)^\top (x^* - x^k)$$

$$\implies f(x^k) - f(x^*) \leq \nabla f(x^k)^\top (x^k - x^*)$$

所以只需上界 $\nabla f(x^k)^\top (x^k - x^*)$ 即可。

令 $g^k = \nabla f(x^k)$。

**精讲内容**：

这一页重申了收敛分析的起点，与 Lecture 13 第 ⑦ 页本质相同。

**分析框架**。投影梯度下降的收敛分析遵循一个经典的三步框架：

**Step 1（凸性上界）**：利用凸函数的一阶特征化（Lecture 3-4），将函数值误差 $f(x^k) - f(x^*)$ 转化为内积 $\nabla f(x^k)^\top(x^k - x^*)$ 的上界问题。这一步对无约束和约束优化完全相同。

**Step 2（代数展开）**：用极化恒等式将内积转化为距离平方的组合，创造出伸缩求和的结构。这是本页之后第 ③-④ 页的内容。

**Step 3（求和+优化步长）**：对 $k = 0, \ldots, K-1$ 求和，利用伸缩消去简化上界，选择最优步长得到最终收敛速率。这是第 ④-⑤ 页的内容。

**引入简记 $g^k = \nabla f(x^k)$**。这不仅是为了书写方便，也在概念上将"梯度"这个对象抽象化——在后续的随机梯度下降（SGD）分析中，$g^k$ 会被替换为随机梯度 $\tilde{g}^k$，分析框架保持不变。

**凸性一阶特征化的直觉**。$f(x^*) \geq f(x^k) + \nabla f(x^k)^\top(x^* - x^k)$ 是说：凸函数在任何点的切线（或更一般地，切超平面）都在函数图像的下方。特别地，$f$ 在 $x^k$ 处的线性近似给出了 $f(x^*)$ 的一个下界。反过来，$f(x^k) - f(x^*) \leq \nabla f(x^k)^\top(x^k - x^*)$ 是说：当前的"多余函数值"被梯度和"到最优解方向"的对齐程度所控制。

这个上界的关键特征是它只涉及**当前点的信息**（$x^k$ 和 $\nabla f(x^k)$）和最优解 $x^*$——不需要知道 $f$ 在其他任何地方的值。这使得我们可以将分析聚焦于迭代序列的几何轨迹。

**教授口吻讲解**：

这一页是"回到战场"。上节课我们准备好了武器（投影的两个性质），现在要打仗了。目标是证明投影 GD 的收敛速率。第一步：凸性把"函数值误差"变成"内积"。这一步你们已经很熟悉了——Lecture 5 分析无约束 GD 时就是这么开始的。新的东西在接下来两页。

### 第 ③ 页：展开内积的上界

**原文翻译**：

$$\nabla f(x^k)^\top (x^k - x^*) = (g^k)^\top (x^k - x^*) = \frac{1}{\gamma}(x^k - y^{k+1})^\top (x^k - x^*)$$

$$\star = \frac{1}{2\gamma}\left(\|x^k - y^{k+1}\|^2 + \|x^k - x^*\|^2 - \|(x^k - y^{k+1}) - (x^k - x^*)\|^2\right)$$

$$= \frac{1}{2\gamma}\left(\gamma^2 \|g^k\|^2 + \|x^k - x^*\|^2 - \|y^{k+1} - x^*\|^2\right) \quad (\ddagger)$$

$\star$ 由 $\|a - b\|^2 = \|a\|^2 + \|b\|^2 - 2a^\top b$ 得出，因此 $a^\top b = \frac{1}{2}(\|a\|^2 + \|b\|^2 - \|a - b\|^2)$。

由投影的第二个性质：

$$\|y^{k+1} - x^*\|^2 \geq \|x^{k+1} - x^*\|^2 + \|y^{k+1} - x^{k+1}\|^2 \geq \|x^{k+1} - x^*\|^2 \quad (\ddagger\ddagger)$$

> **[图示]** 凸集 $\mathbb{X}$，$y^{k+1}$ 在集合外部，$x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$ 在集合边界上，$x^*$ 在集合内部。展示了 Property 2 的几何关系。

由 $(\ddagger)$ 和 $(\ddagger\ddagger)$：

$$\nabla f(x^k)^\top (x^k - x^*) \leq \frac{\gamma}{2}\|g^k\|^2 + \frac{1}{2\gamma}\left(\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2\right)$$

从 $k = 0$ 到 $k = K-1$ 对这个不等式求和：

**精讲内容**：

这一页是收敛分析的技术核心，完成了从"内积"到"距离组合"的转化，并在关键步骤引入了投影的勾股定理型不等式。

**极化恒等式的应用**。设 $a = x^k - y^{k+1} = \gamma g^k$，$b = x^k - x^*$。由 $a^\top b = \frac{1}{2}(\|a\|^2 + \|b\|^2 - \|a-b\|^2)$：

$$\frac{1}{\gamma}(x^k - y^{k+1})^\top(x^k - x^*) = \frac{1}{2\gamma}(\|x^k - y^{k+1}\|^2 + \|x^k - x^*\|^2 - \|y^{k+1} - x^*\|^2)$$

其中 $\|x^k - y^{k+1}\|^2 = \gamma^2\|g^k\|^2$（因为 $y^{k+1} = x^k - \gamma g^k$）。

**投影性质 2 的关键应用**。$(\ddagger)$ 中出现了 $\|y^{k+1} - x^*\|^2$。注意 $x^* \in \mathbb{X}$（最优解是可行的），所以我们可以应用性质 2，取 $x = x^*$，$y = y^{k+1}$：

$$\|y^{k+1} - x^*\|^2 \geq \|x^{k+1} - x^*\|^2$$

这里 $x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$ 是投影点。这个不等式意味着：$-\|y^{k+1} - x^*\|^2 \leq -\|x^{k+1} - x^*\|^2$。代入 $(\ddagger)$，得到：

$$\nabla f(x^k)^\top(x^k - x^*) \leq \frac{\gamma}{2}\|g^k\|^2 + \frac{1}{2\gamma}(\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2)$$

**这一步的深层含义**。通过投影性质 2，我们成功地将"梯度步后的中间点 $y^{k+1}$"替换为"投影后的迭代点 $x^{k+1}$"。这使得上界中出现了 $\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2$ 这个关于迭代序列 $\{x^k\}$ 的**伸缩（telescoping）项**——它衡量的是"第 $k$ 步到第 $k+1$ 步，离最优解近了多少"。

在无约束 GD 中（Lecture 5-8），$y^{k+1} = x^{k+1}$，所以不需要这个替换。投影 GD 的额外代价是：我们丢弃了 $\|y^{k+1} - x^{k+1}\|^2$ 这一项（它是非负的，丢弃它使不等式变松），换取了伸缩结构。这是一个"松弛换结构"的典型trade-off。

**教授口吻讲解**：

看到 $(\ddagger\ddagger)$ 这一步了吗？这就是投影性质 2 发挥作用的地方。它让我们把 $y^{k+1}$ 换成了 $x^{k+1}$，从而恢复了伸缩求和的结构。你们想一想：如果没有这个性质，$y^{k+1}$ 和 $x^{k+1}$ 是不同的点，求和时相邻项无法消去，整个分析就卡住了。所以说，投影的勾股不等式是投影 GD 收敛分析的"那块拼图的最后一块"。

### 第 ④ 页：求和与伸缩消去

**原文翻译**：

$$\sum_{k=0}^{K-1} \nabla f(x^k)^\top (x^k - x^*) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\left(\|x^0 - x^*\|^2 - \|x^K - x^*\|^2\right)$$

$$\leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$$

因此，我们得到：

$$\sum_{k=0}^{K-1}\left(f(x^k) - f(x^*)\right) \leq \sum_{k=0}^{K-1}\nabla f(x^k)^\top (x^k - x^*) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$$

**精讲内容**：

这一页完成了伸缩求和，是整个收敛证明中最令人满足的一步——大量中间项消去，只留下起点和终点。

**伸缩求和（Telescoping Sum）的详细过程**。将上一页的不等式对 $k = 0, 1, \ldots, K-1$ 求和：

$$\sum_{k=0}^{K-1} \nabla f(x^k)^\top(x^k - x^*) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\sum_{k=0}^{K-1}(\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2)$$

第二项展开后：
$$\sum_{k=0}^{K-1}(\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2) = (\|x^0 - x^*\|^2 - \|x^1 - x^*\|^2) + (\|x^1 - x^*\|^2 - \|x^2 - x^*\|^2) + \cdots + (\|x^{K-1} - x^*\|^2 - \|x^K - x^*\|^2)$$

每个 $\|x^k - x^*\|^2$（$k = 1, \ldots, K-1$）出现一次为正、一次为负，相互抵消。最终只剩下首项和末项：
$$= \|x^0 - x^*\|^2 - \|x^K - x^*\|^2$$

因为 $\|x^K - x^*\|^2 \geq 0$，可以进一步放松：
$$\leq \|x^0 - x^*\|^2$$

**最终不等式的结构**。综合凸性一阶特征化（$f(x^k) - f(x^*) \leq \nabla f(x^k)^\top(x^k - x^*)$）和上面的结果：

$$\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$$

这个不等式的右边有两项：
- **梯度项** $\frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2$：与步长 $\gamma$ 成正比——步长越大，梯度带来的累积误差越多。
- **初始距离项** $\frac{1}{2\gamma}\|x^0 - x^*\|^2$：与 $1/\gamma$ 成正比——步长越小，从初始点"走到"最优解附近需要更多步。

这两项之间存在 **trade-off**：$\gamma$ 大则第一项大、第二项小；$\gamma$ 小则反之。选择最优 $\gamma$ 来平衡这两项——这正是下一页要做的。

**与 Lecture 5-8 的对比**。在无约束 GD（假设 Lipschitz 梯度 $L$）中，我们有类似的不等式但用的是 Lipschitz 条件而不是有界梯度假设。两种情况下的伸缩求和结构是相同的——只是上界的具体形式不同。

**教授口吻讲解**：

伸缩求和——这是数学里最美的技巧之一，对吧？几十项一加，中间全消了，只剩首尾。你们高中学级数的时候就见过这个。现在我们在优化理论里用它，效果一样漂亮。右边留下两项：一项跟步长成正比（步子太大会"晃"），一项跟步长成反比（步子太小走太慢）。下一页我们来找那个"刚刚好"的步长。

### 第 ⑤ 页：Convergence of Projected Gradient Descent with Bounded Gradients

**原文翻译**：

**定理**：设 $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，$\mathbb{X}$ 是凸集。$x^* = \arg\min_{x \in \mathbb{X}} f(x)$。假设 $\|x^0 - x^*\| \leq R$ 且 $\|\nabla f(x)\| \leq B$ 对所有 $x \in \mathbb{X}$ 成立。则选择步长 $\gamma = \frac{R}{B\sqrt{K}}$ 时，$K$ 次投影梯度下降迭代满足：

$$\frac{1}{K}\sum_{k=0}^{K-1}\left(f(x^k) - f(x^*)\right) \leq \frac{RB}{\sqrt{K}}$$

**证明**：由我们的预备分析：

$$\sum_{k=0}^{K-1}\left(f(x^k) - f(x^*)\right) \leq \frac{\gamma}{2} K \cdot B^2 + \frac{1}{2\gamma} R^2$$

$$\frac{1}{K}\sum_{k=0}^{K-1}\left(f(x^k) - f(x^*)\right) \leq \frac{\gamma}{2}B^2 + \frac{1}{2\gamma}\frac{R^2}{K}$$

$$= \frac{1}{2}\cdot\frac{R}{B\sqrt{K}}\cdot B^2 + \frac{B\sqrt{K}}{2R}\cdot\frac{R^2}{K} = \frac{RB}{2\sqrt{K}} + \frac{RB}{2\sqrt{K}} = \frac{RB}{\sqrt{K}}$$

证毕！$\blacksquare$

**精讲内容**：

这一页是本讲的核心成果——投影梯度下降在有界梯度假设下的收敛定理。

**定理的假设**：
1. $f$ 是凸函数，$\mathbb{X}$ 是凸集——这是投影梯度下降理论的基本前提。
2. $\|x^0 - x^*\| \leq R$——初始点到最优解的距离有上界 $R$。$R$ 衡量了"起跑线有多远"。
3. $\|\nabla f(x)\| \leq B$ 对所有 $x \in \mathbb{X}$——梯度范数在可行域上有界，上界为 $B$。$B$ 衡量了"函数的最大陡峭程度"。

**定理的结论**：选择步长 $\gamma = R/(B\sqrt{K})$ 后，$K$ 次迭代的**平均**误差 $\leq RB/\sqrt{K}$。

**证明的关键步骤**：

1. 代入有界梯度假设：$\|g^k\|^2 \leq B^2$，所以 $\sum_{k=0}^{K-1}\|g^k\|^2 \leq KB^2$。
2. 代入初始距离假设：$\|x^0 - x^*\|^2 \leq R^2$。
3. 代入最优步长 $\gamma = R/(B\sqrt{K})$，得到两项相等：$\frac{RB}{2\sqrt{K}} + \frac{RB}{2\sqrt{K}} = \frac{RB}{\sqrt{K}}$。

**收敛速率 $O(1/\sqrt{K})$ 的含义**：

- 要达到 $\epsilon$-精度（平均误差 $\leq \epsilon$），需要 $K \geq (RB/\epsilon)^2$ 次迭代。
- 与无约束 GD 在 Lipschitz 梯度假设下的 $O(1/K)$ 收敛速率相比，这个 $O(1/\sqrt{K})$ 速率更慢。原因是这里用的是更弱的"有界梯度"假设（$\|\nabla f\| \leq B$），而不是更强的"Lipschitz 梯度"假设（$\|\nabla^2 f\| \leq L$）。
- 如果同时假设 Lipschitz 梯度，可以得到更快的 $O(1/K)$ 收敛速率（但证明更复杂）。

**步长选择的 trade-off**。最优步长 $\gamma = R/(B\sqrt{K})$ 恰好使两项相等——这是经典的 AM-GM 优化思路。$\gamma$ 与 $1/\sqrt{K}$ 成正比，意味着迭代次数越多，步长越小。这是合理的：随着我们接近最优解，需要更小的步长来精细调节。

**结论是关于"平均误差"而非"最优迭代"**。左边是 $\frac{1}{K}\sum_{k=0}^{K-1}(f(x^k) - f(x^*))$，是所有迭代的平均函数值误差。这不直接说"最后一次迭代 $x^{K-1}$ 有多好"，而是说"整个序列的平均有多好"。实际上，这意味着你可以用 $\bar{x} = \frac{1}{K}\sum_{k=0}^{K-1} x^k$ 作为你的解估计（由 Jensen 不等式，$f(\bar{x}) \leq \frac{1}{K}\sum f(x^k)$），或者直接取最好的那个 $x^k$。

**教授口吻讲解**：

这就是我们的收敛定理！$O(RB/\sqrt{K})$——看起来简单，但你已经花了两堂课来证明它。步长的选择是 $R/(B\sqrt{K})$——注意它需要你提前知道 $R$ 和 $B$，以及打算跑多少步 $K$。这在实践中有时不方便，但理论上它是最优的。你们可能会问：$O(1/\sqrt{K})$ 是不是太慢了？确实，比 Lipschitz 梯度假设下的 $O(1/K)$ 慢。但记住，这里我们只假设了梯度有界——这是一个弱得多的假设。弱假设自然只能给你弱结论。如果你的问题满足更强的条件（比如 Lipschitz 梯度或强凸），可以得到更快的速率。

### 第 ⑥ 页：关于有界梯度假设与计算投影

**原文翻译**：

- 注意，当 $\mathbb{X}$ 有界时，$\|\nabla f(x)\| \leq B$ 对所有 $x \in \mathbb{X}$ 并不是一个很糟糕的假设。

**计算投影**

- 在投影梯度下降的每次迭代中，我们需要计算 $\Pi_\mathbb{X}(y^{k+1})$。

- 计算这个投影等价于求解：

$$\min_{x \in \mathbb{X}} \sum_i (x_i - y_i^{k+1})^2$$

- 这本身又是一个约束优化问题！

- 通过利用范数函数 $\sum(x_i - y_i)^2$ 的结构，我们可以对许多不同的可行集 $\mathbb{X}$ 高效地计算投影。

- 以下是一些投影容易计算的可行集 $\mathbb{X}$ 的例子。

**精讲内容**：

这一页是从理论到实践的过渡，讨论了两个关键问题：假设的合理性和投影的计算。

**有界梯度假设的合理性**。$\|\nabla f(x)\| \leq B$ 对所有 $x \in \mathbb{X}$ 看起来是个强假设——它要求梯度在整个可行域上有界。但如果 $\mathbb{X}$ 本身是有界的（即 $\mathbb{X}$ 是紧集），而 $f$ 是连续可微的，那么 $\|\nabla f(x)\|$ 在 $\mathbb{X}$ 上的上确界是有限的（连续函数在紧集上的最大值存在）。所以对于有界可行域上的光滑凸函数，有界梯度假设是自动满足的。

ML 中的常见情况：
- $\mathbb{X}$ 是 L2-ball $\{\|w\| \leq R\}$（岭回归）：$\mathbb{X}$ 有界，假设自动满足。
- $\mathbb{X}$ 是概率单纯形：有界，假设自动满足。
- $\mathbb{X}$ 是 box $\{0 \leq x_i \leq U_i\}$：有界，假设自动满足。
- $\mathbb{X}$ 是 L1-ball $\{\|x\|_1 \leq R\}$：有界，假设自动满足。

**投影计算的挑战**。$\Pi_\mathbb{X}(y) = \arg\min_{x \in \mathbb{X}} \|x - y\|^2$ 本身是一个约束优化问题——这似乎形成了"鸡生蛋、蛋生鸡"的循环：我们用投影来解约束优化，但投影本身就是约束优化！

破解这个循环的关键在于：投影问题的目标函数 $\|x - y\|^2$ 有非常特殊的结构——它是**可分离的**（separable），即 $\sum_i (x_i - y_i)^2$ 可以分解为每个坐标独立的子问题 $\min_{x_i} (x_i - y_i)^2$。对于许多常见的 $\mathbb{X}$，这种可分离性可以被充分利用，使得投影问题有闭式解或高效算法。

Lecture 14 的下半部分（Box 投影）和 Lecture 15-16（欧几里得球、概率单纯形、L1-ball 投影）将逐一展示这些闭式解。

**教授口吻讲解**：

有人可能会说："有界梯度？这个假设太强了吧？"其实不然。只要你的可行域是有界的——而 ML 里常见的正则化约束几乎都是有界集——梯度就自动有界。所以这个假设没你想的那么"强"。现在我们进入实际操作：投影怎么算？理论上它是一个优化问题，但好消息是，对于最常见的约束集，投影都有漂亮的闭式解。让我们从最简单的开始——Box约束。

### 第 ⑦ 页：Projections on a Box

**原文翻译**：

考虑 $\mathbb{X} = \{x \in \mathbb{R}^n : 0 \leq x_i \leq U_i, \; \forall \, i = 1, \ldots, n\}$。

例如 $n = 2$。

> **[图示]** 三维图，水平轴为 $x_1$ 和 $x_2$，纵轴为目标函数值。可行域为 $x_1$-$x_2$ 平面上的矩形 $[0, U_1] \times [0, U_2]$。

我们要找 $\Pi_\mathbb{X}(\hat{y})$，其中 $\hat{y}$ 是给定点。

我们要求解 $\min_{x \in \mathbb{X}} \|x - \hat{y}\|^2$。

**精讲内容**：

这一页引入了 Box 约束——最简单但也最广泛使用的约束类型。

**Box 约束的定义**。$\mathbb{X} = \{x \in \mathbb{R}^n : 0 \leq x_i \leq U_i, \forall i\}$ 是一个 $n$ 维的"箱子"（box/hyperrectangle），每个坐标 $x_i$ 被限制在 $[0, U_i]$ 范围内。

**Box 约束在 ML 中的应用**：
- **非负约束**（$x_i \geq 0$）：非负矩阵分解（NMF）、非负最小二乘。这是 $U_i = \infty$ 的特例。
- **像素值约束**（$0 \leq x_i \leq 255$）：图像处理/对抗攻击中的像素值范围约束。
- **概率约束**（$0 \leq p_i \leq 1$）：二值分类中的预测概率。
- **权重截断**（$|w_i| \leq C$）：某些正则化策略限制每个权重的绝对值。

**投影问题的形式**。给定点 $\hat{y} \in \mathbb{R}^n$，要求解：
$$\min_{x \in \mathbb{X}} \|x - \hat{y}\|^2 = \min_{0 \leq x_i \leq U_i} \sum_i (x_i - \hat{y}_i)^2$$

这个问题的关键特征是：目标函数 $\sum_i (x_i - \hat{y}_i)^2$ 是关于各坐标 $x_i$ **可分离的**——没有交叉项 $x_i x_j$。同时，约束 $0 \leq x_i \leq U_i$ 也是对每个坐标独立的。这意味着 $n$ 维问题可以分解为 $n$ 个独立的一维问题，每个一维问题都可以解析地求解。

**图示的含义**。三维图展示了 $n=2$ 的情况：水平面上的矩形是可行域 $[0, U_1] \times [0, U_2]$，纵轴是 $\|x - \hat{y}\|^2$ 的值。投影就是在这个矩形上找使 $\|x - \hat{y}\|^2$ 最小的点。由于目标函数是凸的二次函数，等高线是同心椭圆（实际上是圆，因为系数都是1），所以直觉上投影要么是 $\hat{y}$ 本身（如果已在 box 内），要么是 box 边界上离 $\hat{y}$ 最近的点。

**教授口吻讲解**：

Box约束是最友好的约束类型。为什么？因为每个坐标是独立的——$x_1$ 的约束 $[0, U_1]$ 跟 $x_2$ 的约束 $[0, U_2]$ 毫不相干。这意味着你可以把 $n$ 维问题拆成 $n$ 个独立的一维问题来解。一维优化有多简单？画个图就能看出答案。让我们来画。

### 第 ⑧ 页：Box 投影的可分离性与 Case 1

**原文翻译**：

我们要求解

$$\min \sum_i (x_i - \hat{y}_i)^2 \quad \text{s.t.} \quad 0 \leq x_i \leq U_i, \; \forall \, i = 1, \ldots, n$$

这个问题等价于

$$\sum_i \min_{0 \leq x_i \leq U_i} (x_i - \hat{y}_i)^2$$

让我们看三种情况：

**Case 1**：$0 \leq \hat{y}_i \leq U_i$。

> **[图示]** 函数 $(x_i - \hat{y}_i)^2$ 的图像，最小值点 $\hat{y}_i$ 在 $[0, U_i]$ 内部。最小值在可行域内取到。

$$\hat{y}_i = \arg\min_{0 \leq x_i \leq U_i} (x_i - \hat{y}_i)^2$$

**精讲内容**：

这一页正式利用可分离性将投影问题分解，并分析第一种情况。

**可分离性的严格说明**。原问题是：

$$\min_{x} \sum_i (x_i - \hat{y}_i)^2 \quad \text{s.t.} \quad 0 \leq x_i \leq U_i, \forall i$$

由于目标函数是各坐标独立的和，约束也是各坐标独立的，所以联合最小化等价于逐坐标最小化：

$$\min_{x} \sum_i (x_i - \hat{y}_i)^2 = \sum_i \min_{0 \leq x_i \leq U_i} (x_i - \hat{y}_i)^2$$

这个等式成立的充分条件是：(1) 目标函数可分离；(2) 约束可分离。二者缺一不可——如果有像 $\sum x_i = 1$ 这样的耦合约束，分离就不行了（这就是概率单纯形投影更复杂的原因，见 Lecture 15）。

**Case 1 的分析**：当 $\hat{y}_i \in [0, U_i]$ 时，无约束最小值点 $\hat{y}_i$ 恰好在可行域内，约束不起作用。所以约束优化的解就是无约束的解 $x_i^* = \hat{y}_i$。

从图形上看，$(x_i - \hat{y}_i)^2$ 是以 $\hat{y}_i$ 为底的抛物线，当 $\hat{y}_i \in [0, U_i]$ 时，抛物线的底在可行区间内，所以最小值就在 $\hat{y}_i$ 取到。

**这种情况的实际含义**：梯度步后的坐标 $\hat{y}_i$ 还在 box 内，不需要投影。这在实践中是最常见的情况——大多数坐标在大多数迭代中不会违反约束，只有少数坐标偶尔需要被"推回去"。

**教授口吻讲解**：

可分离性是我们的好朋友。目标函数是各坐标的和，约束也是各坐标独立的，所以整个问题"碎"成了 $n$ 个一维小问题。一维优化在区间 $[0, U_i]$ 上？太简单了。Case 1：如果 $\hat{y}_i$ 本来就在区间内——什么都不用做，答案就是 $\hat{y}_i$ 本身。你已经在 box 里了，何必投影？

---

### 第 ⑨ 页：Case 2 与 Case 3

**原文翻译**：

**Case 2**：$\hat{y}_i \leq 0$。

> **[图示]** 函数 $(x_i - \hat{y}_i)^2$ 的图像，最小值点 $\hat{y}_i$ 在 0 的左侧。在 $[0, U_i]$ 上，函数在 $x_i = 0$ 处取最小值。

$$0 = \arg\min_{0 \leq x_i \leq U_i} (x_i - \hat{y}_i)^2$$

**Case 3**：$\hat{y}_i \geq U_i$。

> **[图示]** 函数 $(x_i - \hat{y}_i)^2$ 的图像，最小值点 $\hat{y}_i$ 在 $U_i$ 的右侧。在 $[0, U_i]$ 上，函数在 $x_i = U_i$ 处取最小值。

$$U_i = \arg\min_{0 \leq x_i \leq U_i} (x_i - \hat{y}_i)^2$$

**精讲内容**：

这一页分析了另外两种情况：无约束最小值点在可行域左边（Case 2）和右边（Case 3）。

**Case 2 的分析**：当 $\hat{y}_i \leq 0$ 时，无约束最小值点在可行域 $[0, U_i]$ 的左边。由于 $(x_i - \hat{y}_i)^2$ 是以 $\hat{y}_i$ 为底的抛物线，在 $[0, U_i]$ 上单调递减（从左端点 $0$ 的值 $\hat{y}_i^2$ 到某个更大的值... 不对，实际上在 $\hat{y}_i < 0$ 时，抛物线在 $x_i = 0$ 处的值是 $\hat{y}_i^2$，在 $x_i > 0$ 时更大）。等等，让我纠正一下：实际上 $(x_i - \hat{y}_i)^2$ 以 $\hat{y}_i$ 为最小值点，$\hat{y}_i < 0$ 意味着最小值点在 0 左侧。在 $[0, U_i]$ 上，函数从 $x_i = 0$ 处的值 $\hat{y}_i^2$ 开始单调递增（因为我们在抛物线的右半支），所以约束优化的解是左端点 $x_i^* = 0$。

**Case 3 的分析**：当 $\hat{y}_i \geq U_i$ 时，无约束最小值点在可行域的右边。抛物线在 $[0, U_i]$ 上单调递减（因为我们在抛物线的左半支），所以约束优化的解是右端点 $x_i^* = U_i$。

**三种情况的统一直觉**：Box 投影就是"裁剪"（clipping）——如果坐标值在 box 内就保留，如果超出上限就截断到上限，如果低于下限就截断到下限。这就是你在 NumPy 里用 `np.clip(y, 0, U)` 做的事情。

从几何上看，对每个坐标独立地做裁剪操作，等价于把点 $\hat{y}$ "推"到 box 上离它最近的位置。如果 $\hat{y}$ 在 box 的外面某个角落（比如 $\hat{y}_1 > U_1$ 且 $\hat{y}_2 < 0$），投影结果就是 box 对应的角 $(U_1, 0)$。

**与深度学习的联系**：ReLU 激活函数 $\text{ReLU}(x) = \max(0, x) = [x]^+$ 实际上就是往 $[0, \infty)$ 投影（下界为 0，上界为 $\infty$ 的 box 投影）。更一般地，Clipped ReLU（$\min(U, \max(0, x))$）就是完整的 box 投影。

**教授口吻讲解**：

Case 2 和 3 也很直观。$\hat{y}_i$ 太小了（低于 0）？截到 0。太大了（超过 $U_i$）？截到 $U_i$。就是这么简单——这就是 NumPy 的 `np.clip`。你看，投影梯度下降听起来很学术，但当约束是 box 的时候，"投影"就是"裁剪"，是你们每天在代码里做的事情。

### 第 ⑩ 页：Box 投影的闭式解

**原文翻译**：

因此，

$$\arg\min_{0 \leq x_i \leq U_i} (x_i - \hat{y}_i)^2 = \min\{U_i, \; [\hat{y}_i]^+\}$$

其中 $[a]^+ = \max\{0, a\}$。

因此，$\hat{y}$ 到 $\mathbb{X}$ 的投影为 $\pi = (\pi_1, \ldots, \pi_n)$，其中

$$\pi_i = \min\{U_i, \; [\hat{y}_i]^+\}$$

**精讲内容**：

这一页给出了 Box 投影的闭式解，将前面三种情况统一为一个简洁的公式。

**公式解读**。$\pi_i = \min\{U_i, [\hat{y}_i]^+\}$，其中 $[a]^+ = \max\{0, a\}$。这个公式的逻辑是：

1. 先取 $[\hat{y}_i]^+ = \max\{0, \hat{y}_i\}$：如果 $\hat{y}_i < 0$，截断到 0（处理 Case 2）。
2. 再取 $\min\{U_i, \cdot\}$：如果上一步的结果 $> U_i$，截断到 $U_i$（处理 Case 3）。
3. 如果 $0 \leq \hat{y}_i \leq U_i$，两个操作都不改变值（处理 Case 1）。

等价写法：$\pi_i = \text{clip}(\hat{y}_i, 0, U_i)$。

**计算复杂度**。每个坐标的投影是 $O(1)$ 操作（两次比较），$n$ 个坐标共 $O(n)$。这与一次梯度计算的复杂度相当——投影几乎没有额外开销。

**Box 投影的普遍性**。Box 投影是所有投影中最简单的，因为它的约束完全可分离。但正因为简单，它的应用极为广泛：

1. **梯度裁剪**（Gradient Clipping）：深度学习中常用 $\min\{C, |g_i|\} \cdot \text{sign}(g_i)$ 来裁剪梯度，这本质上是往 $[-C, C]^n$ box 投影。
2. **对抗攻击中的约束**：FGSM/PGD 攻击中，扰动 $\delta$ 需要满足 $x + \delta \in [0, 255]^n$（像素值范围），这需要 box 投影。
3. **变量界约束**：线性规划中的变量上下界约束 $l_i \leq x_i \leq u_i$。

**从 Box 到更复杂的集合**。Box 投影之所以简单，核心原因是约束的可分离性。当约束涉及坐标之间的耦合（如 $\sum x_i = 1$ 或 $\sum x_i^2 \leq R^2$），投影就不再是逐坐标独立的，需要更复杂的技术——这就是 Lecture 15-16 的内容。

**教授口吻讲解**：

这个公式 $\pi_i = \min\{U_i, [\hat{y}_i]^+\}$ 极其简洁——就是"先跟 0 取 max，再跟 $U_i$ 取 min"。Python 一行代码：`np.clip(y, 0, U)`。Box 投影是 $O(n)$ 的——跟算一次梯度一样快。所以当约束是 box 的时候，投影梯度下降几乎没有比普通梯度下降多花什么计算。但是，别高兴得太早——下节课我们要处理欧几里得球和概率单纯形，那里的投影就没这么"免费"了。不过也不会太贵——都有闭式解。

## 核心收获

1. **收敛定理**：投影 GD 在有界梯度假设下以 $O(RB/\sqrt{K})$ 速率收敛（平均误差）。步长 $\gamma = R/(B\sqrt{K})$ 平衡了梯度项和初始距离项。
2. **伸缩求和是核心技巧**：将单步误差上界中的 $\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2$ 项求和消去，是从"单步"到"全局"收敛保证的桥梁。
3. **投影性质 2 是桥梁**：它使得 $y^{k+1}$ 可以被安全地替换为 $x^{k+1}$，恢复伸缩结构。
4. **Box 投影 = 裁剪**：$\pi_i = \text{clip}(\hat{y}_i, 0, U_i)$，$O(n)$ 计算复杂度，无额外开销。
5. **可分离性是关键**：Box 投影之所以简单，根源在于目标函数和约束都关于各坐标可分离。非可分离约束（单纯形、L1-ball）需要更精细的技术。
