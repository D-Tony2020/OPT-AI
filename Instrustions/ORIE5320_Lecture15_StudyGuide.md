### 第 ① 页：Carrying out projections on to specially structured sets

**原文翻译**：

- 为了在 box 以外的集合上做投影，我们需要一些特殊工具。考虑问题：

$$\min \quad f(x) \quad \text{s.t.} \quad g(x) = b, \; x \in \mathbb{X} \tag{‡}$$

其中 $f: \mathbb{R}^n \to \mathbb{R}$，$g: \mathbb{R}^n \to \mathbb{R}$。

- 对固定的 $\lambda \in \mathbb{R}$，考虑问题：

$$\min \quad f(x) + \lambda\bigl(g(x) - b\bigr) \quad \text{s.t.} \quad x \in \mathbb{X} \tag{*}$$

- 令 $x^*(\lambda)$ 为问题 $(*)$ 的唯一最优解。

- 找到 $\lambda^*$ 使得 $g(x^*(\lambda^*)) = b$。

- 则 $x^*(\lambda^*)$ 是问题 $(\ddagger)$ 的最优解。

**精讲内容**：

这一页引入了拉格朗日方法（Lagrangian Method），它是处理等式约束优化的核心工具。

**问题背景**。Lecture 14 中 Box 投影之所以简单，是因为约束 $0 \leq x_i \leq U_i$ 关于各坐标完全可分离。但欧几里得球约束 $\sum x_i^2 \leq R^2$ 或单纯形约束 $\sum x_i = 1$ 涉及所有坐标的联合关系——你不能单独决定每个 $x_i$，因为它们之间被约束"绑"在了一起。

拉格朗日方法的思想是：不要直接处理等式约束 $g(x) = b$，而是把它"吸收"到目标函数中。具体做法是引入一个**拉格朗日乘子** $\lambda$，将约束违反的惩罚项 $\lambda(g(x) - b)$ 加到目标函数上。

**方法的三步走**：

**Step 1（松弛）**：对固定的 $\lambda$，忽略等式约束 $g(x) = b$，求解带有惩罚项的"松弛问题" $(*)$：
$$\min_{x \in \mathbb{X}} f(x) + \lambda(g(x) - b)$$
这个问题只有 $x \in \mathbb{X}$ 的约束（没有 $g(x) = b$ 的约束），通常更容易求解。

**Step 2（调参）**：找到那个"恰好"使等式约束自动满足的 $\lambda^*$——即使得 $g(x^*(\lambda^*)) = b$ 成立。

**Step 3（结论）**：$x^*(\lambda^*)$ 就是原问题 $(‡)$ 的最优解。

**为什么 $\lambda$ 的存在是合理的**？直觉上，$\lambda$ 是等式约束的"价格"：如果 $\lambda$ 大，违反 $g(x) = b$ 的惩罚就重，松弛问题的解会倾向于满足约束；如果 $\lambda$ 小，惩罚轻，解可能偏离约束。随着 $\lambda$ 从 $-\infty$ 到 $+\infty$ 连续变化，$g(x^*(\lambda))$ 也连续变化，所以（在适当条件下）一定存在某个 $\lambda^*$ 使得 $g(x^*(\lambda^*)) = b$。

**前提条件**。这个方法需要几个条件才能正确工作：(1) $(*)$ 有唯一最优解 $x^*(\lambda)$；(2) 存在 $\lambda^*$ 使 $g(x^*(\lambda^*)) = b$。在我们后面讨论的投影问题中，这些条件都自然满足。

**与 Lecture 3-4 中凸函数一阶特征化的联系**。拉格朗日方法的正确性证明（下一页）利用了一个巧妙的代数观察：当 $g(x) = b$ 时，惩罚项 $\lambda(g(x) - b) = 0$。所以对于满足等式约束的任何可行解，惩罚项不起作用——这是证明的关键。

**教授口吻讲解**：

Box 投影用"可分离性"轻松搞定了。但欧几里得球和单纯形不行——坐标之间有耦合。怎么办？我们请出优化理论的老朋友——拉格朗日乘子。核心思想是：你不想直接处理等式约束 $g(x) = b$？好，我把它变成目标函数里的一个惩罚项 $\lambda(g(x) - b)$。然后调 $\lambda$ 直到约束自动满足。这个方法在这门课之前的部分你们见过它的精神——现在我们正式使用它。

### 第 ② 页：拉格朗日方法的正确性证明

**原文翻译**：

**证明**：令 $\hat{x}$ 为 $(‡)$ 的最优解。所以 $g(\hat{x}) = b$，$\hat{x} \in \mathbb{X}$。

同时，$x^*(\lambda^*)$ 满足 $g(x^*(\lambda^*)) = b$，$x^*(\lambda^*) \in \mathbb{X}$。

所以我们得到 $f(\hat{x}) \leq f(x^*(\lambda^*))$。

$x^*(\lambda^*)$ 是 $(*)$（$\lambda = \lambda^*$ 时）的最优解，所以 $x^*(\lambda^*) \in \mathbb{X}$，$g(x^*(\lambda^*)) = b$。

同时我们有 $\hat{x} \in \mathbb{X}$，$g(\hat{x}) = b$。

因为 $x^*(\lambda^*)$ 是 $(*)$ 的最优解，我们得到：

$$f(x^*(\lambda^*)) + \lambda^*\bigl(g(x^*(\lambda^*)) - b\bigr) \leq f(\hat{x}) + \lambda^*\bigl(g(\hat{x}) - b\bigr)$$

两个 $g$ 项都为 0：

$$f(x^*(\lambda^*)) \leq f(\hat{x})$$

因此，$f(x^*(\lambda^*)) = f(\hat{x})$。

同时 $x^*(\lambda^*) \in \mathbb{X}$，$g(x^*(\lambda^*)) = b$。所以 $x^*(\lambda^*)$ 是 $(‡)$ 的可行解，且给出的目标值等于最优目标值。

因此 $x^*(\lambda^*)$ 必定是 $(‡)$ 的最优解。证毕！$\blacksquare$

**精讲内容**：

这一页给出了拉格朗日方法正确性的严格证明，逻辑非常精巧。

**证明的关键洞察**：证明需要展示 $f(x^*(\lambda^*)) = f(\hat{x})$（松弛问题的解和原问题的最优值相同），以及 $x^*(\lambda^*)$ 是原问题 $(‡)$ 的可行解。

**逐步分解**：

**Step 1（不等式 1：原问题更优）**：$\hat{x}$ 是原问题 $(‡)$ 的最优解，$x^*(\lambda^*)$ 是 $(‡)$ 的可行解（因为 $g(x^*(\lambda^*)) = b$ 且 $x^*(\lambda^*) \in \mathbb{X}$），所以 $f(\hat{x}) \leq f(x^*(\lambda^*))$。

**Step 2（不等式 2：松弛问题更优）**：$x^*(\lambda^*)$ 是松弛问题 $(*)$（$\lambda = \lambda^*$）的最优解。$\hat{x}$ 是松弛问题的一个可行解（因为 $\hat{x} \in \mathbb{X}$）。所以松弛目标值在 $x^*(\lambda^*)$ 处 $\leq$ 在 $\hat{x}$ 处：
$$f(x^*(\lambda^*)) + \lambda^*(g(x^*(\lambda^*)) - b) \leq f(\hat{x}) + \lambda^*(g(\hat{x}) - b)$$

**Step 3（利用等式约束消去惩罚项）**：由于 $g(x^*(\lambda^*)) = b$ 和 $g(\hat{x}) = b$，两边的惩罚项 $\lambda^*(\cdot - b)$ 都等于 0。所以：
$$f(x^*(\lambda^*)) \leq f(\hat{x})$$

**Step 4（两个不等式合并）**：从 Step 1 和 Step 3，$f(\hat{x}) \leq f(x^*(\lambda^*))$ 且 $f(x^*(\lambda^*)) \leq f(\hat{x})$，因此 $f(x^*(\lambda^*)) = f(\hat{x})$。

**Step 5（可行性 + 最优值 = 最优解）**：$x^*(\lambda^*)$ 是 $(‡)$ 的可行解（$g = b$，在 $\mathbb{X}$ 内），且目标值等于 $(‡)$ 的最优目标值，因此它就是 $(‡)$ 的最优解。

**证明的精妙之处**在于 Step 3：两个点都满足 $g = b$，所以惩罚项在两个点处都消失了。这意味着松弛问题在这两个点上的比较退化为原目标函数 $f$ 的比较。这是拉格朗日方法的核心机制——乘子 $\lambda$ 的作用是"引导"松弛问题的解去满足等式约束；一旦约束满足了，惩罚项就消失了，松弛问题和原问题"在解处等价"。

**与 KKT 条件的关系**。这个证明实质上是 KKT 条件充分性的一个特例。在更一般的优化理论中（Lecture 3-4 的精神推广），KKT 条件给出了约束优化最优性的充分必要条件，而拉格朗日乘子 $\lambda^*$ 就是 KKT 条件中的对偶变量。

**教授口吻讲解**：

这个证明很精巧，但逻辑其实很清楚。核心trick是什么？两个关键点（原问题的最优解 $\hat{x}$ 和我们找到的 $x^*(\lambda^*)$）都满足 $g = b$，所以惩罚项在两个点上都等于零。这意味着在松弛问题 $(*)$ 中比较这两个点，等价于在原目标函数 $f$ 下比较它们。松弛问题说 $x^*(\lambda^*)$ 更好，原问题说 $\hat{x}$ 更好——两边一比，一定相等。漂亮吧？好，现在让我们用这个工具来算欧几里得球的投影。

### 第 ③ 页：Projections on to the Euclidean Ball

**原文翻译**：

考虑

$$\mathbb{X} = \left\{ x \in \mathbb{R}^n : \sum_{i} x_i^2 \leq R^2 \right\}$$

> **[图示]** 二维坐标系（$x_1$, $x_2$ 轴），原点为圆心、半径为 $R$ 的圆形区域即为欧几里得球 $\mathbb{X}$。

- 我们要找 $\Pi_\mathbb{X}(\hat{y})$，其中 $\hat{y}$ 是给定点。

- 如果 $\hat{y} \in \mathbb{X}$，则 $\Pi_\mathbb{X}(\hat{y}) = \hat{y}$，所以考虑 $\hat{y} \notin \mathbb{X}$ 的情况，即

$$\sum_{i} \hat{y}_i^2 > R^2$$

**精讲内容**：

这一页引入了欧几里得球投影问题——ML 中最重要的投影问题之一。

**欧几里得球的定义**。$\mathbb{X} = \{x : \|x\|^2 \leq R^2\}$ 是以原点为中心、半径为 $R$ 的 $n$ 维闭球。它的边界是球面 $\{x : \|x\|^2 = R^2\}$。

**ML 中的应用**：
- **岭回归**（Ridge Regression）：$\min_{\|w\|_2 \leq R} \|Xw - y\|^2$，约束就是欧几里得球。
- **权重衰减**（Weight Decay）：深度学习中的 L2 正则化等价于限制参数在欧几里得球内。
- **Trust Region 方法**：优化中的信赖域方法在每一步限制 $\|x^{k+1} - x^k\| \leq R$，这是关于中心为 $x^k$ 的欧几里得球约束。
- **差分隐私**：梯度裁剪 $\|g\| \leq C$ 是把梯度投影到欧几里得球上。

**平凡情况的处理**。如果 $\hat{y}$ 已经在球内（$\|\hat{y}\| \leq R$），投影就是 $\hat{y}$ 本身——不需要任何操作。所以有趣的情况是 $\hat{y} \notin \mathbb{X}$，即 $\|\hat{y}\| > R$。

**为什么不能直接用 Box 投影的方法**？欧几里得球约束 $\sum x_i^2 \leq R^2$ 是一个关于所有坐标的耦合约束——你不能独立决定每个 $x_i$，因为它们的平方和受限。Box 投影能逐坐标分解，是因为约束 $0 \leq x_i \leq U_i$ 不涉及坐标间的相互依赖。这里需要用第 ①-② 页建立的拉格朗日方法。

**一个重要的物理直觉**。欧几里得球具有**旋转对称性**——绕原点旋转后球不变。这意味着投影操作应该保持方向不变，只改变模长。直觉上，如果 $\hat{y}$ 在球外，它的投影应该是"沿着 $\hat{y}$ 方向、在球面上"的那个点。即 $\Pi_\mathbb{X}(\hat{y}) = R \cdot \hat{y}/\|\hat{y}\|$——把 $\hat{y}$ "缩放"到球面上。下两页的拉格朗日推导将严格验证这个直觉。

**教授口吻讲解**：

欧几里得球——在 ML 里无处不在。L2 正则化、权重衰减、梯度裁剪，本质上都是在跟这个球打交道。如果 $\hat{y}$ 已经在球内，投影等于自己——这跟 Box 投影的 Case 1 一样，不需要做任何事。有趣的是 $\hat{y}$ 在球外的情况。直觉告诉你：投影应该是"同方向但缩短到球面上"。下面我们用拉格朗日方法严格验证这个直觉。

### 第 ④ 页：欧几里得球投影——拉格朗日求解

**原文翻译**：

我们要求解

$$\min \quad \sum_{i} (x_i - \hat{y}_i)^2 \quad \text{s.t.} \quad \sum_{i} x_i^2 = R^2 \tag{‡}$$

对固定的 $\lambda$，考虑问题：

$$\min_x \left\{ \sum_{i} (x_i - \hat{y}_i)^2 + \lambda\left(\sum_{i} x_i^2 - R^2\right) \right\}$$

最后一个问题等价于：

$$\sum_{i} \min_{x_i} \left\{ (x_i - \hat{y}_i)^2 + \lambda \, x_i^2 \right\} - \lambda R^2$$

每个最小化问题的最优解为：

$$2(x_i - \hat{y}_i) + 2\lambda x_i = 0 \implies x_i = \frac{1}{1+\lambda}\,\hat{y}_i$$

$$\implies x_i^*(\lambda) = \frac{1}{1+\lambda}\,\hat{y}_i$$

找 $\lambda^*$ 使得 $g(x^*(\lambda^*)) = b$，即 $\sum_{i} (x_i^*(\lambda^*))^2 = R^2$。所以我们要：

$$\sum_{i} \frac{1}{(1+\lambda^*)^2}\,\hat{y}_i^2 = R^2 \implies \frac{1}{(1+\lambda^*)^2}\,\|\hat{y}\|^2 = R^2 \implies \frac{1}{1+\lambda^*} = \frac{R}{\|\hat{y}\|}$$

**精讲内容**：

这一页是拉格朗日方法在欧几里得球投影上的具体应用，展示了从一般理论到特定问题的完整求解过程。

**问题转化**。注意到一个关键点：当 $\hat{y} \notin \mathbb{X}$（即 $\|\hat{y}\| > R$）时，投影一定在球面上而不是球内。所以不等式约束 $\sum x_i^2 \leq R^2$ 在最优解处变成等式 $\sum x_i^2 = R^2$。这使得我们可以用等式约束的拉格朗日方法。

**拉格朗日松弛**。对固定 $\lambda$，松弛问题是：
$$\min_x \sum_i (x_i - \hat{y}_i)^2 + \lambda(\sum_i x_i^2 - R^2)$$

展开并重组后，注意到这个问题关于各 $x_i$ 是可分离的——因为原目标函数 $\sum(x_i - \hat{y}_i)^2$ 和惩罚项 $\lambda\sum x_i^2$ 都是各坐标独立的和。所以可以逐坐标求解。

**逐坐标求解**。对第 $i$ 个坐标：
$$\min_{x_i} (x_i - \hat{y}_i)^2 + \lambda x_i^2$$

对 $x_i$ 求导并令导数为零：
$$2(x_i - \hat{y}_i) + 2\lambda x_i = 0$$
$$x_i(1 + \lambda) = \hat{y}_i$$
$$x_i = \frac{\hat{y}_i}{1+\lambda}$$

**惊人的简洁性**：每个 $x_i^*(\lambda)$ 就是 $\hat{y}_i$ 乘以一个**统一的缩放因子** $\frac{1}{1+\lambda}$——不依赖于 $i$！这意味着 $x^*(\lambda)$ 和 $\hat{y}$ 方向相同，只是模长不同。这验证了我们之前的旋转对称性直觉。

**求解 $\lambda^*$**。代入等式约束 $\sum (x_i^*(\lambda^*))^2 = R^2$：
$$\frac{1}{(1+\lambda^*)^2}\sum \hat{y}_i^2 = R^2$$
$$\frac{\|\hat{y}\|^2}{(1+\lambda^*)^2} = R^2$$
$$\frac{1}{1+\lambda^*} = \frac{R}{\|\hat{y}\|}$$

注意 $\frac{1}{1+\lambda^*} = R/\|\hat{y}\| < 1$（因为 $\|\hat{y}\| > R$），所以 $\lambda^* > 0$——乘子为正，意味着约束"在起作用"（球面约束是"紧"的）。

**拉格朗日方法的优美之处**。原问题是一个 $n$ 维的约束优化问题，但通过拉格朗日松弛，我们把它分解为 $n$ 个无约束的一维问题（每个有闭式解），加上一个关于 $\lambda$ 的一维方程（也有闭式解）。总计算复杂度是 $O(n)$——跟 Box 投影一样！

**教授口吻讲解**：

看到没有？拉格朗日方法的威力就在这里。原来的问题是 $n$ 维的，坐标之间被 $\sum x_i^2 = R^2$ 绑在一起。但加了拉格朗日乘子之后，问题变得可分离了！因为 $\lambda x_i^2$ 和 $(x_i - \hat{y}_i)^2$ 都是各坐标独立的。所以你可以逐坐标求解，每个坐标的解是 $\hat{y}_i/(1+\lambda)$——同一个缩放因子。然后解一个关于 $\lambda$ 的方程就行了。整个过程 $O(n)$。漂亮不漂亮？

### 第 ⑤ 页：欧几里得球投影公式

**原文翻译**：

因此

$$x_i^*(\lambda^*) = \frac{1}{1+\lambda^*}\,\hat{y}_i = \frac{R}{\|\hat{y}\|}\,\hat{y}_i$$

在这种情况下，$(‡)$ 的最优解为 $x^*$，其中

$$x_i^* = \frac{R}{\|\hat{y}\|}\,\hat{y}_i$$

因此，$\hat{y}$ 到欧几里得球的投影为 $\pi$，其中

$$\pi_i = \frac{R}{\|\hat{y}\|}\,\hat{y}_i \qquad \left(\pi = \frac{R}{\|\hat{y}\|}\,\hat{y}\right)$$

> **[图示]** 二维坐标系中的圆（欧几里得球）。点 $\hat{y}$ 在圆外。从原点指向 $\hat{y}$ 的方向上的单位向量为 $\hat{y}/\|\hat{y}\|$。投影点 $\Pi_\mathbb{X}(\hat{y}) = R\cdot\hat{y}/\|\hat{y}\|$ 位于圆的边界上，即沿 $\hat{y}$ 方向、距原点 $R$ 的点。

- 欧几里得球投影对岭回归（Ridge Regression）很有用。

**精讲内容**：

这一页给出了欧几里得球投影的最终公式——整门课中最优雅的结果之一。

**投影公式**：

$$\Pi_\mathbb{X}(\hat{y}) = \begin{cases} \hat{y} & \text{if } \|\hat{y}\| \leq R \\ \frac{R}{\|\hat{y}\|}\hat{y} & \text{if } \|\hat{y}\| > R \end{cases}$$

当 $\hat{y}$ 在球外时，投影公式 $\pi = R\hat{y}/\|\hat{y}\|$ 的含义极为清晰：

1. **方向保持**：$\pi$ 和 $\hat{y}$ 方向相同（$\pi = c \cdot \hat{y}$，$c > 0$）。
2. **模长截断**：$\|\pi\| = R$，即投影点恰好在球面上。
3. **缩放因子**：$c = R/\|\hat{y}\| < 1$，即将 $\hat{y}$ 按比例缩小到球面上。

这个公式等价于：$\pi = R \cdot \text{unit}(\hat{y})$，其中 $\text{unit}(\hat{y}) = \hat{y}/\|\hat{y}\|$ 是单位化（归一化）操作。

**与 Box 投影的对比**：
| 特性 | Box 投影 | 欧几里得球投影 |
|------|---------|------------|
| 操作 | 逐坐标裁剪 $\text{clip}(y_i, 0, U_i)$ | 全局缩放 $R\hat{y}/\|\hat{y}\|$ |
| 可分离性 | 完全可分离 | 不可分离（但通过拉格朗日松弛后变为可分离） |
| 几何意义 | 推到最近的面/边/角 | 沿径向方向缩到球面 |
| 计算复杂度 | $O(n)$（逐坐标） | $O(n)$（计算范数 + 缩放） |

**在深度学习中的应用**：
- **梯度裁剪（Gradient Clipping by Norm）**：如果 $\|g\| > C$，将梯度缩放为 $g \leftarrow C \cdot g/\|g\|$——这就是往半径为 $C$ 的欧几里得球投影。这是训练 Transformer 等深度网络时的标准技巧。
- **差分隐私中的梯度裁剪**：DP-SGD 要求每个样本的梯度范数 $\leq C$，裁剪操作就是欧几里得球投影。
- **对比学习中的归一化**：SimCLR 等方法将嵌入向量归一化到单位球面上，这是欧几里得球投影（$R=1$）。

**岭回归的联系**。岭回归的约束形式是：
$$\min_{\|w\| \leq R} \|Xw - y\|^2$$

用投影梯度下降求解时，每一步的投影就是欧几里得球投影。由于投影是 $O(n)$ 的，投影梯度下降的每步开销与无约束 GD 相当。

**教授口吻讲解**：

$\pi = R\hat{y}/\|\hat{y}\|$——多简洁的公式啊。方向不变，模长截到 $R$。你们在深度学习里天天用这个——梯度裁剪 `if norm(g) > C: g = C * g / norm(g)` 就是欧几里得球投影。现在你知道了它的数学原理。计算代价？$O(n)$——算一次范数再乘个标量。跟 Box 投影一样便宜。但下一个例子——概率单纯形——就要稍微复杂一点了。

### 第 ⑥ 页：Projection on to a Probability Simplex

**原文翻译**：

考虑

$$\mathbb{X} = \left\{ x \in \mathbb{R}^n : \sum_{i} x_i = 1, \; x_i \geq 0, \; \forall\, i = 1, \ldots, n \right\}$$

> **[图示]** 二维坐标系（$x_1$, $x_2$ 轴），第一象限内从 $(1,0)$ 到 $(0,1)$ 的线段即为概率单纯形。

- 我们要找 $\Pi_\mathbb{X}(\hat{y})$，其中 $\hat{y}$ 是给定点。

**精讲内容**：

这一页引入了概率单纯形投影——在 ML 中与分类、概率模型密切相关的约束集。

**概率单纯形的定义与结构**。概率单纯形（Probability Simplex）$\Delta^n = \{x \in \mathbb{R}^n : \sum x_i = 1, x_i \geq 0\}$ 是所有 $n$ 维概率分布的集合。它有两类约束：
1. **等式约束**：$\sum x_i = 1$（归一化）——这是耦合约束。
2. **不等式约束**：$x_i \geq 0$（非负性）——这是可分离约束。

在 $n = 2$ 时，单纯形是从 $(1,0)$ 到 $(0,1)$ 的线段。在 $n = 3$ 时，是三角形（顶点为 $(1,0,0)$, $(0,1,0)$, $(0,0,1)$）。

**ML 中的应用**：
- **Softmax 层**：分类模型的输出层将 logits 映射为概率分布，输出在单纯形上。
- **混合模型**：EM 算法中，混合权重 $\pi_k$ 满足 $\sum \pi_k = 1, \pi_k \geq 0$。
- **注意力机制**：Transformer 中的注意力权重是在单纯形上的（softmax 输出）。
- **在线学习**：多臂赌博机问题中的策略是概率分布。

**与欧几里得球投影的对比**。欧几里得球只有一个耦合约束 $\sum x_i^2 \leq R^2$，没有非负约束。概率单纯形有一个耦合等式约束 $\sum x_i = 1$ 加上 $n$ 个非负不等式约束 $x_i \geq 0$。非负约束使得投影公式更复杂——某些坐标可能被"截断"到 0。

**教授口吻讲解**：

概率单纯形——这是 ML 里最自然的约束之一。你训练一个分类器，输出是概率分布，它就得在这个单纯形上。跟欧几里得球比，单纯形多了非负约束 $x_i \geq 0$。这会带来一个新的复杂性：投影后某些坐标可能变成 0——"截断效应"。让我们用拉格朗日方法来解决它。

### 第 ⑦ 页：概率单纯形投影——拉格朗日求解

**原文翻译**：

我们要求解

$$\min \quad \sum_{i} (x_i - \hat{y}_i)^2 \quad \text{s.t.} \quad \sum_{i} x_i = 1, \; x_i \geq 0, \; \forall\, i = 1, \ldots, n$$

对固定的 $\lambda$，考虑问题：

$$\min_{x_i \geq 0} \left\{ \sum_{i} (x_i - \hat{y}_i)^2 + \lambda\left(\sum_{i} x_i - 1\right) \right\}$$

最后一个问题等价于：

$$\sum_{i} \min_{x_i \geq 0} \left\{ (x_i - \hat{y}_i)^2 + \lambda \, x_i \right\} - \lambda$$

每个最小化问题的求解：

$$2(x_i - \hat{y}_i) + \lambda = 0 \implies x_i = \hat{y}_i - \frac{\lambda}{2}$$

因此，

$$x_i^*(\lambda) = \left[\hat{y}_i - \frac{\lambda}{2}\right]^+$$

**精讲内容**：

这一页将拉格朗日方法应用于概率单纯形投影，展示了非负约束如何引入"截断"效应。

**问题的层次结构**。概率单纯形有两类约束：等式 $\sum x_i = 1$ 和不等式 $x_i \geq 0$。拉格朗日方法处理等式约束 $\sum x_i = 1$，不等式约束 $x_i \geq 0$ 被保留在松弛问题中。这与欧几里得球的处理方式平行——松弛等式约束，保留简单的可分离不等式约束。

**拉格朗日松弛**。对固定 $\lambda$，松弛等式约束 $\sum x_i = 1$：
$$\min_{x_i \geq 0} \sum_i (x_i - \hat{y}_i)^2 + \lambda(\sum_i x_i - 1)$$

展开后，$\lambda \sum x_i = \sum \lambda x_i$，所以目标函数仍然是关于各 $x_i$ 可分离的。加上约束 $x_i \geq 0$ 也是可分离的，所以可以逐坐标求解。

**逐坐标求解**。对第 $i$ 个坐标：
$$\min_{x_i \geq 0} (x_i - \hat{y}_i)^2 + \lambda x_i$$

无约束最优解：令导数 $2(x_i - \hat{y}_i) + \lambda = 0$，得 $x_i = \hat{y}_i - \lambda/2$。

但我们有约束 $x_i \geq 0$，所以：
- 如果 $\hat{y}_i - \lambda/2 \geq 0$：无约束解满足约束，$x_i^* = \hat{y}_i - \lambda/2$。
- 如果 $\hat{y}_i - \lambda/2 < 0$：无约束解违反约束，最优解是边界点 $x_i^* = 0$。

统一写法：$x_i^*(\lambda) = [\hat{y}_i - \lambda/2]^+$，其中 $[a]^+ = \max\{0, a\}$。

**与欧几里得球投影的关键区别**。在欧几里得球投影中，所有坐标的缩放因子相同（$1/(1+\lambda)$），不存在截断效应。在单纯形投影中，每个坐标被平移（$\hat{y}_i - \lambda/2$）后截断到非负——不同坐标可能有不同的行为（有些截断到 0，有些保持正值）。这使得投影公式更丰富。

**物理直觉**。$\lambda/2$ 可以理解为一个"统一门槛"——所有坐标减去这个门槛，减完后如果为负就截断到 0。$\lambda$ 越大，门槛越高，越多坐标被截断到 0（投影结果越"稀疏"）。

**教授口吻讲解**：

注意跟欧几里得球的区别。在球上，拉格朗日松弛后每个坐标是"缩放"——乘以同一个系数。在单纯形上，每个坐标是"平移后截断"——减去同一个常数，然后 $\max$ 到 0。为什么有截断？因为单纯形有非负约束 $x_i \geq 0$。如果某个 $\hat{y}_i$ 本来就很小，减去 $\lambda/2$ 后会变成负数，这时候就被"钉"在 0 上。

### 第 ⑧ 页：分段最优解的图示

**原文翻译**：

> **[图示——左图]** 绘制 $(x_i - \hat{y}_i)^2 + \lambda x_i$ 关于 $x_i$ 的图像，横轴 $x_i \geq 0$。当 $\hat{y}_i - \lambda/2 > 0$ 时，无约束最小值点 $\hat{y}_i - \lambda/2$ 在可行域内，因此最优解为 $x_i^* = \hat{y}_i - \lambda/2$。
>
> **[图示——右图]** 同样的函数图像，当 $\hat{y}_i - \lambda/2 < 0$ 时，无约束最小值点在可行域外（$x_i < 0$），因此在约束 $x_i \geq 0$ 下最优解为 $x_i^* = 0$。

**精讲内容**：

这一页通过图示直观解释了 $x_i^*(\lambda) = [\hat{y}_i - \lambda/2]^+$ 这个公式的两种情况。

**左图（正常情况）**：函数 $h(x_i) = (x_i - \hat{y}_i)^2 + \lambda x_i$ 是一个向上开口的抛物线（因为 $(x_i - \hat{y}_i)^2$ 的二次系数为正）。它的最小值点在 $x_i = \hat{y}_i - \lambda/2$。当这个点在 $x_i \geq 0$ 的范围内（即 $\hat{y}_i - \lambda/2 > 0$），约束不影响，最优解就是 $\hat{y}_i - \lambda/2$。

**右图（截断情况）**：同样的抛物线，但现在最小值点 $\hat{y}_i - \lambda/2 < 0$——在可行域外。由于 $h(x_i)$ 在 $[0, +\infty)$ 上是先递减后递增的抛物线，但递减的部分在可行域外（$x_i < 0$），所以在可行域 $x_i \geq 0$ 上 $h$ 是单调递增的（从 $x_i = 0$ 开始）。因此最优解是左端点 $x_i^* = 0$。

**$\lambda$ 增大时的效应**。$\lambda x_i$ 项相当于给目标函数加了一个正斜率——$\lambda$ 越大，抛物线的对称轴 $\hat{y}_i - \lambda/2$ 越向左偏移，使得更多坐标的最优解被"推"到 0。从最优化的角度，增大 $\lambda$ 相当于增加"归一化的代价"，迫使更多坐标为零来满足 $\sum x_i = 1$ 的约束。

**与 soft-thresholding 的联系**。$[a - \tau]^+$ 这种"减去门槛然后截断"的操作在优化和信号处理中非常常见，称为 soft-thresholding（软阈值化）。它是 LASSO（L1 正则化）的 proximal operator 的核心组件。这里它以不同的面貌出现在单纯形投影中——数学结构是相同的。

**教授口吻讲解**：

这两张图把一切说清楚了。左图：无约束最小值点在可行域内——直接取。右图：无约束最小值点在可行域外——截断到 0。统一起来就是 $[\hat{y}_i - \lambda/2]^+$。这个 $[\cdot]^+$ 操作你们以后会反复见到——在 LASSO、proximal 方法、稀疏优化里。它是优化里的"标配"。

### 第 ⑨ 页：概率单纯形投影公式

**原文翻译**：

找 $\lambda^*$ 使得 $g(x^*(\lambda^*)) = b$，即 $\sum_{i} x_i^*(\lambda^*) = 1$。所以我们要：

$$\sum_{i} \left[\hat{y}_i - \frac{\lambda^*}{2}\right]^+ = 1$$

找 $\lambda^*$ 使得

$$\sum_{i} \left[\hat{y}_i - \frac{\lambda^*}{2}\right]^+ = 1$$

（这总是可能的。）

然后 $x_i^*(\lambda^*) = [\hat{y}_i - \lambda^*/2]^+$。

因此，要将 $\hat{y}$ 投影到概率单纯形上，找 $\lambda^*$ 使得 $\sum_{i} [\hat{y}_i - \lambda^*/2]^+ = 1$，然后投影 $\pi$ 为

$$\pi_i = \left[\hat{y}_i - \frac{\lambda^*}{2}\right]^+$$

**精讲内容**：

这一页给出了概率单纯形投影的最终公式，包括如何确定拉格朗日乘子 $\lambda^*$。

**投影公式**。给定 $\hat{y}$，单纯形投影为：

1. 找 $\lambda^*$ 使得 $\sum_i [\hat{y}_i - \lambda^*/2]^+ = 1$。
2. $\pi_i = [\hat{y}_i - \lambda^*/2]^+$。

**如何找 $\lambda^*$**？$h(\lambda) = \sum_i [\hat{y}_i - \lambda/2]^+$ 是 $\lambda$ 的连续、分段线性、严格单调递减函数（在 $h > 0$ 的范围内）。具体来说：
- 当 $\lambda \to -\infty$ 时，所有 $[\hat{y}_i - \lambda/2]^+ = \hat{y}_i - \lambda/2 \to +\infty$，所以 $h(\lambda) \to +\infty$。
- 当 $\lambda$ 足够大时，所有 $[\hat{y}_i - \lambda/2]^+ = 0$，$h(\lambda) = 0$。

由介值定理，一定存在唯一的 $\lambda^*$ 使得 $h(\lambda^*) = 1$。

**计算 $\lambda^*$ 的高效算法**。虽然没有像欧几里得球投影那样的闭式解（欧几里得球直接解出了 $\lambda^* = \|\hat{y}\|/R - 1$），但 $\lambda^*$ 可以通过以下方法高效计算：

1. **排序法**（$O(n\log n)$）：将 $\hat{y}_1, \ldots, \hat{y}_n$ 排序，然后逐段检查哪些坐标被截断到 0。具体来说，令 $\hat{y}_{(1)} \geq \hat{y}_{(2)} \geq \cdots \geq \hat{y}_{(n)}$，找最大的 $k$ 使得 $\hat{y}_{(k)} - \lambda^*/2 > 0$，其中 $\lambda^*/2 = (\sum_{j=1}^k \hat{y}_{(j)} - 1)/k$。
2. **二分搜索法**（$O(n\log(1/\epsilon))$）：$h(\lambda)$ 是关于 $\lambda$ 的单调递减函数，可以对 $\lambda$ 做二分搜索。

**与欧几里得球投影的对比**：
| 特性 | 欧几里得球投影 | 概率单纯形投影 |
|------|------------|------------|
| 拉格朗日乘子 $\lambda^*$ | 闭式解 $R/\|\hat{y}\| - 1$ | 需要解方程 $\sum[\hat{y}_i - \lambda^*/2]^+ = 1$ |
| 投影公式 | $\pi = R\hat{y}/\|\hat{y}\|$（统一缩放） | $\pi_i = [\hat{y}_i - \lambda^*/2]^+$（平移截断） |
| 截断效应 | 无 | 部分坐标可能被截断到 0 |
| 计算复杂度 | $O(n)$ | $O(n\log n)$（排序）或 $O(n)$（线性时间算法） |

**教授口吻讲解**：

公式出来了：$\pi_i = [\hat{y}_i - \lambda^*/2]^+$。直觉上，这就是"所有坐标减去同一个常数，然后截断到非负，使得总和恰好为 1"。$\lambda^*$ 就是那个"恰好"的常数。怎么找它？可以排序后逐段检查，$O(n\log n)$——不如 Box 和球投影的 $O(n)$ 快，但也相当高效了。实际上存在 $O(n)$ 的线性时间算法，不过排序法已经足够实用。

### 第 ⑩ 页：概率单纯形投影的几何示意

**原文翻译**：

> **[图示——上方]** 二维概率单纯形（从 $(1,0)$ 到 $(0,1)$ 的线段）。点 $\hat{y}$ 位于第一象限中单纯形上方，投影 $\Pi_\mathbb{X}(\hat{y})$ 在单纯形边上，为 $\hat{y}$ 到单纯形的垂直投影。
>
> **[图示——下方]** 点 $\hat{y}$ 在 $x_1$ 轴附近（靠近 $(1,0)$ 端），投影 $\Pi_\mathbb{X}(\hat{y})$ 在单纯形靠近 $x_1$ 顶点的位置。

**精讲内容**：

这一页通过二维图示展示了概率单纯形投影的两种典型情况。

**情况一（上方图）：$\hat{y}$ 在单纯形正上方**。此时 $\hat{y}$ 的两个坐标都为正，且 $\hat{y}_1 + \hat{y}_2 > 1$。投影结果 $\Pi_\mathbb{X}(\hat{y})$ 在单纯形的内部（线段的中间部分），两个坐标都大于 0。从几何上看，这是 $\hat{y}$ 到线段 $\{x_1 + x_2 = 1, x_1 \geq 0, x_2 \geq 0\}$ 的欧几里得最近点。由于线段的法方向是 $(1,1)/\sqrt{2}$，投影就是沿这个方向将 $\hat{y}$ "推"到线段上——即两个坐标各减去 $(\hat{y}_1 + \hat{y}_2 - 1)/2$。

这对应于 $\lambda^*/2 = (\hat{y}_1 + \hat{y}_2 - 1)/2$，两个坐标都未被截断到 0。

**情况二（下方图）：$\hat{y}$ 偏向某个坐标轴**。比如 $\hat{y}_1$ 大而 $\hat{y}_2$ 小（甚至为负）。此时垂直投影到 $x_1 + x_2 = 1$ 的直线上可能得到 $x_2 < 0$ 的点——这不满足非负约束。所以投影结果被"推"到单纯形的顶点附近，$x_2 = 0$，$x_1 = 1$（或接近顶点的位置）。

这对应于 $[\hat{y}_2 - \lambda^*/2]^+ = 0$——第二个坐标被截断到 0。

**两种情况的统一理解**。$\lambda^*$ 的大小决定了有多少坐标被截断。当 $\hat{y}$ 的各坐标比较均匀时，$\lambda^*$ 较小，没有坐标被截断——投影是简单的"等量减少"。当 $\hat{y}$ 的某些坐标很小时，$\lambda^*$ 较大，这些小坐标被截断到 0——投影结果变得"稀疏"。

**稀疏性的 ML 意义**。在分类问题中，如果模型对某些类别的预测 logit 很小，投影到概率单纯形后这些类别的概率会被设为 0。这与 sparsemax（softmax 的稀疏替代）的行为一致——sparsemax 就是基于单纯形投影的。

**教授口吻讲解**：

看二维的例子最清楚。单纯形就是一条线段。如果 $\hat{y}$ 在线段"正上方"——两个坐标都正——那投影是简单的垂直投影。但如果 $\hat{y}$ 偏到一边，垂直投影可能跑到线段外面（$x_2 < 0$），这时候非负约束就起作用了，把你"拽"到线段的端点附近。这就是截断效应。在高维里，概率单纯形是一个 $(n-1)$ 维的多面体，有顶点、边、面——投影可能落在内部、某个面上、某条边上、甚至某个顶点上，取决于 $\hat{y}$ 的结构。好了，下节课——最后一讲——我们要处理 L1-ball 的投影。那里会用到今天的单纯形投影作为子程序。

1. **拉格朗日方法**是处理耦合等式约束的通用工具：将等式约束"吸收"到目标函数中，使问题变得可分离，然后通过选择合适的乘子 $\lambda^*$ 使约束自动满足。
2. **欧几里得球投影** $\pi = R\hat{y}/\|\hat{y}\|$：方向不变，仅缩放模长到球面。$O(n)$ 计算。直接对应深度学习中的梯度裁剪。
3. **概率单纯形投影** $\pi_i = [\hat{y}_i - \lambda^*/2]^+$：各坐标减去统一门槛后截断到非负，门槛 $\lambda^*/2$ 通过 $\sum \pi_i = 1$ 确定。$O(n\log n)$ 计算。
4. **截断效应**是非负约束的体现：不同于欧几里得球投影的"统一缩放"，单纯形投影会将某些坐标"归零"——这是稀疏性的来源。
5. **一条清晰的难度递增线**：Box 投影（完全可分离，$O(n)$）→ 欧几里得球投影（拉格朗日使之可分离，$\lambda^*$ 有闭式解，$O(n)$）→ 概率单纯形投影（拉格朗日 + 截断，$\lambda^*$ 需解方程，$O(n\log n)$）。
