### 第 ① 页：Logistic Regression（回顾）

**原文翻译**：

考虑一个两类分类问题，数据集为 $\{(x_j, y_j) : j = 1, \ldots, M\}$，其中 $x_j \in \mathbb{R}^n$，$y_j \in \{+1, -1\}$。

我们的预测函数形如：

$$\phi(x_j, w) = \frac{1}{1 + e^{w^\top x_j}}$$

其中 $\phi(x, w)$ 可解释为特征向量 $x_j$ 的数据点属于类 $+1$ 的概率。

$1 - \phi(x_j, w) = \frac{e^{w^\top x_j}}{1 + e^{w^\top x_j}}$ 是特征向量 $x_j$ 的数据点属于类 $-1$ 的概率。

**精讲内容**：

本页是 Lecture 01 最后一页的简要回顾，但值得在此补充一些 Lecture 01 未展开的数学细节。

逻辑回归的预测函数 $\phi(x, w) = \frac{1}{1 + e^{w^\top x}}$ 本质上是对**对数几率（log-odds）** 的建模。定义几率（odds）为正类概率与负类概率之比：
$$\text{odds} = \frac{P(y=+1|x)}{P(y=-1|x)} = \frac{\phi(x,w)}{1-\phi(x,w)} = \frac{1/(1+e^{w^\top x})}{e^{w^\top x}/(1+e^{w^\top x})} = e^{-w^\top x}$$

取对数得到 **log-odds**：
$$\ln \frac{P(y=+1|x)}{P(y=-1|x)} = -w^\top x$$

这揭示了逻辑回归的本质：**它假设对数几率是特征的线性函数**。决策边界 $w^\top x = 0$ 对应 odds = 1，即两类概率相等。$w^\top x > 0$ 时正类概率低于 0.5（在这个符号约定下），$w^\top x < 0$ 时正类概率高于 0.5。

注意教授这里使用的符号约定 $\phi(x, w) = \frac{1}{1 + e^{w^\top x}}$（指数项为正）与更常见的 $\sigma(z) = \frac{1}{1 + e^{-z}}$ 差一个负号。这只是约定不同——等价于对参数 $w$ 取反。在阅读不同教材时，要注意这种约定差异，避免混淆。

从优化的角度，最重要的性质是：逻辑回归的预测函数 $\phi$ 关于参数 $w$ 是**无穷次可微的（smooth）**，这保证了我们可以使用梯度方法来优化。具体来说，$\phi$ 关于 $w$ 的梯度为：
$$\nabla_w \phi(x, w) = -\phi(x,w)(1-\phi(x,w)) \cdot x$$

这个优美的形式（预测值乘以 $1-$ 预测值再乘以输入）使得梯度计算极其高效——不需要额外的矩阵运算。

**教授口吻讲解**：

上一讲结束时我们引入了逻辑回归但没有推导完。今天我们把它推导完整，然后做两步推广。你把这页当做"回忆上节课"就好。重点记住：逻辑回归的核心假设是"对数几率是线性的"——这一条假设决定了整个模型的形式。如果你问"为什么是 Sigmoid？"，答案就是：因为 Sigmoid 是唯一满足"log-odds 线性"假设的概率函数。不是拍脑袋选的。

### 第 ② 页：对数似然推导

**原文翻译**：

| 数据点 # | 特征向量 | 标签 | 似然 |
|:---:|:---:|:---:|:---:|
| 1 | $x_1$ | $+1$ | $\frac{1}{1 + e^{w^\top x_1}}$ |
| 2 | $x_2$ | $-1$ | $\frac{e^{w^\top x_2}}{1 + e^{w^\top x_2}}$ |
| 3 | $x_3$ | $-1$ | $\frac{e^{w^\top x_3}}{1 + e^{w^\top x_3}}$ |
| $\vdots$ | $\vdots$ | $\vdots$ | $\vdots$ |
| $M$ | $x_M$ | $+1$ | $\frac{1}{1 + e^{w^\top x_M}}$ |

数据集的似然：
$$L(w) = \frac{1}{1 + e^{w^\top x_1}} \cdot \frac{e^{w^\top x_2}}{1 + e^{w^\top x_2}} \cdot \frac{e^{w^\top x_3}}{1 + e^{w^\top x_3}} \cdots \frac{1}{1 + e^{w^\top x_M}}$$

对数似然：
$$\ln L(w) = -\ln(1 + e^{w^\top x_1}) + w^\top x_2 - \ln(1 + e^{w^\top x_2}) + w^\top x_3 - \ln(1 + e^{w^\top x_3}) - \cdots - \ln(1 + e^{w^\top x_M})$$

$$= -\sum_{j=1}^{M} \ln(1 + e^{w^\top x_j}) + \sum_{j=1}^{M} \mathbb{1}(y_j = -1) \cdot w^\top x_j$$

最大化对数似然：
$$\max_{w \in \mathbb{R}^n} \left\{ -\sum_{j=1}^{M} \ln(1 + e^{w^\top x_j}) + \sum_{j=1}^{M} \mathbb{1}(y_j = -1) \cdot w^\top x_j \right\}$$

**精讲内容**：

这一页完成了逻辑回归从概率模型到优化问题的**完整推导链**。让我逐步拆解。

**第一步：写出每个数据点的似然**。

对于标签 $y_j = +1$ 的数据点：
$$P(y_j = +1 | x_j, w) = \frac{1}{1 + e^{w^\top x_j}}$$

取对数：$\ln P = -\ln(1 + e^{w^\top x_j})$

对于标签 $y_j = -1$ 的数据点：
$$P(y_j = -1 | x_j, w) = \frac{e^{w^\top x_j}}{1 + e^{w^\top x_j}}$$

取对数：$\ln P = w^\top x_j - \ln(1 + e^{w^\top x_j})$

注意两种情况的对数似然都包含 $-\ln(1 + e^{w^\top x_j})$ 这一项，区别仅在于 $y_j = -1$ 时多出一项 $w^\top x_j$。

**第二步：合并为紧凑表达式**。

利用指示函数 $\mathbb{1}(y_j = -1)$，可以将两种情况统一写为：
$$\ln P(y_j | x_j, w) = -\ln(1 + e^{w^\top x_j}) + \mathbb{1}(y_j = -1) \cdot w^\top x_j$$

对所有 $M$ 个数据点求和：
$$\ln L(w) = -\sum_{j=1}^M \ln(1 + e^{w^\top x_j}) + \sum_{j=1}^M \mathbb{1}(y_j = -1) \cdot w^\top x_j$$

**第三步：建立优化问题**。

$$\max_{w \in \mathbb{R}^n} \ln L(w)$$

这是一个**无约束优化问题**——没有对 $w$ 的显式约束。但目标函数是非线性的（包含 $\ln(1 + e^{w^\top x_j})$ 和 $w^\top x_j$ 的混合项）。

从凸优化的视角看，这个目标函数有一个极为重要的性质：**负对数似然 $-\ln L(w)$ 是凸函数**。这是因为：

1. $\ln(1 + e^z)$（log-sum-exp 的特例）是 $z$ 的凸函数。证明：二阶导数 $\frac{e^z}{(1+e^z)^2} = \sigma(z)(1-\sigma(z)) > 0$。
2. 凸函数的非负加权和仍然是凸函数。
3. $-w^\top x_j$ 是线性的，因此也是凸函数。

所以 $-\ln L(w)$ 是凸函数，等价地，$\ln L(w)$ 是**凹函数**。凹函数的最大化等价于凸函数的最小化——这意味着逻辑回归的 MLE 问题是一个**凸优化问题**。

回顾 Lecture 01 的教训：非线性优化可能失败（盒子问题初始点 $(0,0)$）。但凸优化不会！凸性保证了**任何局部最优点都是全局最优点**，梯度下降只要步长合适就一定能收敛到最优解。这就是逻辑回归在实践中如此稳定可靠的数学根源。

**教授口吻讲解**：

推导的关键步骤是从乘积（似然）到求和（对数似然）。为什么这很重要？你想想，$M$ 可能是一百万。一百万个 0 到 1 之间的数相乘？在计算机里直接变成零了——这叫数值下溢。取对数之后变成一百万个负数相加——完全没问题。而且对数还有个好处：乘积的导数（链式法则，极其复杂）变成了求和的导数（逐项求导，非常简单）。所以 $\ln$ 这一步不仅是数值上的必要手段，也是数学上的极大简化。你以后每次看到 Cross-Entropy Loss，记住它的原型就是这个对数似然。

### 第 ③ 页：Multi-Class Logistic Regression

**原文翻译**：

考虑一个多类分类问题，数据集形如 $\{(x_j, y_j) : j = 1, \ldots, M\}$，其中 $x_j \in \mathbb{R}^n$，$y_j \in \{1, \ldots, K\}$。

我们的预测函数形如：

$$\phi_k(x_j, w_1, \ldots, w_K) = \frac{e^{w_k^\top x_j}}{\sum_{\ell=1}^{K} e^{w_\ell^\top x_j}}$$

其中 $\phi_k(x_j, w_1, \ldots, w_K)$ 是特征向量 $x_j$ 的数据点属于第 $k$ 类的概率。

我们使用最大似然估计来估计参数 $(w_1, w_2, \ldots, w_K) \in \mathbb{R}^{n \times K}$。

**精讲内容**：

这一页实现了第一次推广：**从二分类到多类分类**。核心是引入 **Softmax 函数**。

**Softmax 函数** 是 Sigmoid 的多类推广。给定 $K$ 个类别的线性得分 $z_k = w_k^\top x$（$k = 1, \ldots, K$），Softmax 将它们转化为概率分布：

$$\text{softmax}(z)_k = \frac{e^{z_k}}{\sum_{\ell=1}^K e^{z_\ell}}$$

Softmax 的关键性质：

1. **非负性**：$e^{z_k} > 0$，所以 $\text{softmax}(z)_k > 0$
2. **归一化**：$\sum_{k=1}^K \text{softmax}(z)_k = 1$（所有类概率之和为 1）
3. **单调性**：$z_k$ 越大，$\text{softmax}(z)_k$ 越大（得分高的类概率高）
4. **极限行为**：当某个 $z_k$ 远大于其他所有 $z_\ell$ 时，$\text{softmax}(z)_k \to 1$——Softmax 逼近 argmax

注意参数的结构：二分类逻辑回归只有一个参数向量 $w \in \mathbb{R}^n$；多类逻辑回归有 $K$ 个参数向量 $w_1, \ldots, w_K \in \mathbb{R}^n$，总参数量为 $nK$。每个 $w_k$ 可以理解为"第 $k$ 类的代表方向"——数据点 $x_j$ 与 $w_k$ 的内积 $w_k^\top x_j$ 越大，$x_j$ 越可能属于第 $k$ 类。

当 $K = 2$ 时，Softmax 退化为 Sigmoid。具体地：
$$\frac{e^{w_1^\top x}}{e^{w_1^\top x} + e^{w_2^\top x}} = \frac{1}{1 + e^{(w_2 - w_1)^\top x}} = \sigma(-(w_2 - w_1)^\top x)$$

所以二分类只需要一个参数向量 $w = w_2 - w_1$（差向量），而多类分类需要 $K$ 个独立的参数向量。

Softmax 与 $\arg\max$ 的关系值得深思：$\arg\max$ 是"硬"选择（选得分最高的类），Softmax 是"软"选择（给每个类一个概率权重）。"Softmax" 这个名字正是因为它是 $\max$ 函数的"软化"（smooth approximation）版本。数学上：
$$\ln\left(\sum_k e^{z_k}\right) \approx \max_k z_k$$
当所有 $z_k$ 的差距很大时，近似越精确。

**教授口吻讲解**：

从二分类到多类，数学上发生了什么？Sigmoid 变成了 Softmax——一个参数向量变成了 $K$ 个参数向量。为什么？因为二分类时你只需要一条"分界线"（$w^\top x = 0$），但 $K$ 类分类你需要 $K$ 条——每条线把一个类和其余类分开。Softmax 的名字很有意思：它是 soft 版的 max。你想选概率最大的类？那是 argmax，不可微。你想"差不多选概率最大的类，但也给其他类留点概率"？那就是 Softmax，可微。可微才能做梯度下降，这就是为什么 Softmax 在深度学习中无处不在。

### 第 ④ 页：多类逻辑回归的似然与对数似然

**原文翻译**：

| 数据点 | 特征向量 | 标签 | 似然 |
|:---:|:---:|:---:|:---:|
| 1 | $x_1$ | 3 | $\frac{e^{w_3^\top x_1}}{\sum_{\ell=1}^K e^{w_\ell^\top x_1}}$ |
| 2 | $x_2$ | 1 | $\frac{e^{w_1^\top x_2}}{\sum_{\ell=1}^K e^{w_\ell^\top x_2}}$ |
| 3 | $x_3$ | 4 | $\frac{e^{w_4^\top x_3}}{\sum_{\ell=1}^K e^{w_\ell^\top x_3}}$ |
| $\vdots$ | $\vdots$ | $\vdots$ | $\vdots$ |
| $M$ | $x_M$ | 3 | $\frac{e^{w_3^\top x_M}}{\sum_{\ell=1}^K e^{w_\ell^\top x_M}}$ |

似然函数（各数据点似然之积）：

$$L(w_1, \ldots, w_K) = \prod_{j=1}^M \frac{e^{w_{y_j}^\top x_j}}{\sum_{\ell=1}^K e^{w_\ell^\top x_j}}$$

对数似然：

$$\ln L = \sum_{j=1}^M \sum_{k=1}^K \mathbb{1}(y_j = k) \, w_k^\top x_j - \sum_{j=1}^M \ln\left(\sum_{\ell=1}^K e^{w_\ell^\top x_j}\right)$$

**精讲内容**：

这一页将似然函数的推导从二分类推广到了多类。推导过程完全平行，但对数似然的最终形式更加优雅。

**似然的构建逻辑**：对于数据点 $j$（标签 $y_j = k$），它被正确分类的概率是：
$$P(y_j = k | x_j, w_1, \ldots, w_K) = \frac{e^{w_k^\top x_j}}{\sum_{\ell=1}^K e^{w_\ell^\top x_j}}$$

假设数据点独立，整个数据集的似然是各数据点似然之积。

**对数似然的推导**：

对第 $j$ 个数据点（$y_j = k$）：
$$\ln P(y_j | x_j) = \ln e^{w_k^\top x_j} - \ln\left(\sum_{\ell=1}^K e^{w_\ell^\top x_j}\right) = w_k^\top x_j - \ln\left(\sum_{\ell=1}^K e^{w_\ell^\top x_j}\right)$$

使用指示函数统一各类：
$$\ln P(y_j | x_j) = \sum_{k=1}^K \mathbb{1}(y_j = k) \, w_k^\top x_j - \ln\left(\sum_{\ell=1}^K e^{w_\ell^\top x_j}\right)$$

对所有 $M$ 个数据点求和得到最终的对数似然。

第二项 $-\sum_j \ln(\sum_\ell e^{w_\ell^\top x_j})$ 就是著名的 **log-sum-exp** 函数的负值。log-sum-exp 在优化理论中有一个极其重要的地位：它是**凸函数**，而且是 $\max$ 函数的光滑近似。

因此负对数似然：
$$-\ln L = \sum_{j=1}^M \ln\left(\sum_{\ell=1}^K e^{w_\ell^\top x_j}\right) - \sum_{j=1}^M \sum_{k=1}^K \mathbb{1}(y_j = k) \, w_k^\top x_j$$

是关于 $(w_1, \ldots, w_K)$ 的**凸函数**（log-sum-exp 是凸的，减去线性项仍然是凸的）。这意味着多类逻辑回归的 MLE 仍然是凸优化问题——全局最优解有保证。

如果你学过深度学习，你会认出这个负对数似然就是 **Cross-Entropy Loss**：
$$\text{CE} = -\sum_{j=1}^M \sum_{k=1}^K \mathbb{1}(y_j = k) \ln\left(\frac{e^{w_k^\top x_j}}{\sum_\ell e^{w_\ell^\top x_j}}\right)$$

展开后与上面的负对数似然完全相同。所以 Cross-Entropy Loss 就是 Softmax 回归的负对数似然——两者是同一个东西的不同名字。

**教授口吻讲解**：

你注意到没有，对数似然恰好分成了两部分：第一部分 $\sum \mathbb{1}(y_j=k) w_k^\top x_j$ 是"正确类别的线性得分之和"，第二部分 $-\sum \ln(\sum e^{w_\ell^\top x_j})$ 是 log-sum-exp 的惩罚。直觉上，我们想让正确类别的得分 $w_{y_j}^\top x_j$ 尽量大（第一项），但又不能让所有得分都无限大——log-sum-exp 项会惩罚这种"普遍膨胀"。这就像考试：你想让自己的分数高（第一项），但全班平均分也在涨（第二项），你要的不是绝对分数高，而是"你的分数显著高于平均"——这正是 Softmax 在做的事。

### 第 ⑤ 页：多类逻辑回归最终优化问题

**原文翻译**：

$$\max_{(w_1, \ldots, w_K) \in \mathbb{R}^{n \times K}} \left\{ \sum_{j=1}^{M} \sum_{k=1}^{K} \mathbb{1}(y_j = k) \, w_k^\top x_j - \sum_{j=1}^{M} \ln\left(\sum_{\ell=1}^{K} e^{w_\ell^\top x_j}\right) \right\}$$

**精讲内容**：

这一页将多类逻辑回归正式写成了一个**优化问题**。虽然只有一个公式，但它浓缩了前面所有推导的精华。

**问题结构分析**：

| 要素 | 具体形式 | 维度 |
|------|---------|------|
| 决策变量 | $(w_1, \ldots, w_K)$ | $\mathbb{R}^{n \times K}$（共 $nK$ 个标量参数） |
| 目标函数 | 对数似然 $\ln L(w_1, \ldots, w_K)$ | $\mathbb{R}^{nK} \to \mathbb{R}$ |
| 约束 | 无 | 无约束优化 |
| 凸性 | 凹目标（等价于凸最小化） | 全局最优有保证 |

参数总量为 $nK$。对于一个有 1000 个特征、10 个类别的问题，参数量为 10000——线性回归级别。但当 $n$ 和 $K$ 都很大时（比如 ImageNet：$n$ = 图片像素数 = 150528，$K$ = 1000 个类别），直接用多类逻辑回归需要 1.5 亿个参数，而且模型是线性的——远远不够。这就是为什么需要深度学习。

这个优化问题的**梯度**可以显式写出。对 $w_k$ 求梯度：
$$\nabla_{w_k} \ln L = \sum_{j=1}^M \left[\mathbb{1}(y_j = k) - \frac{e^{w_k^\top x_j}}{\sum_\ell e^{w_\ell^\top x_j}}\right] x_j = \sum_{j=1}^M \left[\mathbb{1}(y_j = k) - \phi_k(x_j, w)\right] x_j$$

这个梯度的形式极其优美：它是"真实标签 $\mathbb{1}(y_j = k)$ 减去预测概率 $\phi_k$"与特征 $x_j$ 的加权和。**预测越准确，残差越小，梯度越小，更新幅度越小**——自动实现了"学得好就少改"的自适应行为。

这个梯度公式将在 Lecture 03-04 的梯度下降中被直接使用。

**教授口吻讲解**：

这个公式就是多类逻辑回归的全部——一个漂亮的凸优化问题。但我要你想一个问题：这个模型好用吗？答案是：取决于特征 $x_j$。如果特征设计得好（比如手工提取的 SIFT 特征），多类逻辑回归其实效果不错。但如果特征就是原始像素？那基本没用——因为"像素的线性组合"根本抓不住图像中的复杂模式。怎么办？让机器自己学特征——这就是深度学习的出发点。我们下一页就来讲。

### 第 ⑥ 页：数值示例

**原文翻译**：

特征为每个数据点的（收入, 受教育年数）。

| 数据点 | 特征向量 | 标签 | 似然 |
|:---:|:---:|:---:|:---:|
| 1 | (50K, 11) | A | $\frac{e^{0.1 \times 50 + 0.5 \times 11}}{e^{0.1 \times 50 + 0.5 \times 11} + e^{0.4 \times 50 + 0.1 \times 11}}$ |
| 2 | (150K, 16) | B | $\frac{e^{0.4 \times 150 + 0.1 \times 16}}{e^{0.1 \times 150 + 0.5 \times 16} + e^{0.4 \times 150 + 0.1 \times 16}}$ |

参数向量：$(w_A, w_B) = ((0.1, 0.5), (0.4, 0.1))$

**精讲内容**：

这个数值示例帮助建立 Softmax 的具体计算直觉。

**数据解读**：

- 数据点 1：收入 50K、受教育 11 年的人，属于类别 A
- 数据点 2：收入 150K、受教育 16 年的人，属于类别 B

两个类别的参数向量揭示了模型对特征的利用方式：

- 类别 A 的权重 $w_A = (0.1, 0.5)$：更重视教育年数（权重 0.5 > 0.1）
- 类别 B 的权重 $w_B = (0.4, 0.1)$：更重视收入（权重 0.4 > 0.1）

**手动计算数据点 1 的似然**：

分子（类别 A 的得分指数）：$e^{0.1 \times 50 + 0.5 \times 11} = e^{5 + 5.5} = e^{10.5}$

分母第一项（类别 A）：$e^{10.5}$
分母第二项（类别 B）：$e^{0.4 \times 50 + 0.1 \times 11} = e^{20 + 1.1} = e^{21.1}$

所以数据点 1 被归为类别 A 的概率：
$$P(A | x_1) = \frac{e^{10.5}}{e^{10.5} + e^{21.1}} = \frac{1}{1 + e^{10.6}} \approx 0.0000$$

这意味着以当前参数，模型几乎把数据点 1 也归为 B 类！尽管真实标签是 A。这告诉我们这组参数 $(w_A, w_B)$ 不是最优的——MLE 会找到更好的参数。

这个例子的教学意义在于：**任意给定一组参数，我们都能计算似然；MLE 的目标是找到使似然最大的那组参数**。参数空间中存在无穷多个候选——优化算法帮我们在这个无穷空间中找到最佳的一个。

**教授口吻讲解**：

这个例子算下来你会发现一个有趣的事：用这组参数，第一个人（低收入、中等教育）被模型判为 B 类的概率接近 100%——但真实标签是 A！这说明什么？这组参数很差。如果我们做 MLE，优化器会自动调整 $w_A$ 和 $w_B$ 使得数据点 1 被归为 A 类的概率变大。这就是"训练"在做的事情——不断调参数，让模型的预测和真实标签越来越一致。

### 第 ⑦ 页：Deep Learning

**原文翻译**：

考虑多类分类问题，数据点 $\{(x_j, y_j) : j = 1, \ldots, M\}$，$x_j \in \mathbb{R}^n$，$y_j \in \{1, 2, \ldots, K\}$。

> [图示说明] 深度神经网络的层级结构图：
>
> **Layer 1:** 输入 $x_j$ → 参数 $(Q^1, g^1)$，其中 $Q^1 \in \mathbb{R}^{d_1 \times n}$, $g^1 \in \mathbb{R}^{d_1}$ → 输出 $a_j^1$
>
> **Layer 2:** 输入 $a_j^1$ → 参数 $(Q^2, g^2)$，其中 $Q^2 \in \mathbb{R}^{d_2 \times d_1}$, $g^2 \in \mathbb{R}^{d_2}$ → 输出 $a_j^2$
>
> **Layer L:** 输入 $a_j^{L-1}$ → 参数 $(Q^L, g^L)$，其中 $Q^L \in \mathbb{R}^{d_L \times d_{L-1}}$, $g^L \in \mathbb{R}^{d_L}$ → 输出 $a_j^L$

各层计算公式：
$$a_j^1 = \sigma_1(Q^1 x_j + g^1), \quad a_j^2 = \sigma_2(Q^2 a_j^1 + g^2)$$

其中 $\sigma$ 可以是 Sigmoid $\sigma(t) = \frac{1}{1+e^t}$ 或 ReLU $\sigma(t) = \max\{t, 0\}$。

**精讲内容**：

这一页实现了第二次推广——也是最关键的推广：**从手工特征到学习特征**。

**核心思想**：多类逻辑回归的局限在于它依赖手工设计的特征 $x_j$。如果原始特征不够好（比如原始像素），模型表现就差。深度学习的解决方案是：**不用手工特征，让神经网络自动学习特征**。

**神经网络的数学结构**：每一层执行"仿射变换 + 非线性激活"：
$$a_j^\ell = \sigma_\ell(Q^\ell a_j^{\ell-1} + g^\ell)$$

其中：
- $Q^\ell \in \mathbb{R}^{d_\ell \times d_{\ell-1}}$：权重矩阵（$d_\ell$ 行 $d_{\ell-1}$ 列）
- $g^\ell \in \mathbb{R}^{d_\ell}$：偏置向量
- $\sigma_\ell$：逐元素激活函数
- $a_j^0 = x_j$（输入层就是原始特征）

**维度变换链**：
$$x_j \in \mathbb{R}^n \xrightarrow{Q^1: d_1 \times n} a_j^1 \in \mathbb{R}^{d_1} \xrightarrow{Q^2: d_2 \times d_1} a_j^2 \in \mathbb{R}^{d_2} \xrightarrow{\cdots} a_j^L \in \mathbb{R}^{d_L}$$

每一层可以改变维度。例如第一层可以将 $n = 784$（28x28 灰度图像像素）映射到 $d_1 = 256$，第二层映射到 $d_2 = 128$，以此类推。

**激活函数的必要性**：如果没有非线性激活（$\sigma$ 是恒等函数），那么多层网络：
$$Q^L(Q^{L-1}(\cdots Q^1 x \cdots)) = (Q^L Q^{L-1} \cdots Q^1) x = Wx$$

退化为一个单层线性变换！$L$ 层和 1 层完全等价。非线性激活函数是深度网络表达能力的**根源**——它使得多层组合能表示远比线性函数复杂得多的映射。

**Sigmoid vs ReLU**：

| 对比 | Sigmoid $\frac{1}{1+e^{-t}}$ | ReLU $\max(t, 0)$ |
|------|------|------|
| 输出范围 | $(0, 1)$ | $[0, +\infty)$ |
| 梯度范围 | $(0, 0.25]$ | $\{0, 1\}$ |
| 梯度消失 | 严重（深层网络） | 不会（正区间梯度恒 1） |
| 计算成本 | 高（指数运算） | 极低（比较运算） |
| 现代使用 | 很少用于隐藏层 | 标准选择 |

**教授口吻讲解**：

深度学习从优化的角度看，做了一件事：**把手工特征 $x_j$ 替换为可学习特征 $z_j = z(x_j, Q^1, g^1, \ldots, Q^L, g^L)$**。你不需要知道什么是好特征——网络自己学。代价是什么？参数量爆炸——每一层的 $Q^\ell$ 和 $g^\ell$ 都是需要优化的参数。而且优化问题不再是凸的——因为 $\sigma$ 的嵌套组合引入了大量非凸性。这就是为什么深度学习训练如此"玄学"——同一个网络结构，换一个初始化、换一个学习率，结果可能天差地别。我们后面几讲的任务就是理解：即使优化问题不是凸的，什么条件下梯度下降仍然能给出好的结果？

### 第 ⑧ 页：深度网络的整体表达式

**原文翻译**：

给定神经网络参数 $(Q^1, g^1, Q^2, g^2, \ldots, Q^L, g^L)$，对应于第一层的输入 $x_j$，最后一层的输出为：

$$\sigma_L\left(Q^L \, \sigma_{L-1}\left(\ldots \sigma_2\left(Q^2 \, \sigma_1(Q^1 x_j + g^1) + g^2\right) \ldots \right) + g^L\right)$$

> [图示说明] 嵌套结构标注：最内层 = "output from layer 1"，次内层 = "output from second layer"，最外层 = "output from last layer"。整个表达式记为 $z(x_j, Q^1, g^1, Q^2, g^2, \ldots, Q^L, g^L)$。

通过将 $z(x_j, Q^1, g^1, \ldots, Q^L, g^L)$ 作为数据点 $j$ 的特征，$y_j$ 作为标签，求解一个多类逻辑回归问题。

**精讲内容**：

这一页将整个神经网络**压缩为一个函数** $z(\cdot)$，然后揭示了深度学习的完整架构：**神经网络特征提取 + Softmax 分类器**。

**嵌套函数的本质**：

整个网络可以写成函数的复合：
$$z = \sigma_L \circ (\text{affine}_L) \circ \sigma_{L-1} \circ (\text{affine}_{L-1}) \circ \cdots \circ \sigma_1 \circ (\text{affine}_1)$$

其中 $\text{affine}_\ell(a) = Q^\ell a + g^\ell$。这种"仿射 + 非线性"的交替组合，正是深度网络的**核心模式**。

**深度学习 = 特征学习 + 分类器**：

教授在这一页明确指出了深度学习的两段式结构：

1. **前端（Feature Extractor）**：神经网络 $z(x_j, Q^1, g^1, \ldots, Q^L, g^L)$ 将原始输入 $x_j \in \mathbb{R}^n$ 映射为**学习到的特征** $z_j \in \mathbb{R}^{d_L}$
2. **后端（Classifier）**：多类逻辑回归 Softmax 利用学习到的特征 $z_j$ 进行分类

这个两段式结构在现代深度学习中无处不在：
- BERT = Transformer 特征提取 + 线性分类头
- ResNet = 卷积特征提取 + 全连接分类头
- GPT = Transformer 特征提取 + Softmax 下一个 token 预测

两段的参数**同时优化**：$(Q^1, g^1, \ldots, Q^L, g^L)$（网络参数）和 $(w_1, \ldots, w_K)$（分类头参数）一起通过梯度下降更新。这就是**端到端训练（end-to-end training）**。

从优化的角度，总参数量为：
$$\underbrace{d_1 n + d_1}_{\text{Layer 1}} + \underbrace{d_2 d_1 + d_2}_{\text{Layer 2}} + \cdots + \underbrace{d_L d_{L-1} + d_L}_{\text{Layer L}} + \underbrace{K d_L}_{\text{分类头}}$$

对于 GPT-3 级别的模型，这个总量约为 1750 亿。优化一个 1750 亿维的非凸函数——这就是深度学习训练的数学本质。

这个嵌套结构也解释了**反向传播（Backpropagation）** 的必要性：对数似然关于网络参数的梯度需要通过链式法则逐层传播。Lecture 03-04 中的梯度下降正是解决这个问题的工具。

**教授口吻讲解**：

你看到了整个画面了吗？一句话总结深度学习：**用神经网络自动学习特征，然后在学到的特征上做 Softmax 分类**。就这么简单。全部的"深度"都在特征学习这一步——网络越深，学到的特征越抽象、越有表达力。但"越深"也意味着优化问题越非凸、越难训练。这就是为什么我们需要后面几讲的理论——梯度下降在什么条件下能有效地训练这种深层网络？步长该怎么选？收敛速率是多少？这些问题，我们从下一讲开始系统地回答。

## 核心收获

**收获一：从似然到优化的推导链是 ML 建模的通用范式**

概率模型 → 似然函数（独立假设下取乘积）→ 对数似然（乘积变求和）→ 最大化对数似然 = 最小化负对数似然（= Cross-Entropy Loss）。这条推导链适用于几乎所有参数化模型，是从统计建模到优化求解的"标准桥梁"。掌握这条链，你就能为任何新模型自动推导出损失函数和优化目标。

**收获二：Softmax 是 Sigmoid 的多类推广，它将 argmax "软化"为可微操作**

Softmax 的核心价值在于将离散的类别选择转化为连续的概率分布，使得梯度下降成为可能。log-sum-exp 的凸性保证了多类逻辑回归仍然是凸优化问题。但当 Softmax 与深度网络结合后，整体问题变为非凸——这是后续课程需要处理的核心挑战。

**收获三：深度学习 = 可学习特征提取器 + Softmax 分类器，训练是一个大规模非凸优化问题**

神经网络将手工特征替换为学习特征，通过多层"仿射 + 非线性"的嵌套实现复杂的非线性映射。端到端训练同时优化特征提取器和分类器的参数。但代价是优化问题变成了高维非凸问题——没有全局最优的理论保证。这正是 Lecture 03 开始引入梯度下降，Lecture 04 引入凸性概念，Lecture 05-09 建立收敛理论的动机所在。
