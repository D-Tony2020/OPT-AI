Lectures 9-11 解决的是**无约束优化**问题——在整个 $\mathbb{R}^n$ 上最小化凸函数。但实际中大量的优化问题是**有约束的**：参数必须非负（$x \geq 0$）、概率必须求和为 1、模型权重必须在某个范围内。

本讲从凸集的基本性质出发（交集的凸性、下水平集的凸性），证明凸约束产生的可行域是凸集，然后引入**投影梯度下降 (Projected Gradient Descent)**——将无约束 GD 推广到约束优化的核心算法。

本讲是从"无约束"到"有约束"的范式转换，也是从"纯理论"到"实际可用"的关键一步。

### 第 1 页：凸集

**原文翻译**：

- **定义**：集合 $C \subseteq \mathbb{R}^n$ 是**凸的**，如果对所有 $x, y \in C$ 和 $\lambda \in [0, 1]$，有 $\lambda x + (1 - \lambda)y \in C$。
- 【图示说明——左图】椭圆形集合（凸集），内部取两点 $x$ 和 $y$，连线段完全在集合内部，凸组合点 $\lambda x + (1-\lambda)y$ 在集合中。标注"convex"。
- 【图示说明——右图】弯月形/豆形集合（非凸集），内部取两点 $x$ 和 $y$，连线段部分在集合外部，凸组合点 $\lambda x + (1-\lambda)y$ 在集合外。标注"not convex"。
- 直觉上，对于凸集，从集合中任意一点到任意另一点都有"直线视线"（line of sight）。

**精讲内容**：

本页回顾了凸集的定义，为后续的约束优化理论做铺垫。

**凸集的定义**：集合 $C$ 是凸的，当且仅当任取 $C$ 中两点 $x, y$，它们之间的整条线段都在 $C$ 内。数学上，$\lambda x + (1-\lambda)y \in C$ 对所有 $\lambda \in [0, 1]$ 成立。

这个定义在 Lec 01-04 中已经出现过，但现在我们需要从更深的角度理解它——因为它将决定约束优化问题的结构。

**几何直觉：直线视线 (Line of Sight)**。在凸集中，从任何一点"看"向任何另一点，中间没有"障碍物"——整条连线都在集合内部。非凸集则可能有"凹陷"或"缺口"，使得某些点之间的连线穿出集合。

**常见的凸集例子**：

| 凸集 | 数学表达 | ML 中的应用 |
|------|---------|-----------|
| 超平面 | $\{x : a^Tx = b\}$ | 线性约束 |
| 半空间 | $\{x : a^Tx \leq b\}$ | 线性不等式约束 |
| 球体 | $\{x : \|x - c\| \leq r\}$ | 权重范数约束（正则化） |
| 多面体 | $\{x : Ax \leq b\}$ | 线性规划的可行域 |
| 概率单纯形 | $\{x : x_i \geq 0, \sum x_i = 1\}$ | 概率分布约束 |
| 正半定锥 | $\{X : X \succeq 0\}$ | 半定规划 |

**非凸集的例子**：整数集 $\mathbb{Z}^n$（两个整数之间的凸组合通常不是整数）、单位球面 $\{x : \|x\| = 1\}$（注意是球面不是球体——球面上两点的连线会穿入球内部）。

凸集为什么对优化重要？因为凸集上的凸优化保留了"局部最优=全局最优"的性质（Lec 09）。如果可行域是非凸的，即使目标函数是凸的，优化也可能变得极其困难（NP-hard）。

**教授口吻讲解**：

凸集——你们在第一、二讲就见过了。今天为什么又提起？因为我们要从"无约束优化"转向"约束优化"。约束优化中，你不是在整个 $\mathbb{R}^n$ 上找最优解，而是在一个集合里找。如果这个集合是凸的，加上目标函数是凸的，那前面所有的好性质（局部即全局等）都还在。如果集合不是凸的——那就麻烦了。

所以这一页看起来是回顾，实际上是在为后面的约束优化做基础建设。凸可行域 + 凸目标函数 = 可以高效求解。这是凸优化的核心框架。

### 第 2 页：凸集的交集性质

**原文翻译**：

- 如果 $C_1, C_2, \ldots, C_\ell$ 都是凸集，则 $C_1 \cap C_2 \cap \cdots \cap C_\ell$ 也是凸集。
- 【图示说明】三个凸集 $C_1, C_2, C_3$ 的 Venn 图。三者的交集（红色斜线区域）也是凸的。
- 旁注：a bunch of convex sets 的交集仍然是凸的。
- **证明**：取任意 $x, y \in C_1 \cap C_2 \cap \cdots \cap C_\ell$ 和 $\lambda \in [0, 1]$，要证 $\lambda x + (1-\lambda)y \in C_1 \cap \cdots \cap C_\ell$。
  - 因为 $x \in C_1 \cap \cdots \cap C_\ell$，所以 $x \in C_1$。类似 $y \in C_1$。因为 $C_1$ 是凸的，$\lambda x + (1-\lambda)y \in C_1$。
  - 类似地，$\lambda x + (1-\lambda)y \in C_2, \ldots, C_\ell$。
  - 因此 $\lambda x + (1-\lambda)y \in C_1 \cap \cdots \cap C_\ell$。证毕。

**精讲内容**：

本页证明了一个基本但重要的性质：**凸集的交集仍然是凸集**。这个结果是约束优化理论的基石。

**定理**：如果 $C_1, C_2, \ldots, C_\ell$ 都是凸集，则 $C_1 \cap C_2 \cap \cdots \cap C_\ell$ 也是凸集。

证明的逻辑结构非常清晰：

1. 取交集中任意两点 $x, y$
2. 因为 $x, y$ 在交集中，它们在**每一个** $C_i$ 中
3. 因为每个 $C_i$ 是凸的，凸组合 $\lambda x + (1-\lambda)y$ 在每个 $C_i$ 中
4. 因此凸组合在所有 $C_i$ 的交集中

这个证明虽然简单，但蕴含着深刻的结构性意义。

**重要推论**：这个结果可以推广到**任意（可能无穷）个**凸集的交集。如果 $\{C_\alpha\}_{\alpha \in I}$ 是一族凸集（$I$ 可以是无穷集），则 $\bigcap_{\alpha \in I} C_\alpha$ 也是凸集。证明完全相同。

**反例：凸集的并集通常不是凸集**。例如 $C_1 = \{x : x \leq 0\}$ 和 $C_2 = \{x : x \geq 1\}$ 都是凸集，但 $C_1 \cup C_2$ 不是凸集（0 和 1 都在并集中，但 0.5 不在）。这说明"交集保持凸性"是一个特殊的好性质。

**为什么这个结果重要？** 因为在约束优化中，每个约束定义一个可行集，总的可行域是所有约束可行集的**交集**。如果每个约束的可行集是凸的，那么总的可行域也是凸的——这正是下一页要展开的内容。

**教授口吻讲解**：

这个证明简单到你可能觉得"这有什么好证的？"但它的重要性在于应用。

想想实际的优化问题——你通常有很多约束：$g_1(x) \leq 0$, $g_2(x) \leq 0$, ..., $g_m(x) \leq 0$。每个约束定义一个集合 $C_i = \{x : g_i(x) \leq 0\}$。可行域是 $C_1 \cap C_2 \cap \cdots \cap C_m$。

今天这个定理告诉你：**如果每个 $C_i$ 是凸的，那可行域就是凸的。** 而下一页会告诉你：**如果 $g_i$ 是凸函数，那 $C_i$ 就是凸的。** 两者结合，你就知道：凸约束函数 → 凸可行域。这是凸优化理论的闭环。

另外注意——**并集不保持凸性**！交集可以，并集不行。这是一个需要记住的不对称性。

### 第 3 页：凸函数的下水平集是凸集

**原文翻译**：

- 设 $f: \mathbb{R}^n \to \mathbb{R}$ 是凸函数，定义 $C = \{x \in \mathbb{R}^n : f(x) \leq 0\}$。则 $C$ 是凸集。
- **证明**：取任意 $x, y \in C$ 和 $\lambda \in [0, 1]$。要证 $\lambda x + (1-\lambda)y \in C$。
  - 因为 $x \in C$，$f(x) \leq 0$。因为 $y \in C$，$f(y) \leq 0$。
  - $f(x) \leq 0 \times \lambda$：$\lambda f(x) \leq 0$
  - $f(y) \leq 0 \times (1-\lambda)$：$(1-\lambda)f(y) \leq 0$
  - 相加：$\lambda f(x) + (1-\lambda)f(y) \leq 0$
  - 由凸性：$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y) \leq 0$
  - 因此 $\lambda x + (1-\lambda)y \in C$。证毕。

**精讲内容**：

本页证明了另一个基本但极其重要的结论：**凸函数的下水平集（sublevel set）是凸集**。

**下水平集**的一般定义：对函数 $f$ 和常数 $\alpha$，$\{x : f(x) \leq \alpha\}$ 称为 $f$ 在水平 $\alpha$ 处的下水平集。本页取 $\alpha = 0$ 是不失一般性的（可以把 $f$ 替换为 $f - \alpha$ 来处理一般情况）。

**证明的关键技巧**：两步走。

第一步利用"非正性 + 凸组合"：因为 $f(x) \leq 0$ 和 $f(y) \leq 0$，而 $\lambda, (1-\lambda) \geq 0$，所以 $\lambda f(x) + (1-\lambda)f(y) \leq 0$。

第二步利用"凸性定义"：$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$。

两步结合：$f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y) \leq 0$，所以凸组合点在 $C$ 中。

**注意：下水平集的凸性不要求 $f$ 本身是凸的**——有些非凸函数也可能有凸的下水平集（这种函数称为拟凸函数 / quasiconvex）。但反过来，**凸函数的下水平集一定是凸的**——这就是本定理的内容。

**直觉**：凸函数的"等高线"围出的区域是凸的。想象一个碗——碗底下面的区域（$f(x) \leq c$）总是一个"圆盘"或"椭圆盘"，这些都是凸集。但一个非凸函数——比如有两个谷底的函数——其下水平集可能是两个分离的区域（非凸）。

**与约束优化的连接**：在约束优化 $\min f(x)$ s.t. $g_i(x) \leq 0$ 中，每个约束 $g_i(x) \leq 0$ 定义的可行集 $C_i = \{x : g_i(x) \leq 0\}$ 正好是凸函数 $g_i$ 的 0-下水平集。如果 $g_i$ 是凸的，那 $C_i$ 是凸的。所有 $C_i$ 的交集也是凸的（上一页的定理）。所以：**凸约束函数产生凸可行域**。

**教授口吻讲解**：

证明只有几行，但结论非常有力。让我给你翻译成人话：如果 $f$ 是凸函数，那 $\{x : f(x) \leq 0\}$ 这个集合是凸的。

为什么？两个关键：(1) $f(x) \leq 0$, $f(y) \leq 0$，加权平均 $\lambda f(x) + (1-\lambda)f(y) \leq 0$——两个非正数的凸组合还是非正的。(2) 凸性说 $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$——函数在凸组合点的值不超过加权平均。两者结合，凸组合点处的函数值也 $\leq 0$。

这个结论对约束优化至关重要：每个凸约束 $g_i(x) \leq 0$ 切出一个凸集 $C_i$，所有凸约束的交集还是凸集。这就是为什么"凸约束 + 凸目标 = tractable"。

### 第 4 页：约束优化问题的可行域是凸集

**原文翻译**：

- 为什么前两个结果有用？考虑约束优化问题：

$$\min f(x) \quad \text{s.t.} \quad g_1(x) \leq 0, g_2(x) \leq 0, \ldots, g_m(x) \leq 0$$

其中 $g_1, g_2, \ldots, g_m$ 是凸函数。

- 可行解集为 $C = \{x \in \mathbb{R}^n : g_1(x) \leq 0, g_2(x) \leq 0, \ldots, g_m(x) \leq 0\}$
- 定义 $C_i = \{x \in \mathbb{R}^n : g_i(x) \leq 0\}$。因为 $g_i$ 是凸的，$C_i$ 是凸的（第 3 页结论）。
- $C = C_1 \cap C_2 \cap \cdots \cap C_m$。因为所有 $C_i$ 都是凸的，$C$ 也是凸的（第 2 页结论）。
- 因此，如果优化问题的所有不等式约束的左边都是凸函数，则可行解集是凸集。

**精讲内容**：

本页将前两页的结果组合起来，得出约束优化的核心结论：**凸约束函数产生凸可行域**。

这是一个"搭积木"式的推理：

$$\text{凸函数 } g_i \xrightarrow{\text{第3页}} \text{凸集 } C_i = \{g_i(x) \leq 0\} \xrightarrow{\text{第2页}} \text{凸集 } C = \bigcap C_i$$

每一步都很简单，但组合起来给出了一个强大的结论，是整个凸优化理论的支柱之一。

**约束优化的分类**：

| 类型 | 目标函数 | 约束函数 | 可行域 | 难度 |
|------|---------|---------|--------|------|
| 凸优化 | 凸 | 凸 | 凸集 | Polynomial time |
| 线性规划 | 线性 | 线性 | 多面体（凸） | Polynomial time |
| 二次规划 | 二次凸 | 线性 | 多面体（凸） | Polynomial time |
| 半定规划 | 线性（矩阵） | 半定约束（凸） | 半定锥（凸） | Polynomial time |
| 非凸优化 | 非凸 | 非凸 | 可能非凸 | 通常 NP-hard |

**ML 中的约束优化实例**：

1. **正则化问题**：$\min_x f(x)$ s.t. $\|x\| \leq r$（权重约束）。$g(x) = \|x\| - r$ 是凸的，可行域是球体（凸集）。
2. **概率约束**：$\min_p D_{KL}(p \| q)$ s.t. $p_i \geq 0, \sum p_i = 1$。可行域是概率单纯形（凸集）。
3. **SVM**：$\min_w \|w\|^2$ s.t. $y_i(w^Tx_i + b) \geq 1$。可行域由线性约束定义（凸集）。
4. **LASSO 的约束形式**：$\min_w \|Xw - y\|^2$ s.t. $\|w\|_1 \leq t$。$\ell_1$ 球是凸集。

在所有这些例子中，约束函数是凸的，因此可行域是凸的，投影梯度下降等凸优化算法可以高效求解。

**教授口吻讲解**：

好，现在把第 2 页和第 3 页串起来。

你有一个优化问题：$\min f(x)$ s.t. $g_1(x) \leq 0, \ldots, g_m(x) \leq 0$。如果所有 $g_i$ 都是凸函数，那每个约束切出一个凸集 $C_i$（第 3 页），所有约束的交集 $C$ 还是凸集（第 2 页）。

这就是凸优化的"黄金法则"：**凸目标 + 凸约束 = 可行域是凸集 = 好优化**。你的 ML 模型中，只要损失函数和约束都是凸的，你就可以放心用梯度下降（加上投影）来求解——保证找到全局最优。

但如果约束不是凸的呢？那可行域可能是非凸的——比如整数约束 $x \in \mathbb{Z}^n$。这就变成了组合优化，通常是 NP-hard 的。所以"凸性"不只是一个数学性质——它是决定问题能不能高效求解的根本分水岭。

### 第 5 页：投影梯度下降——严格凸性

**原文翻译**：

- **定义**：函数 $f: \mathbb{R}^n \to \mathbb{R}$ 是**严格凸的**，如果

$$f(\lambda x + (1-\lambda)y) < \lambda f(x) + (1-\lambda)f(y) \quad \text{对所有 } x, y \in \mathbb{R}^n, x \neq y, \lambda \in (0, 1)$$

- 设 $g: \mathbb{R}^n \to \mathbb{R}$ 是严格凸函数，$\mathbb{X}$ 是凸集。考虑约束优化问题：

$$\min g(x) \quad \text{s.t.} \quad x \in \mathbb{X}$$

则该问题有**唯一**的最优解。

**精讲内容**：

本页引入了**严格凸性 (Strict Convexity)** 的概念，并给出了约束优化中解的唯一性定理。

**严格凸 vs 凸**：

| 性质 | 凸函数 | 严格凸函数 |
|------|--------|-----------|
| 定义 | $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$ | $f(\lambda x + (1-\lambda)y) < \lambda f(x) + (1-\lambda)f(y)$（$x \neq y$） |
| 不等号 | $\leq$（可以取等） | $<$（严格小于，不能取等） |
| 几何意义 | 弦在曲线上方（可以重合） | 弦严格在曲线上方（不重合） |
| 最优解 | 可能有多个（如线性函数在多面体上） | **唯一** |
| 例子 | $f(x) = |x|$（非光滑但凸） | $f(x) = x^2$（二次函数） |

严格凸性比凸性"强一点"——它排除了函数在两点之间是线性的可能。直观地说，严格凸函数的图形是"真正弯曲的碗"，不包含任何平坦的线段。

**唯一性定理**：严格凸函数在凸集上的最小值点是唯一的。

为什么唯一性重要？在投影操作中，我们需要找凸集中离某个点最近的点——这是一个 $\min \|y - x\|^2$ s.t. $y \in \mathbb{X}$ 的问题。$\|y - x\|^2$ 关于 $y$ 是严格凸的，所以投影是唯一定义的——不会出现"最近点有两个"的尴尬情况。

**常见的严格凸函数**：
- $f(x) = \|x\|^2$（欧几里得范数的平方）
- $f(x) = x^TAx$，其中 $A \succ 0$（正定矩阵的二次形式）
- 加了 $\ell_2$ 正则化的任何凸函数 $f(x) + \frac{\lambda}{2}\|x\|^2$（$\lambda > 0$）

**$\ell_2$ 正则化的新理解**：在 ML 中，给损失函数加 $\ell_2$ 正则化（Ridge Regression）不仅防止过拟合，还**保证了解的唯一性**——因为 $f(x) + \frac{\lambda}{2}\|x\|^2$ 是严格凸的（即使 $f$ 只是凸的但不严格凸）。

**教授口吻讲解**：

严格凸性和凸性的区别就在一个"严格"上——不等号从 $\leq$ 变成 $<$。看起来是小改动，但结论变了：**最优解唯一**。

举个例子——$f(x) = |x|$，这是凸的但不是严格凸的（$x$ 和 $-x$ 之间的线段恰好在函数图上）。最小化 $f(x) = |x|$ 在 $[-1, 1]$ 上，最优解是 $x = 0$，恰好唯一。但如果考虑 $f(x) = 0$（常数函数，凸但不严格凸），在 $[-1, 1]$ 上每个点都是最优的——解不唯一。

严格凸函数像一个"真正的碗"——只有一个碗底。不严格凸的函数可能有一个"平底"——平底上每个点都是最优的。

为什么我们现在关心唯一性？因为下面要定义"投影"——把一个点投影到凸集上最近的点。如果最近点有多个，投影就不是一个函数（不确定选哪个）。严格凸性保证最近点唯一，投影是一个well-defined的算子。

### 第 6 页：严格凸函数在凸集上有唯一最优解的证明

**原文翻译**：

**证明**：用反证法。假设 $x^*$ 和 $y^*$（$x^* \neq y^*$）都是最优解，最优目标值为 $z^*$。

- 因为 $x^*$ 是最优解：$x^* \in \mathbb{X}$，$g(x^*) = z^*$。
- 因为 $y^*$ 是最优解：$y^* \in \mathbb{X}$，$g(y^*) = z^*$。
- 因为 $x^* \in \mathbb{X}$，$y^* \in \mathbb{X}$，且 $\mathbb{X}$ 是凸的：$\lambda x^* + (1-\lambda)y^* \in \mathbb{X}$。
- 因为 $g$ 是严格凸的，$g(x^*) = z^*$，$g(y^*) = z^*$：

$$g(\lambda x^* + (1-\lambda)y^*) < \lambda g(x^*) + (1-\lambda)g(y^*) = \lambda z^* + (1-\lambda)z^* = z^*$$

- 因此 $\lambda x^* + (1-\lambda)y^*$ 是可行的（在 $\mathbb{X}$ 中），且目标值严格优于 $z^*$。
- 这与 $x^*, y^*$ 是最优解矛盾。证毕！

**精讲内容**：

本页用反证法证明了严格凸函数在凸集上最优解的唯一性。证明结构清晰而优雅。

**证明的逻辑链条**：

1. **假设有两个最优解** $x^* \neq y^*$，它们的目标值相同为 $z^*$。
2. **构造一个"更好"的可行点**：取凸组合 $c = \lambda x^* + (1-\lambda)y^*$（$\lambda \in (0,1)$）。
3. **可行性**：$c \in \mathbb{X}$（因为 $\mathbb{X}$ 是凸的，$x^*, y^* \in \mathbb{X}$）。
4. **目标值更好**：$g(c) < \lambda g(x^*) + (1-\lambda)g(y^*) = z^*$（严格凸性）。
5. **矛盾**：$c$ 是可行的且目标值 $< z^*$，但 $z^*$ 应该是最优值。

这个证明技巧在凸优化理论中反复出现——**通过凸组合构造新的可行点来推导矛盾**。核心利用了两个性质：凸集的凸组合封闭性（保证可行性）和严格凸性（保证目标值严格改善）。

**如果去掉"严格"会怎样？** 如果 $g$ 只是凸的（不严格凸），那步骤 4 变为 $g(c) \leq z^*$，不是严格小于——没有矛盾。所以唯一性结论不成立。

**对比 Lec 09 的局部最优=全局最优证明**：两个证明都使用了凸组合技巧，但目的不同。Lec 09 证明的是"局部即全局"（不需要严格凸性），本页证明的是"最优解唯一"（需要严格凸性）。

**投影算子的well-definedness**：投影 $\Pi_\mathbb{X}(x) = \arg\min_{y \in \mathbb{X}} \|y - x\|^2$。这里 $g(y) = \|y - x\|^2$ 是关于 $y$ 的严格凸函数（Hessian 为 $2I \succ 0$），$\mathbb{X}$ 是凸集。由本页定理，投影的结果是唯一的——$\Pi_\mathbb{X}(x)$ 是一个well-defined的函数。

**教授口吻讲解**：

反证法。假设有两个最优解——然后取它们的凸组合。凸集保证凸组合还是可行的。严格凸性保证凸组合处的目标值**严格更好**。矛盾！

这个证明你要掌握，因为它出现的频率很高。不只是这个具体定理——"取凸组合构造矛盾"是凸优化中最常用的证明技巧之一。每当你需要证明某个性质在凸集上成立时，试试取两个反例点的凸组合。

现在我们知道了：严格凸 + 凸集 = 唯一最优解。这保证了投影是唯一确定的——我们可以放心定义投影算子了。

### 第 7 页：投影梯度下降的动机

**原文翻译**：

- 当我们对约束优化问题应用梯度下降时，沿负梯度方向移动可能会把我们带到可行域之外。
- 【图示说明——左图】凸集 $\mathbb{X}$（蓝色椭圆）。在集合内部点 $x$ 处，沿负梯度方向走步长 $\gamma$ 后到达点 $x - \gamma\nabla f(x)$，该点在集合**外部**（红色标注）。
- 【图示说明——右图】同样的凸集 $\mathbb{X}$。从 $x$ 处沿负梯度走到 $x - \gamma\nabla f(x)$（在集合外部），然后将其**投影**回集合表面（蓝色虚线箭头），到达集合边界上的点。
- 投影梯度下降适用于约束优化问题。假设当前在点 $x \in \mathbb{X}$，沿负梯度方向走步长 $\gamma$ 到达 $x - \gamma\nabla f(x)$。
- 如果 $x - \gamma\nabla f(x)$ 不在可行域 $\mathbb{X}$ 中，则将其投影回 $\mathbb{X}$——找 $\mathbb{X}$ 中离 $x - \gamma\nabla f(x)$ 最近的点。

**精讲内容**：

本页给出了投影梯度下降 (Projected Gradient Descent, PGD) 的直觉动机。

**核心问题**：在约束优化 $\min_{x \in \mathbb{X}} f(x)$ 中，标准 GD 更新 $x^{k+1} = x^k - \gamma \nabla f(x^k)$ 可能产生一个不在 $\mathbb{X}$ 中的点。即使当前点 $x^k$ 是可行的，走了一步后 $x^{k+1}$ 可能违反约束。

**直觉解决方案**：走完一步后，如果点跑出了可行域，就把它"拉回来"——找可行域中离它最近的点。这就是"投影"操作。

图示完美展示了这个两步过程：
1. **梯度步 (Gradient Step)**：从 $x \in \mathbb{X}$ 出发，沿负梯度走到 $y = x - \gamma\nabla f(x)$（可能在 $\mathbb{X}$ 外）
2. **投影步 (Projection Step)**：将 $y$ 投影回 $\mathbb{X}$，得到 $\Pi_\mathbb{X}(y)$

**为什么投影是合理的？** 投影找的是可行域中"最近的"点——在所有可行点中，它是与 GD 更新点 $y$ 距离最小的。从直觉上说，投影保留了尽可能多的"GD 信息"——它不是随便找一个可行点，而是找最接近 GD 想去的方向的可行点。

**投影的计算开销**：投影操作本身是一个优化问题——$\min_{y \in \mathbb{X}} \|y - x\|^2$。对于某些特殊的可行域，投影有解析解：

| 可行域 $\mathbb{X}$ | 投影公式 | 复杂度 |
|---|---|---|
| 球体 $\|x\| \leq r$ | $\Pi(x) = r \cdot x / \max(\|x\|, r)$ | $O(n)$ |
| 非负象限 $x \geq 0$ | $\Pi(x) = \max(x, 0)$（逐元素） | $O(n)$ |
| 超平面 $a^Tx = b$ | $\Pi(x) = x + \frac{b - a^Tx}{\|a\|^2} a$ | $O(n)$ |
| 概率单纯形 | 排序 + 阈值 | $O(n \log n)$ |
| 一般凸集 | 需要迭代求解 | 可能很昂贵 |

当投影可以高效计算时（如上述前四种情况），PGD 的每步开销几乎与标准 GD 相同。当投影本身需要迭代求解时，PGD 的效率取决于投影问题的求解速度。

**教授口吻讲解**：

投影梯度下降的想法再简单不过了：GD 走了一步，如果走出了可行域，就把它拉回来——找可行域中最近的点。

看左图——你在可行域里面，沿梯度走一步，走出去了。怎么办？看右图——画一条"最短路径"回到可行域边界。这就是投影。

实际中的关键问题是：投影好不好算？对球体、非负象限、概率单纯形——有现成的公式，非常快。对一般凸集——投影本身就是一个优化问题，可能很慢。所以 PGD 在实践中是否高效，很大程度上取决于你的约束结构。

好消息是：ML 中最常见的约束（权重范数约束、非负约束、概率约束）都有高效的投影。所以 PGD 在 ML 中非常实用。

### 第 8 页：投影算子与投影梯度下降算法

**原文翻译**：

- 定义 $x$ 到凸集 $\mathbb{X}$ 的**投影**为：

$$\Pi_\mathbb{X}(x) = \arg\min_{y \in \mathbb{X}} \|y - x\|^2$$

- 【图示说明】圆形凸集 $\mathbb{X}$。点 $x$ 在集合外部。从 $x$ 向集合作"垂线"，垂足即为投影 $\Pi_\mathbb{X}(x)$——集合中距 $x$ 最近的点。
- 注意：如果 $x \in \mathbb{X}$，则 $\Pi_\mathbb{X}(x) = x$。
- 投影梯度下降的流程：从初始点 $x^0$ 开始，在每次迭代 $k$ 中执行：

$$y^{k+1} = x^k - \gamma \nabla f(x^k)$$

$$x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$$

**精讲内容**：

本页给出了投影算子的精确数学定义和 PGD 算法的完整描述。

**投影算子 $\Pi_\mathbb{X}$**：

$$\Pi_\mathbb{X}(x) = \arg\min_{y \in \mathbb{X}} \|y - x\|^2$$

这是一个凸优化问题（$\|y - x\|^2$ 关于 $y$ 是严格凸的，$\mathbb{X}$ 是凸集），由第 6 页的定理，解存在且唯一。因此 $\Pi_\mathbb{X}$ 是一个well-defined的映射 $\mathbb{R}^n \to \mathbb{X}$。

**投影的基本性质**：

1. **幂等性**：$\Pi_\mathbb{X}(\Pi_\mathbb{X}(x)) = \Pi_\mathbb{X}(x)$（已经在集合中的点投影不变）
2. **如果 $x \in \mathbb{X}$，则 $\Pi_\mathbb{X}(x) = x$**（可行点的投影就是自身）
3. **非扩张性**：$\|\Pi_\mathbb{X}(x) - \Pi_\mathbb{X}(y)\| \leq \|x - y\|$（投影不会增大两点间距离）

非扩张性特别重要——它保证投影操作不会"放大"误差。如果两个点在投影前离得近，投影后也离得近。这个性质是 PGD 收敛性分析的关键。

**PGD 算法**：

```
输入: 初始点 x⁰ ∈ X, 步长 γ, 迭代次数 K
对 k = 0, 1, ..., K-1:
    y^{k+1} = x^k - γ ∇f(x^k)    [梯度步: 沿负梯度方向走一步]
    x^{k+1} = Π_X(y^{k+1})        [投影步: 拉回可行域]
输出: x^K
```

PGD 与标准 GD 的区别**仅仅是多了一步投影**。如果 $\mathbb{X} = \mathbb{R}^n$（无约束），投影步退化为恒等映射（$\Pi_{\mathbb{R}^n}(y) = y$），PGD 退化为标准 GD。

**PGD 的收敛性**：PGD 的收敛性分析与标准 GD 类似，只需要做适当的修改来考虑投影步。在光滑凸函数 + 凸可行域的设置下，PGD 也可以达到 $O(1/K)$ 的收敛速率。证明中的关键工具是投影的非扩张性——它保证投影步不会破坏 GD 步带来的进展。

**PGD 在 ML 中的应用实例**：

1. **$\ell_2$ 约束训练**：$\min_w L(w)$ s.t. $\|w\| \leq r$。每步 GD 后，如果 $\|w\| > r$，将 $w$ 缩放到球面上：$w \leftarrow r \cdot w / \|w\|$。

2. **非负矩阵分解 (NMF)**：$\min_{W, H \geq 0} \|X - WH\|_F^2$。每步 GD 后，将负元素截断为 0：$W \leftarrow \max(W, 0)$。

3. **稀疏优化**：$\min_w L(w)$ s.t. $\|w\|_1 \leq t$。投影到 $\ell_1$ 球上（需要排序算法）。

4. **Wasserstein GAN 的权重裁剪**：原始 WGAN 将判别器权重裁剪到 $[-c, c]$——这正是投影到超矩形上。

**教授口吻讲解**：

PGD 算法本身很简单——先走一步 GD，再投影回可行域。就这两步。

投影的数学定义是"找可行域中最近的点"。这个定义之所以有意义，是因为前面证明了严格凸函数在凸集上有唯一最优解——所以"最近点"是唯一确定的。

实际使用中你需要关心两件事：第一，投影好不好算——对球体、非负象限这些简单集合，投影有解析解；对复杂集合可能需要迭代求解。第二，收敛速率是多少——对光滑凸函数，PGD 和标准 GD 的速率一样，都是 $O(1/K)$。

PGD 是约束优化中最基本的算法——就像 GD 是无约束优化中最基本的算法一样。后续的更高级方法（近端梯度下降、Frank-Wolfe 等）都可以看作 PGD 的变体或替代。

## 核心收获

**收获一：凸约束函数产生凸可行域——这是凸优化可解性的结构保证**

通过"凸函数的下水平集是凸集"和"凸集的交集是凸集"两个基本定理的组合，我们建立了一条完整的推理链：凸约束函数 → 凸可行集 → 凸可行域。这保证了约束凸优化问题保留了无约束凸优化的所有好性质（局部即全局、GD 类算法可用等），是整个凸优化理论的结构基础。

**收获二：严格凸性保证最优解唯一——这使得投影操作well-defined**

严格凸性是比凸性更强的条件，它排除了"平底"的情况，保证最优解唯一。这个性质的直接应用是投影算子的well-definedness——$\Pi_\mathbb{X}(x) = \arg\min_{y \in \mathbb{X}} \|y-x\|^2$ 有唯一解。在 ML 中，$\ell_2$ 正则化将凸目标函数变为严格凸的，不仅防止过拟合，还保证了解的唯一性。

**收获三：投影梯度下降是从无约束到有约束的自然推广**

PGD = GD + 投影。它保留了 GD 的简单性和收敛性，同时处理了约束。当投影可以高效计算时（球体、非负象限、概率单纯形等），PGD 是约束凸优化的首选算法。PGD 的思想——"先做一步无约束优化，再拉回可行域"——是约束优化中最基本的范式，后续的近端方法、Frank-Wolfe 方法等都是这一范式的变体。
