### 第 ① 页：Projected Gradient Descent

**原文翻译**：

对于凸函数 $f: \mathbb{R}^n \to \mathbb{R}$ 和凸集 $\mathbb{X} \subseteq \mathbb{R}^n$，考虑约束最小化问题：

$$\min \quad f(x) \quad \text{s.t.} \quad x \in \mathbb{X}$$

投影梯度下降的工作方式如下：从某个初始点 $x^0$ 出发，依次生成序列 $y^1, x^1, y^2, x^2, y^3, x^3, \ldots$：

$$y^{k+1} = x^k - \gamma \, \nabla f(x^k)$$

$$x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$$

**精讲内容**：

这一页是整个约束优化部分的开篇，也是本课程后半段最重要的算法框架。让我们从"为什么需要它"开始理解。

回顾 Lecture 5-8 的梯度下降：$x^{k+1} = x^k - \gamma \nabla f(x^k)$。这个更新完全不关心 $x$ 是否在某个约束集合里——你从可行域内的某个点出发，沿负梯度方向走一步后，完全有可能跑到可行域外面。在无约束问题里这不是问题，因为整个 $\mathbb{R}^n$ 都是可行的。但一旦有约束 $x \in \mathbb{X}$，你就不能简单地执行普通 GD 了。

投影梯度下降的思路非常自然——"两步走"策略：
1. **梯度步**（Gradient Step）：先忽略约束，按普通 GD 走一步，得到中间点 $y^{k+1} = x^k - \gamma \nabla f(x^k)$。
2. **投影步**（Projection Step）：如果 $y^{k+1}$ 跑到了 $\mathbb{X}$ 外面，就把它"拉回"可行域。具体方法是找 $\mathbb{X}$ 中离 $y^{k+1}$ 最近的点，即 $x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$。

注意这里的记号：$y^{k+1}$ 是"裸的"梯度步结果（可能不可行），$x^{k+1}$ 是投影后的真正迭代点（保证可行）。算法产生的序列 $x^0, x^1, x^2, \ldots$ 始终在 $\mathbb{X}$ 内。

从 ML 的角度看，约束优化无处不在。例如：
- **岭回归**（Ridge Regression）：参数的 L2 范数 $\|w\|_2 \leq R$，对应欧几里得球约束。
- **LASSO**：参数的 L1 范数 $\|w\|_1 \leq R$，对应 L1-ball 约束。
- **概率模型**：softmax 输出层的参数需要在概率单纯形 $\{p: \sum p_i = 1, p_i \geq 0\}$ 上。
- **非负矩阵分解**（NMF）：$W \geq 0, H \geq 0$，对应非负约束（box约束的特例）。

这些约束在 Lecture 14-16 中会逐一展示如何计算投影。

关键问题是：投影操作不会"破坏"梯度下降的收敛性吗？直觉上，投影把你从梯度指向的方向"拽偏"了——你本来想往负梯度方向走，结果被强行拉回到可行域边界上的某个点。接下来几页将证明投影不仅不破坏收敛性，反而有很好的数学性质保证收敛速率。

最后强调一个前提条件：$f$ 是凸函数、$\mathbb{X}$ 是凸集。这两个凸性假设缺一不可——凸集保证投影是唯一的（非凸集上的"最近点"可能不唯一），凸函数保证局部最优就是全局最优。

**教授口吻讲解**：

同学们，到这一讲为止，我们一直在做"没有围墙的优化"——整个空间都是你的操场，想往哪跑就往哪跑。但现实世界不是这样的，对吧？你训练一个推荐系统，概率得加起来等于 1；你做 LASSO，参数的绝对值和得小于某个预算。今天我们要学的投影梯度下降，本质上就是给梯度下降"加了一面墙"——你跑出去了，我就把你推回来。两步走：先自由奔跑，再执行纪律。这个思想非常优雅，也非常实用。不过你可能会担心：每次被推回来不会浪费功夫吗？接下来几页的数学会告诉你——不会。

### 第 ② 页：投影梯度下降示意图与投影的重要性质引入

**原文翻译**：

> **[图示]** 一个凸集 $\mathbb{X}$（封闭区域）。从集合内部的点 $x^k$ 出发，沿负梯度方向走一步（红色线段），到达 $y^{k+1} = x^k - \gamma \nabla f(x^k)$（可能在 $\mathbb{X}$ 外部），然后投影回 $\mathbb{X}$ 的边界上，得到 $x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$（箭头指向边界上的点）。

**定义**：点 $y$ 到凸集 $\mathbb{X}$ 的**投影**为

$$\Pi_\mathbb{X}(y) = \arg\min_{x \in \mathbb{X}} \|x - y\|^2$$

**回忆**：如果 $g$ 是凸函数，

$$g(w) = g(z) + \nabla g(z)^\top (w - z) + o(\|w - z\|) \quad \text{(Taylor展开)}$$

**回忆**：$x^\top y = \|x\| \cdot \|y\| \cdot \cos\theta$，其中 $\theta$ 是 $x$ 和 $y$ 之间的夹角。因此，如果 $x^\top y \leq 0$，则 $x$ 和 $y$ 之间的夹角是钝角。

**精讲内容**：

这一页做了两件事：给出投影的精确数学定义，以及为证明投影的关键性质准备工具。

**投影的定义**。$\Pi_\mathbb{X}(y)$ 是凸集 $\mathbb{X}$ 中离 $y$ 最近的点。注意这是一个优化问题本身——$\arg\min_{x \in \mathbb{X}} \|x - y\|^2$——你要在 $\mathbb{X}$ 中找到使欧几里得距离平方最小的点。由于 $\|x - y\|^2$ 是关于 $x$ 的严格凸函数（它的 Hessian 是 $2I$，正定），而 $\mathbb{X}$ 是凸集，所以这个优化问题的解是**唯一**的。这就是为什么我们可以用 $\Pi_\mathbb{X}(y)$ 这个函数记号——它对每个 $y$ 都给出唯一的投影点。

**图示的几何直觉**。想象你站在一个围栏围起来的凸形草地里（$\mathbb{X}$），当前位置是 $x^k$。梯度 $\nabla f(x^k)$ 告诉你"上坡最快的方向"，你想往反方向走以下降。你走了一步到 $y^{k+1}$——但你可能走出了围栏。投影操作就是把你从围栏外的位置 $y^{k+1}$"推"到围栏边界上离你最近的那个点 $x^{k+1}$。

**为什么要回忆 Taylor 展开和内积几何**？这是在为第 ③-⑤ 页的投影钝角性质证明做准备。证明的核心思路是：对 $g(x) = \|x\|^2$ 应用 Taylor 展开，然后利用内积与角度的关系来得出几何结论。

Taylor 展开 $g(w) = g(z) + \nabla g(z)^\top(w-z) + o(\|w-z\|)$ 是凸函数的一阶近似，这里的 $o(\|w-z\|)$ 是高阶无穷小项。当 $g$ 是凸函数时，我们知道 $g(w) \geq g(z) + \nabla g(z)^\top(w-z)$（凸性的一阶特征化，Lecture 3-4）。但这里我们不是用不等式，而是用等式形式的 Taylor 展开——因为 $g(x) = \|x\|^2$ 是二次函数，它的 Taylor 展开可以精确计算。

内积与角度的关系 $x^\top y = \|x\|\|y\|\cos\theta$ 是线性代数的基本结论。关键推论是：$x^\top y \leq 0$ 等价于 $\cos\theta \leq 0$，即 $\theta \geq 90°$——两个向量之间的夹角是钝角。这个几何解释将在下一页给出投影的核心性质——投影向量和连接向量之间的夹角是钝角。

**精讲内容**（续）：

从 ML 实践的角度来说，投影的计算效率至关重要。如果 $\mathbb{X}$ 很复杂，$\arg\min_{x \in \mathbb{X}} \|x - y\|^2$ 本身就是一个难以求解的优化问题，那么每一步的投影都会成为瓶颈。幸运的是，对于 ML 中最常见的几种约束集——box 约束、欧几里得球、概率单纯形、L1-ball——投影都有闭式解或高效算法，这将在 Lecture 14-16 中逐一展示。

**教授口吻讲解**：

看这个图，很直观吧？你从 $x^k$ 出发，按梯度走了一步，走到了"围墙"外面——没关系，投影帮你找到围墙上离你最近的那个点。但我要强调一件事：这个投影本身就是一个优化问题！你要解 $\arg\min_{x \in \mathbb{X}} \|x-y\|^2$。所以投影梯度下降的实用性完全取决于"这个投影好不好算"。如果投影跟原问题一样难，那就没意义了。好消息是，对于 ML 里常见的约束集，投影要么有闭式解，要么有非常快的算法。下面两节课我们会一个一个看。现在先回到数学——我们需要证明投影有一些漂亮的性质，保证算法能收敛。

### 第 ③ 页：$\nabla\|x\|^2$ 的计算与投影的重要性质

**原文翻译**：

**注意**：令 $g(x) = \|x\|^2$，计算 $\nabla g(x)$。

$$g(x) = \sum_i (x_i)^2, \quad \frac{\partial g(x)}{\partial x_i} = 2x_i$$

$$\nabla g(x) = (2x_1, \ldots, 2x_n) = 2x$$

**投影的重要性质**：设 $\mathbb{X}$ 是凸集，$x \in \mathbb{X}$，$y \in \mathbb{R}^n$。则有：

$$(x - \Pi_\mathbb{X}(y))^\top (y - \Pi_\mathbb{X}(y)) \leq 0$$

> **[图示]** 凸集 $\mathbb{X}$（封闭区域）。点 $y$ 在集合外部，$\Pi_\mathbb{X}(y)$ 在集合边界上，$x$ 在集合内部。从 $\Pi_\mathbb{X}(y)$ 分别指向 $x$ 的向量 $(x - \Pi_\mathbb{X}(y))$ 和指向 $y$ 的向量 $(y - \Pi_\mathbb{X}(y))$ 之间的夹角是钝角（$\geq 90°$）。

**精讲内容**：

这一页分为两个部分：一个简单的梯度计算，和一个极其重要的投影性质。

**Part 1：$\nabla\|x\|^2 = 2x$**。这个计算虽然简单，但它是后面证明的关键构件。$g(x) = \|x\|^2 = \sum_i x_i^2$ 是一个各分量独立的二次函数，对每个 $x_i$ 求偏导得到 $2x_i$，组合起来得到梯度 $\nabla g(x) = 2x$。为什么要单独拿出来算？因为接下来的证明中，我们会对 $g(x) = \|x\|^2$ 使用 Taylor 展开，需要知道它的梯度。

**Part 2：投影的钝角性质（Obtuse Angle Property）**。这是本讲最核心的数学结论之一。

用文字描述这个性质：对于凸集 $\mathbb{X}$ 中的任意点 $x$，和空间中的任意点 $y$，从投影点 $\Pi_\mathbb{X}(y)$ 分别看向 $x$ 和 $y$——这两个方向之间的夹角一定是钝角（$\geq 90°$）。

几何直觉是什么？想象你站在一个凸形湖泊（$\mathbb{X}$）的岸边（$\Pi_\mathbb{X}(y)$），面向湖外某个目标 $y$。那么湖内的任何一个点 $x$，相对于你来说，都在你的"背后或者两侧"——不会出现在你和 $y$ 之间。这就是"钝角"的直觉含义：从投影点看，可行域内的点和被投影的点"方向相反"。

为什么这个性质如此重要？它是后面勾股定理型不等式（第 ⑥ 页）的基础，而勾股定理型不等式又是收敛分析（第 ⑦-⑧ 页和 Lecture 14）的核心工具。可以说，没有这个钝角性质，整个投影梯度下降的收敛理论就无从建立。

从数学上，$(x - \Pi_\mathbb{X}(y))^\top (y - \Pi_\mathbb{X}(y)) \leq 0$ 等价于说：向量 $\overrightarrow{\Pi_\mathbb{X}(y) \to x}$ 和向量 $\overrightarrow{\Pi_\mathbb{X}(y) \to y}$ 的内积 $\leq 0$。由内积与角度的关系，这意味着夹角 $\geq 90°$。

这个性质与凸集的"支撑超平面"概念密切相关（Lecture 3）。投影点 $\Pi_\mathbb{X}(y)$ 处有一个超平面，以 $(y - \Pi_\mathbb{X}(y))$ 为法向量，将整个凸集 $\mathbb{X}$ 与点 $y$ 分开。钝角性质正是在说：$\mathbb{X}$ 中所有点都在这个超平面的"$y$ 的对面"。

**教授口吻讲解**：

$\nabla\|x\|^2 = 2x$ 这个计算你们应该闭着眼睛都会做了。重点是下面那个钝角性质——这是投影理论的"宪法条款"，后面所有的收敛定理都依赖它。你看那个图：从投影点往 $x$ 看和往 $y$ 看，两个方向之间的夹角是钝角。直觉上很合理——凸集把"里面的点"和"外面被投影的点"分在了两边。接下来两页我们要严格证明它。证明方法非常优雅，用的是 Taylor 展开加上一个 $\lambda \to 0$ 的极限论证。

### 第 ④ 页：投影钝角性质的证明（上）

**原文翻译**：

**证明**：我们想要证明 $(x - \pi)^\top(y - \pi) \leq 0$，其中 $\pi = \Pi_\mathbb{X}(y)$。

> **[图示]** 凸集 $\mathbb{X}$，$y$ 在集合外部，$\pi = \Pi_\mathbb{X}(y)$ 在边界上，$x$ 在集合内部。

对任意 $\lambda \in [0, 1)$，因为 $\mathbb{X}$ 是凸集，$x \in \mathbb{X}$，$\pi \in \mathbb{X}$，所以

$$\lambda x + (1 - \lambda) \pi \in \mathbb{X}$$

因为 $\pi = \arg\min_{z \in \mathbb{X}} \|z - y\|^2$，而 $\lambda x + (1-\lambda)\pi \in \mathbb{X}$，所以：

$$\|y - \pi\|^2 \leq \|y - (\lambda x + (1 - \lambda)\pi)\|^2 = \|y - \pi + \lambda(\pi - x)\|^2 \quad (\ddagger)$$

对 $g(x) = \|x\|^2$ 应用 Taylor 定理，令 $w = y - \pi + \lambda(\pi - x)$，$z = y - \pi$：

$$\|y - \pi + \lambda(\pi - x)\|^2 = \|y - \pi\|^2 + 2\lambda(y - \pi)^\top(\pi - x) + o(\lambda \|\pi - x\|)$$

由 $(\ddagger)$：$\|y - \pi\|^2$ 出现在不等式两边。

**精讲内容**：

这一页是钝角性质证明的上半部分，展示了一个非常经典的凸优化证明技巧——**凸组合扰动法**。

**证明策略**。我们要证 $(x - \pi)^\top(y - \pi) \leq 0$，其中 $\pi = \Pi_\mathbb{X}(y)$。证明的关键思路是：$\pi$ 是 $\mathbb{X}$ 中离 $y$ 最近的点，所以 $\mathbb{X}$ 中任何其他点到 $y$ 的距离都不比 $\pi$ 到 $y$ 的距离短。特别地，我们取 $x$ 和 $\pi$ 的凸组合 $\lambda x + (1-\lambda)\pi$——这个点也在 $\mathbb{X}$ 内（凸性保证），所以它到 $y$ 的距离 $\geq$ $\pi$ 到 $y$ 的距离。然后让 $\lambda \to 0$，通过 Taylor 展开提取出内积 $(y-\pi)^\top(\pi - x)$ 的符号信息。

**逐步分解**：

**Step 1**：构造可行扰动。取 $\lambda \in [0,1)$，凸组合 $\lambda x + (1-\lambda)\pi$ 由凸性属于 $\mathbb{X}$。当 $\lambda = 0$ 时就是 $\pi$ 本身；当 $\lambda$ 略大于 0 时，是从 $\pi$ 朝 $x$ 方向迈出的一小步。

**Step 2**：利用最优性。$\pi$ 是 $\mathbb{X}$ 中到 $y$ 最近的点，所以：
$$\|y - \pi\|^2 \leq \|y - (\lambda x + (1-\lambda)\pi)\|^2$$
右边可以改写为 $\|y - \pi + \lambda(\pi - x)\|^2$——这就是在 $(y - \pi)$ 的基础上加了一个小扰动 $\lambda(\pi - x)$。

**Step 3**：Taylor 展开。对 $g(v) = \|v\|^2$ 在 $v = y - \pi$ 处展开，扰动量为 $\lambda(\pi - x)$：
$$g(v + \delta) = g(v) + \nabla g(v)^\top \delta + o(\|\delta\|)$$
其中 $\nabla g(v) = 2v = 2(y-\pi)$，$\delta = \lambda(\pi-x)$。代入得：
$$\|y - \pi + \lambda(\pi-x)\|^2 = \|y-\pi\|^2 + 2\lambda(y-\pi)^\top(\pi-x) + o(\lambda\|\pi-x\|)$$

注意，实际上 $g(x) = \|x\|^2$ 是精确的二次函数，所以高阶项应该是 $\lambda^2\|\pi-x\|^2$（而不仅仅是 $o$）。但作者用更一般的 $o(\lambda\|\pi-x\|)$ 来表述，这样证明对一般的凸函数也适用。

**Step 4**：合并不等式。将 Taylor 展开代入 $(\ddagger)$：
$$\|y-\pi\|^2 \leq \|y-\pi\|^2 + 2\lambda(y-\pi)^\top(\pi-x) + o(\lambda\|\pi-x\|)$$
两边消去 $\|y-\pi\|^2$，得到下一页的起始点。

这个证明技巧（凸组合 + 最优性条件 + Taylor 展开 + 取极限）在凸优化理论中极为常见。Lecture 3 中凸函数一阶特征化的证明就用了类似方法。

**教授口吻讲解**：

这个证明的技巧非常值得学会。核心思路是什么？$\pi$ 是最近点，所以它是"局部最优"的——你从 $\pi$ 往 $\mathbb{X}$ 内任何方向走一小步，距离 $y$ 都不会变近。我们就利用这个"一阶最优性"来提取信息。具体方法是：沿着 $\pi \to x$ 的方向走一个 $\lambda$ 小步（$\lambda$ 很小所以是"小扰动"），然后 Taylor 展开看一阶项的符号。这跟你们在微积分里用的"验证极值点"方法是一样的——在极小值点，函数值沿任何方向的一阶变化量 $\geq 0$。

### 第 ⑤ 页：投影钝角性质的证明（下）

**原文翻译**：

继续：

$$\|y - \pi\|^2 \leq \|y - \pi\|^2 + 2\lambda(y - \pi)^\top(\pi - x) + o(\lambda \|\pi - x\|) \quad \text{对任意 } \lambda \in [0, 1)$$

$$\Longrightarrow \quad 0 \leq 2\lambda(y - \pi)^\top(\pi - x) + o(\lambda \|\pi - x\|)$$

$$\Longrightarrow \quad 0 \leq 2(y - \pi)^\top(\pi - x) + \frac{o(\lambda \|\pi - x\|)}{\lambda}$$

$$\Longrightarrow \quad 0 \leq 2(y - \pi)^\top(\pi - x) + \frac{o(\lambda \|\pi - x\|)}{\lambda \cdot \|\pi - x\|} \cdot \|\pi - x\|$$

当 $\lambda \to 0$ 时：

$$\lim_{\lambda \to 0} \frac{o(\lambda \|\pi - x\|)}{\lambda \cdot \|\pi - x\|} = 0$$

$$\Longrightarrow \quad 0 \leq 2(y - \pi)^\top(\pi - x)$$

$$\Longrightarrow \quad (y - \pi)^\top(x - \pi) \leq 0$$

证毕！$\blacksquare$

**精讲内容**：

这一页完成了钝角性质的证明，展示了一个精细的极限论证过程。

从上一页的结论出发：

$$0 \leq 2\lambda(y - \pi)^\top(\pi - x) + o(\lambda \|\pi - x\|) \quad \forall \lambda \in [0,1)$$

**Step 5**：除以 $\lambda$。因为 $\lambda > 0$（我们取 $\lambda \in (0,1)$），可以两边除以 $\lambda$：
$$0 \leq 2(y - \pi)^\top(\pi - x) + \frac{o(\lambda \|\pi - x\|)}{\lambda}$$

**Step 6**：分析余项。关键是理解 $\frac{o(\lambda\|\pi-x\|)}{\lambda}$ 在 $\lambda \to 0$ 时的行为。$o(\lambda\|\pi-x\|)$ 的定义是：当 $\lambda\|\pi-x\| \to 0$ 时，$\frac{o(\lambda\|\pi-x\|)}{\lambda\|\pi-x\|} \to 0$。所以：
$$\frac{o(\lambda\|\pi-x\|)}{\lambda} = \frac{o(\lambda\|\pi-x\|)}{\lambda\|\pi-x\|} \cdot \|\pi-x\|$$

当 $\lambda \to 0$ 时，第一个因子趋向 0，第二个因子 $\|\pi-x\|$ 是常数，所以整个余项趋向 0。

**Step 7**：取极限。在不等式 $0 \leq 2(y-\pi)^\top(\pi-x) + (\text{趋向0的项})$ 中令 $\lambda \to 0$：
$$0 \leq 2(y-\pi)^\top(\pi-x)$$

**Step 8**：改写。$(y-\pi)^\top(\pi-x) \geq 0$ 等价于 $(y-\pi)^\top(x-\pi) \leq 0$，即 $(x-\pi)^\top(y-\pi) \leq 0$。这正是我们要证明的钝角性质。

**为什么要用 $\lambda \to 0$ 而不是直接取某个特定的 $\lambda$**？因为对于一般的凸函数，我们只知道 Taylor 展开的余项是 $o(\|w-z\|)$ 而不知道具体的二阶项。通过取 $\lambda \to 0$ 的极限，一阶项 $2\lambda(y-\pi)^\top(\pi-x)$ 被保留下来（除以 $\lambda$ 后变成常数），而高阶余项被消灭。这是从"无穷小扰动"中提取一阶信息的标准手法。

实际上，对于 $g(x) = \|x\|^2$ 这个特定的二次函数，高阶项恰好是 $\lambda^2\|\pi-x\|^2$，不需要取极限也能直接计算。但教授采用了更一般的 $o(\cdot)$ 方法，好处是这个证明框架可以推广到其他凸函数的最优性条件证明中。

回顾整个证明的逻辑链条：凸性 $\Rightarrow$ 凸组合可行 $\Rightarrow$ 最优性给出不等式 $\Rightarrow$ Taylor展开提取一阶项 $\Rightarrow$ 取极限消灭高阶项 $\Rightarrow$ 得到内积的符号 $\Rightarrow$ 钝角性质。

**教授口吻讲解**：

证明完成了！回顾一下我们做了什么：从"$\pi$ 是最近点"这个事实出发，通过一个巧妙的凸组合扰动和极限论证，得出了"钝角"这个几何结论。这种"扰动 + Taylor + 取极限"的三连击是优化理论的标准武器，你们以后写论文也会反复用到。现在我们手里有了一个强有力的工具——接下来用它推导出一个更直接有用的不等式。

### 第 ⑥ 页：投影的勾股定理型不等式

**原文翻译**：

以下是投影性质的一个直接推论：

设 $\mathbb{X}$ 为凸集，$x \in \mathbb{X}$，$y \in \mathbb{R}^n$。则：

$$\|x - y\|^2 \geq \|x - \Pi_\mathbb{X}(y)\|^2 + \|y - \Pi_\mathbb{X}(y)\|^2$$

> **[图示]** 三角形：顶点分别为 $y$（集合外）、$\Pi_\mathbb{X}(y)$（集合边界）、$x$（集合内）。凸集 $\mathbb{X}$ 包含 $\Pi_\mathbb{X}(y)$ 和 $x$。不等式类似勾股定理但取 $\geq$，因为 $\Pi_\mathbb{X}(y)$ 处的角为钝角。

**回忆**：$\|a + b\|^2 = \|a\|^2 + \|b\|^2 + 2a^\top b$。

**证明**：令 $\pi = \Pi_\mathbb{X}(y)$。

$$\|x - y\|^2 = \|x - \pi + \pi - y\|^2 = \|x - \pi\|^2 + \|\pi - y\|^2 + 2(x - \pi)^\top(\pi - y)$$

$$= \|x - \pi\|^2 + \|y - \pi\|^2 - 2(x - \pi)^\top(y - \pi)$$

由前一个结果，$(x - \pi)^\top(y - \pi) \leq 0$，所以 $-2(x - \pi)^\top(y - \pi) \geq 0$。

$$\|x - y\|^2 \geq \|x - \pi\|^2 + \|y - \pi\|^2$$

证毕！$\blacksquare$

**精讲内容**：

这一页将钝角性质转化为一个更直接有用的"距离不等式"——我称之为**投影的勾股定理型不等式**（Pythagorean Inequality）。

**勾股定理的类比**。回忆标准的勾股定理：在直角三角形中，$c^2 = a^2 + b^2$，其中 $c$ 是斜边。这里的三角形有三个顶点：$y$（集合外）、$\pi = \Pi_\mathbb{X}(y)$（边界上）、$x$（集合内）。在 $\pi$ 处的角是钝角（$\geq 90°$，由钝角性质保证），所以"对面"那条边（即 $x$ 到 $y$，对应"超斜边"）的长度平方 $\geq$ 另外两边长度平方之和。如果 $\pi$ 处恰好是 $90°$，就退化为标准的勾股定理（等号成立）。

**证明的关键步骤**：

1. 将 $x - y$ 分解为 $(x - \pi) + (\pi - y)$——从 $x$ 到 $y$ 绕了 $\pi$ 一圈。
2. 展开范数平方：$\|a + b\|^2 = \|a\|^2 + \|b\|^2 + 2a^\top b$。这里 $a = x - \pi$，$b = \pi - y$，交叉项是 $2(x-\pi)^\top(\pi-y) = -2(x-\pi)^\top(y-\pi)$。
3. 由钝角性质，$(x-\pi)^\top(y-\pi) \leq 0$，所以 $-2(x-\pi)^\top(y-\pi) \geq 0$。
4. 因此，$\|x-y\|^2 = \|x-\pi\|^2 + \|y-\pi\|^2 + (\text{非负项}) \geq \|x-\pi\|^2 + \|y-\pi\|^2$。

**这个不等式为什么对收敛分析至关重要**？在 Lecture 14 的收敛证明中，我们需要从 $\|y^{k+1} - x^*\|^2$（中间点到最优点的距离）过渡到 $\|x^{k+1} - x^*\|^2$（投影点到最优点的距离）。勾股定理型不等式告诉我们：

$$\|y^{k+1} - x^*\|^2 \geq \|x^{k+1} - x^*\|^2 + \|y^{k+1} - x^{k+1}\|^2$$

即投影不会增加到 $x^*$ 的距离（因为 $x^* \in \mathbb{X}$）。这正是投影"不破坏收敛性"的数学保证——投影步虽然改变了迭代点的位置，但它只会把你拉得离最优解更近，不会更远。

这个性质在计算几何和优化理论中被称为 **Firm Non-expansiveness**（稳固非扩张性），是投影算子最核心的性质之一。

**教授口吻讲解**：

这个不等式非常漂亮，也非常有用。它的直觉是：投影只会"帮忙"，不会"捣乱"。你从 $y^{k+1}$ 投影到 $x^{k+1}$，到可行域内任何一点的距离只会缩短或不变，绝不会增大。这就像你在地图上找最近的餐馆——走到最近的路口后，你到目的地的距离不可能比你原来在野外时更远。这个性质是我们能把无约束 GD 的收敛分析"平移"到约束情况的关键。

### 第 ⑦ 页：Convergence of Projected Gradient Descent with Bounded Gradients

**原文翻译**：

投影梯度下降为：

$$y^{k+1} = x^k - \gamma \, \nabla f(x^k), \quad x^{k+1} = \Pi_\mathbb{X}(y^{k+1})$$

让我们改进之前证明无投影梯度下降收敛时的预备分析。

令 $x^* = \arg\min_{x \in \mathbb{X}} f(x)$，我们想要上界 $f(x^k) - f(x^*)$。

由凸性的一阶特征化：

$$f(x^*) \geq f(x^k) + \nabla f(x^k)^\top (x^* - x^k)$$

$$\Longrightarrow \quad f(x^k) - f(x^*) \leq \nabla f(x^k)^\top (x^k - x^*)$$

所以只需要上界 $\nabla f(x^k)^\top (x^k - x^*)$ 即可。

$$\nabla f(x^k)^\top (x^k - x^*) = \frac{1}{\gamma} (x^k - y^{k+1})^\top (x^k - x^*)$$

**回忆**：$\|a - b\|^2 = \|a\|^2 + \|b\|^2 - 2a^\top b \implies 2a^\top b = \|a\|^2 + \|b\|^2 - \|a - b\|^2$。

**精讲内容**：

这一页开始将前面建立的投影数学性质应用于实际的收敛性分析。

**证明策略回顾**。在 Lecture 5-8 的无约束 GD 中，我们用类似的策略：先用凸性的一阶特征化将 $f(x^k) - f(x^*)$ 转化为一个内积的上界问题，再用代数恒等式展开这个内积。投影梯度下降的分析完全平行，只是多了一个投影步需要处理。

**凸性的一阶特征化**。这是 Lecture 3-4 的核心定理：对凸函数 $f$，$f(y) \geq f(x) + \nabla f(x)^\top(y-x)$ 对所有 $x, y$ 成立。用 $x = x^k$，$y = x^*$ 代入，得到：
$$f(x^*) \geq f(x^k) + \nabla f(x^k)^\top(x^* - x^k)$$
移项得 $f(x^k) - f(x^*) \leq \nabla f(x^k)^\top(x^k - x^*)$。

这个不等式的含义是：当前误差 $f(x^k) - f(x^*)$ 被梯度和"当前点到最优点方向"的内积所控制。如果梯度和 $x^k - x^*$ 的方向一致（$\nabla f$ 指向"远离 $x^*$"的方向），则内积大，误差上界松；如果梯度接近垂直于 $x^k - x^*$，则内积小，误差上界紧。

**替换梯度**。利用 $y^{k+1} = x^k - \gamma \nabla f(x^k)$，得 $\nabla f(x^k) = \frac{1}{\gamma}(x^k - y^{k+1})$。代入：
$$\nabla f(x^k)^\top(x^k - x^*) = \frac{1}{\gamma}(x^k - y^{k+1})^\top(x^k - x^*)$$

**极化恒等式**。$2a^\top b = \|a\|^2 + \|b\|^2 - \|a-b\|^2$ 是内积和范数之间的经典关系（也叫极化恒等式）。它让我们把"内积"转化为"距离"，从而可以利用望远镜求和（telescoping sum）来消去中间项。这个技巧在 Lecture 5-8 的无约束 GD 分析中已经见过了。

下一页将应用这个恒等式展开内积，并与投影的勾股定理型不等式结合，完成分析的关键步骤。

**教授口吻讲解**：

这里的证明结构跟我们之前分析无约束 GD 时几乎一样——凸性给你一个"误差 $\leq$ 内积"的起点，然后你用代数恒等式把内积拆成一些距离的组合。唯一的新元素是投影——我们需要在某个地方用上前面证明的投影性质。下一页你就会看到它在哪里发挥作用。

---

### 第 ⑧ 页：收敛分析的展开（续）

**原文翻译**：

因此，

$$\nabla f(x^k)^\top (x^k - x^*) = \frac{1}{2\gamma} \left( \|x^k - y^{k+1}\|^2 + \|x^k - x^*\|^2 - \|(x^k - y^{k+1}) - (x^k - x^*)\|^2 \right)$$

$$= \frac{1}{2\gamma} \left( \|x^k - y^{k+1}\|^2 + \|x^k - x^*\|^2 - \|y^{k+1} - x^*\|^2 \right)$$

**精讲内容**：

这一页应用极化恒等式完成了内积的展开。

**具体展开**。设 $a = x^k - y^{k+1}$，$b = x^k - x^*$。由极化恒等式：
$$2a^\top b = \|a\|^2 + \|b\|^2 - \|a - b\|^2$$

代入：
$$\frac{2}{\gamma}(x^k - y^{k+1})^\top(x^k - x^*) = \frac{1}{\gamma}\left(\|x^k - y^{k+1}\|^2 + \|x^k - x^*\|^2 - \|(x^k - y^{k+1}) - (x^k - x^*)\|^2\right)$$

简化第三项：$(x^k - y^{k+1}) - (x^k - x^*) = x^* - y^{k+1} = -(y^{k+1} - x^*)$，所以 $\|\cdot\|^2 = \|y^{k+1} - x^*\|^2$。

最终得到：
$$\nabla f(x^k)^\top(x^k - x^*) = \frac{1}{2\gamma}\left(\|x^k - y^{k+1}\|^2 + \|x^k - x^*\|^2 - \|y^{k+1} - x^*\|^2\right)$$

**各项的含义**：
- $\|x^k - y^{k+1}\|^2 = \gamma^2\|\nabla f(x^k)\|^2$：梯度步的"步长平方"，即每一步走了多远。
- $\|x^k - x^*\|^2$：当前迭代点到最优解的距离平方——这是"当前误差"的度量。
- $\|y^{k+1} - x^*\|^2$：梯度步后的中间点到最优解的距离平方。

注意到 $\|x^k - x^*\|^2 - \|y^{k+1} - x^*\|^2$ 形成了一个**伸缩（telescoping）结构**：如果我们对 $k = 0, 1, \ldots, K-1$ 求和，中间项会大量消去。但这里有个问题——伸缩消去需要相邻项匹配：第 $k$ 步的 $-\|y^{k+1} - x^*\|^2$ 需要和第 $k+1$ 步的 $+\|x^{k+1} - x^*\|^2$ 消去。可是 $y^{k+1} \neq x^{k+1}$（一个是梯度步后的点，一个是投影后的点）！

这就是投影的勾股定理型不等式发挥作用的地方。由第 ⑥ 页：
$$\|y^{k+1} - x^*\|^2 \geq \|x^{k+1} - x^*\|^2 + \|y^{k+1} - x^{k+1}\|^2 \geq \|x^{k+1} - x^*\|^2$$

因此 $-\|y^{k+1} - x^*\|^2 \leq -\|x^{k+1} - x^*\|^2$，这样就可以将 $y^{k+1}$ 替换为 $x^{k+1}$，恢复伸缩消去结构。这个替换在 Lecture 14 第 ③-④ 页中完成。

**与无约束 GD 的对比**。在无约束 GD 中，$y^{k+1} = x^{k+1}$（没有投影步），所以伸缩消去直接成立。投影梯度下降的额外代价是一个"距离损失" $\|y^{k+1} - x^{k+1}\|^2$（投影"移动"了多远），但由勾股不等式，这个损失只会让上界更松，不会破坏收敛性。

**教授口吻讲解**：

到这里为止，我们把内积展开成了三个距离的组合。关键观察是：$\|x^k - x^*\|^2$ 和 $\|y^{k+1} - x^*\|^2$ 几乎可以形成伸缩消去——但中间隔了一个投影步。怎么办？用第 ⑥ 页的勾股不等式！这就是"钝角性质 $\to$ 勾股不等式 $\to$ 伸缩消去 $\to$ 收敛定理"这条因果链的核心环节。Lecture 14 会完成最后的求和和步长优化。今天就到这里——你们已经掌握了投影梯度下降的全部数学基础设施，下节课就是"收割"成果的时候。

## 核心收获

1. **约束优化的基本范式**：投影梯度下降 = 梯度步 + 投影步，是处理凸约束优化最自然的方法。
2. **投影的两大数学性质**：钝角性质 $(x-\pi)^\top(y-\pi) \leq 0$ 和勾股定理型不等式 $\|x-y\|^2 \geq \|x-\pi\|^2 + \|y-\pi\|^2$，是收敛分析的基石。
3. **凸组合扰动法**：利用凸集的凸组合封闭性和最优性条件，通过 Taylor 展开和取极限来证明几何性质——这是凸优化理论中最常用的证明范式。
4. **投影不破坏收敛性**：勾股不等式保证投影步只会缩短到最优解的距离，从而保持无约束 GD 的收敛速率量级。
5. **从无约束到约束的迁移**：本讲的分析框架与 Lecture 5-8 的无约束 GD 分析完全平行，额外的数学"开销"仅仅是投影的两个性质——这展示了凸优化理论的优美统一性。
