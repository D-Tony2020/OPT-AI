### 第 1 页：常步长梯度下降的凸函数收敛

**原文翻译**：

- 我们要用常步长 $\gamma$ 的梯度下降法来最小化凸函数 $f: \mathbb{R}^n \to \mathbb{R}$。
- 我们生成点序列 $x^0, x^1, x^2, \ldots$，使用更新规则 $x^{k+1} = x^k - \gamma \nabla f(x^k)$，其中 $x^0$ 是我们选择的初始点。
- 使用常步长时，我们**无法**建立收敛到最小值的结论。我们能做到的最好结果是在最小值附近弹跳。即便如此，我们的收敛结果仍然是有意义的。

**精讲内容**：

本页开门见山地指出了常步长梯度下降的一个根本局限：**不能精确收敛到最优解**。

梯度下降的更新规则是 $x^{k+1} = x^k - \gamma \nabla f(x^k)$，其中 $\gamma > 0$ 是固定的步长。直觉上，梯度 $\nabla f(x^k)$ 指向函数增大最快的方向，负梯度方向则是函数减小最快的方向。沿负梯度走一步步长 $\gamma$，就是试图让函数值下降。

但问题在于：**步长 $\gamma$ 是固定的**。当我们离最优点 $x^*$ 很远时，$\gamma$ 可能正好合适；但当我们接近 $x^*$ 时，同样的 $\gamma$ 可能太大了——我们会"跨过" $x^*$ 跑到另一边，然后又被梯度拉回来，如此反复弹跳。

这个观察引出了本讲的核心问题：**既然不能精确收敛，我们能对"弹跳幅度"给出什么定量保证？** 答案是：虽然每一步的函数值可能上下波动，但**平均误差**会随迭代次数 $K$ 单调减小——而且我们可以精确量化减小的速率。

这与 ML 实践中的直觉是一致的。在训练神经网络时，我们通常用固定学习率（或分段固定的学习率），训练损失不会单调下降，而是在一个范围内波动。但"平均趋势"是下降的。本讲的理论正是为这种现象提供数学基础。

与 Lec 09 的连接：Lec 09 告诉我们凸函数的驻点是全局最优——但没说我们需要多少步才能接近驻点。本讲开始定量回答这个问题。

**教授口吻讲解**：

好，上一讲我们证明了一个很好的理论结果：凸函数上，找到梯度为零的点就找到了全局最优。但梯度下降能不能真的让梯度趋于零？用常步长的话——很遗憾，不能精确做到。最好的情况是在最优点附近"弹来弹去"。

你可能觉得这听起来很糟糕。但别急——弹跳的幅度是可以控制的，而且随着迭代次数增加，平均误差会越来越小。今天的目标就是精确量化这个"越来越小"是什么速度。

### 第 2 页：常步长梯度下降的弹跳示例

**原文翻译**：

- 【图示说明】函数 $f(x) = (x-1)^2$ 的曲线，最小值在 $x = 1$ 处。$f'(x) = 2(x-1)$。
- 从 $x^0 = 2$ 开始，使用常步长 $\gamma = 1$ 最小化 $f(x) = (x-1)^2$。
  - $f'(x^0) = f'(2) = 2$，$x^1 = x^0 - \gamma f'(x^0) = 2 - 1 \cdot 2 = 0$
  - $f'(x^1) = f'(0) = -2$，$x^2 = x^1 - \gamma f'(x^1) = 0 - 1 \cdot (-2) = 2$
  - $f'(x^2) = f'(2) = 2$，$x^3 = 0, x^4 = 2, x^5 = 0, \ldots$
- 所以我们会在最小值 $x^* = 1$ 附近弹跳。

**精讲内容**：

本页用一个极简的一维例子完美展示了常步长 GD 的弹跳现象。

函数 $f(x) = (x - 1)^2$ 是最简单的凸函数之一——一元二次函数，最小值点在 $x^* = 1$，最小值为 $f(x^*) = 0$。梯度为 $f'(x) = 2(x - 1)$。

从 $x^0 = 2$ 出发，步长 $\gamma = 1$：
- 在 $x^0 = 2$ 处：梯度 $f'(2) = 2 > 0$（函数在增大方向），更新 $x^1 = 2 - 1 \cdot 2 = 0$。我们从 $x^* = 1$ 的右边跳到了左边！
- 在 $x^1 = 0$ 处：梯度 $f'(0) = -2 < 0$（函数在减小方向——向右走），更新 $x^2 = 0 - 1 \cdot (-2) = 2$。又跳回了右边！
- 如此循环：$x^k$ 在 0 和 2 之间永久弹跳，永远不到达 $x^* = 1$。

为什么会弹跳？因为步长 $\gamma = 1$ 对这个函数来说太大了。在 $x = 2$ 处，离最优点距离为 1，但梯度大小为 2——走一步 $\gamma \cdot |f'| = 1 \cdot 2 = 2$，远超过实际需要的距离 1。结果就是过冲 (overshoot)。

**最优步长是什么？** 对 $f(x) = (x-1)^2$，$f'(x) = 2(x-1)$，Lipschitz 常数 $L = 2$（即 $|f''(x)| = 2$）。如果选择 $\gamma = 1/L = 0.5$，则 $x^1 = 2 - 0.5 \cdot 2 = 1 = x^*$——一步到位！但在实际问题中我们通常不知道 $L$ 的精确值，所以需要一般性的收敛理论。

这个例子虽然简单，但揭示了 GD 收敛分析中的核心张力：**步长太大导致弹跳（overshooting），步长太小导致进展缓慢（slow progress）**。本讲后续的分析就是在这两个极端之间找到最佳平衡。

与 Lec 11 的对比预告：在光滑凸函数（有 Lipschitz 连续梯度）的假设下，选择 $\gamma = 1/L$ 可以保证函数值**单调下降**，不会弹跳。这就是光滑性带来的额外好处——但本讲的分析不假设光滑性，所以弹跳是不可避免的。

**教授口吻讲解**：

看这个例子——步长 $\gamma = 1$，最优点在 $x^* = 1$，但你在 0 和 2 之间永远弹来弹去。像乒乓球一样，永远到不了目的地。

你可能会问：那减小步长不就行了？没错，如果 $\gamma = 0.5$，一步就到了。但关键是：在实际问题中你不知道 $\gamma$ 应该取多少才是"刚好的"。你能做的是选一个"差不多好"的步长，然后**忍受弹跳，但保证平均误差在下降**。今天的定理就是给你这个保证。

另外注意一点：虽然 $x^k$ 在 0 和 2 之间弹跳，但函数值 $f(0) = f(2) = 1$，而 $f(x^*) = 0$。误差恒为 1，不减小。所以光看"弹跳"不能给我们任何收敛结果——我们需要更精细的分析框架。

### 第 3 页：预备分析

**原文翻译**：

- 回顾：$f: \mathbb{R}^n \to \mathbb{R}$ 是凸的当且仅当 $f(y) \geq f(x) + \nabla^T f(x) \cdot (y - x)$ 对所有 $x, y \in \mathbb{R}^n$ 成立。
- 一个有用的恒等式：$\|x - y\|^2 = \|x\|^2 + \|y\|^2 - 2x^T y \qquad (\ddagger)$
- **证明**：$\|x - y\|^2 = (x - y)^T(x - y) = x^T x - 2x^T y + y^T y = \|x\|^2 - 2x^T y + \|y\|^2$
- 预备分析的设置：$x^{k+1} = x^k - \gamma \nabla f(x^k)$，记 $g^k = \nabla f(x^k)$，所以 $x^{k+1} = x^k - \gamma g^k$。
- 令 $x^* = \arg\min_{x \in \mathbb{R}^n} f(x)$——$f$ 的最小值点。
- 我们的目标是确保接近 $f$ 在 $x^*$ 处的值。

**精讲内容**：

本页为收敛分析准备了两个关键工具，然后定义了分析的符号体系。

**工具一：一阶刻画**。这是 Lec 09 证明的核心定理，现在作为工具被直接调用。在收敛分析中，一阶刻画的标准用法是：令 $x = x^k$（当前迭代点），$y = x^*$（最优点），得到：

$$f(x^*) \geq f(x^k) + \nabla^T f(x^k) \cdot (x^* - x^k)$$

变形为：$f(x^k) - f(x^*) \leq (g^k)^T (x^k - x^*)$

这个不等式把**函数值差距**（左边）转化为**梯度和位移的内积**（右边）。左边是我们要控制的目标（误差），右边是更容易分析的量。

**工具二：向量范数恒等式**。$\|x - y\|^2 = \|x\|^2 + \|y\|^2 - 2x^Ty$，这是欧几里得范数的基本性质，本质上是"完全平方展开"的向量版本。

这个恒等式为什么重要？因为它允许我们将**内积 $x^Ty$ 转化为范数差**：

$$x^T y = \frac{1}{2}(\|x\|^2 + \|y\|^2 - \|x - y\|^2)$$

在下一页中，我们需要上界 $(g^k)^T(x^k - x^*)$——一个内积。利用这个恒等式，我们可以把内积转化为 $\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2$ 这样的"距离差"，然后利用伸缩和技巧消去中间项。

**符号约定**：$g^k = \nabla f(x^k)$ 是第 $k$ 步的梯度，$x^*$ 是全局最优点。更新规则简写为 $x^{k+1} = x^k - \gamma g^k$。这套符号将贯穿 Lec 10-11 的所有分析。

这个"预备分析"框架的构建思路是：一阶刻画把误差归结为内积 → 恒等式把内积转化为范数差 → 范数差可以用伸缩和消去。三步组合起来，就能从局部的单步分析推导出全局的多步累积误差上界。

**教授口吻讲解**：

好，现在我们开始搭建分析框架。这个框架非常重要——它不是只用一次的。Lec 10 和 Lec 11 的收敛证明**都**建立在同一个预备分析之上，只是在最后一步使用不同的假设。

你需要记住两个基本工具：一阶刻画（把函数值差转化为内积）和向量范数恒等式（把内积转化为范数差）。这两个工具组合起来，就是凸优化收敛分析的标准套路。

我来给你们一个"鸟瞰图"：我们要上界 $f(x^k) - f(x^*)$。一阶刻画说它 $\leq (g^k)^T(x^k - x^*)$。范数恒等式说这个内积可以写成 $\|x^k - x^*\|^2$ 和 $\|x^{k+1} - x^*\|^2$ 的差。对 $k$ 求和，中间项消掉——这就是伸缩和。最后只剩下初始距离和一些梯度项。

### 第 4 页：上界推导

**原文翻译**：

- 我们要上界 $f(x^k) - f(x^*)$。
- 从一阶刻画出发：$f(x^*) \geq f(x^k) + \nabla^T f(x^k) \cdot (x^* - x^k)$
- 推出：$f(x^k) - f(x^*) \leq \nabla^T f(x^k) \cdot (x^k - x^*) = (g^k)^T(x^k - x^*) \quad (\star)$
- 因此，要上界 $f(x^k) - f(x^*)$，只需上界 $(g^k)^T(x^k - x^*)$。
- 由梯度下降更新：$x^{k+1} = x^k - \gamma g^k \implies g^k = \frac{1}{\gamma}(x^k - x^{k+1})$
- 接下来上界 $(g^k)^T(x^k - x^*)$。

**精讲内容**：

本页是预备分析的第一步：利用一阶刻画将函数值误差转化为内积。

核心不等式 $(\star)$ 的推导过程：

$$f(x^*) \geq f(x^k) + (g^k)^T(x^* - x^k)$$

这是一阶刻画对 $x = x^k, y = x^*$ 的直接应用。变形得：

$$f(x^k) - f(x^*) \leq (g^k)^T(x^k - x^*)$$

右边 $(g^k)^T(x^k - x^*)$ 的直觉是什么？$g^k$ 是梯度方向，$x^k - x^*$ 是从最优点到当前点的位移。它们的内积衡量了"梯度在位移方向上的分量"——如果梯度指向远离最优点的方向（内积大），说明误差大；如果梯度几乎与位移正交（内积小），说明我们已经接近最优了。

接下来的关键代换是：由 $x^{k+1} = x^k - \gamma g^k$ 推出 $g^k = \frac{1}{\gamma}(x^k - x^{k+1})$。这个代换把**梯度**（函数信息）替换为**迭代点的差**（几何信息）。为什么要这么做？因为下一步我们要用向量范数恒等式——它处理的是范数和内积，用迭代点的差来表示更方便。

这一步的意义在于**问题的转化**：从"分析函数值差距"转化为"分析几何距离"。函数值是"高度信息"，不容易直接分析。但几何距离——$\|x^k - x^*\|$ 表示当前点离最优点有多远——更容易通过迭代关系来追踪。

这个转化思路在优化理论中极其常见：**把优化问题（关于函数值）转化为几何问题（关于迭代点的距离）**，然后利用迭代关系的递推结构来分析。

**教授口吻讲解**：

第一步：一阶刻画。$f(x^k) - f(x^*) \leq (g^k)^T(x^k - x^*)$。这个不等式你要背下来——它是所有凸 GD 收敛分析的起手式。

然后我们做了一个代换：$g^k = (x^k - x^{k+1})/\gamma$。这个代换看起来平淡无奇，但它把梯度（一个抽象的函数导数）变成了两个相邻迭代点的差（一个具体的几何量）。为什么？因为接下来我们要用范数恒等式——它需要的输入是向量，不是梯度。

记住这个套路：一阶刻画 → 代换梯度 → 范数恒等式 → 伸缩和。每一步都很自然，但串在一起就能给出强力的收敛结果。

### 第 5 页：展开内积并利用伸缩和

**原文翻译**：

$$(g^k)^T(x^k - x^*) = \frac{1}{\gamma}(x^k - x^{k+1})^T(x^k - x^*)$$

由 $(\ddagger)$ 式：

$$= \frac{1}{2\gamma}\left(\|x^k - x^{k+1}\|^2 + \|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2\right)$$

$$= \frac{\gamma}{2}\|g^k\|^2 + \frac{1}{2\gamma}\left(\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2\right)$$

对 $k = 0, 1, \ldots, K-1$ 求和：

$$\sum_{k=0}^{K-1}(g^k)^T(x^k - x^*) = \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\left(\|x^0 - x^*\|^2 - \|x^K - x^*\|^2\right)$$

（利用伸缩和）

$$\leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$$

结合 $(\star)$ 式：

$$\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$$

预备分析到此结束。

**精讲内容**：

本页是预备分析的核心步骤，包含三个精巧的数学操作。

**第一步：内积转范数差**。利用恒等式 $(\ddagger)$，将内积 $(x^k - x^{k+1})^T(x^k - x^*)$ 转化为三个范数的线性组合：

$$a^T b = \frac{1}{2}(\|a\|^2 + \|b\|^2 - \|a - b\|^2)$$

其中 $a = x^k - x^{k+1}$，$b = x^k - x^*$，$a - b = x^{k+1} - x^*$（注意符号）。

代入后得到的三项各有含义：
- $\|x^k - x^{k+1}\|^2 = \gamma^2 \|g^k\|^2$：步长乘以梯度大小的平方——衡量"走了多远"
- $\|x^k - x^*\|^2$：当前点到最优点的距离平方——"还差多远"
- $-\|x^{k+1} - x^*\|^2$：下一步到最优点的距离平方的负值——"走完一步后还差多远"

**第二步：伸缩和 (Telescoping Sum)**。对 $k = 0, 1, \ldots, K-1$ 求和时，$\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2$ 会发生大面积消除：

$$\sum_{k=0}^{K-1}(\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2)$$
$$= (\|x^0 - x^*\|^2 - \|x^1 - x^*\|^2) + (\|x^1 - x^*\|^2 - \|x^2 - x^*\|^2) + \cdots$$
$$= \|x^0 - x^*\|^2 - \|x^K - x^*\|^2$$

中间所有项都消掉了！只剩下首项和末项。由于 $\|x^K - x^*\|^2 \geq 0$，我们进一步放缩：

$$\|x^0 - x^*\|^2 - \|x^K - x^*\|^2 \leq \|x^0 - x^*\|^2$$

**第三步：组合得到预备分析最终结果**。

$$\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$$

这个不等式的右边由两项组成：
- **梯度项** $\frac{\gamma}{2}\sum\|g^k\|^2$：与步长 $\gamma$ 成正比——步长越大，梯度项的贡献越大（过冲效应）
- **初始距离项** $\frac{1}{2\gamma}\|x^0 - x^*\|^2$：与步长 $\gamma$ 成反比——步长越小，初始距离的贡献越大（进展缓慢效应）

两项之间存在 **trade-off**：增大 $\gamma$ 会增大梯度项但减小距离项，减小 $\gamma$ 反之。最优步长就是让两项平衡的那个 $\gamma$——这正是下一个板块要做的事。

伸缩和技巧是离散数学和数值分析中最常用的技巧之一。它的本质是：当你有一个"前一步 - 后一步"的差分形式时，求和后中间项全部抵消，只留下首尾。这与微积分中的 Newton-Leibniz 公式 $\int_a^b f'(x)dx = f(b) - f(a)$ 是离散版本的对应。

**教授口吻讲解**：

这一页是整个分析的关键一步。三个操作串起来：内积变范数差 → 求和 → 伸缩和。最终你得到一个非常漂亮的上界，只依赖于两个东西：梯度大小和初始距离。

伸缩和为什么如此有效？因为 $\|x^k - x^*\|^2 - \|x^{k+1} - x^*\|^2$ 描述的是"每一步距最优点的距离变化"。累加起来，所有中间变化抵消，只留下"一开始离多远"减去"最后离多远"。这就像你坐电梯从 10 楼到 1 楼——中间经过 9、8、7...楼，但你关心的只是起点和终点。

注意右边两项的 trade-off：$\gamma$ 大，梯度项大（步子太大，弹跳严重）；$\gamma$ 小，距离项大（步子太小，走不动）。下一个定理就是找到最佳平衡点。

### 第 6 页：预备分析的总结与解读

**原文翻译**：

- 预备分析的步骤总结：
  1. 从一阶刻画出发。
  2. 利用 $\|x - y\|^2 = \|x\|^2 + \|y\|^2 - 2x^T y$ 将内积转化为范数差。
  3. 利用伸缩和。
- 最终不等式的解读：
  - $\sum(f(x^k) - f(x^*))$ 是前 $K$ 步的累积"误差"。
  - 如果 $\gamma$ 很小，上界由 $(1/(2\gamma))\|x^0 - x^*\|^2$ 主导——步长太小，我们会被困在初始点附近，误差取决于初始点离 $x^*$ 的距离。
  - 如果 $\gamma$ 很大，上界由 $(\gamma/2)\sum\|g^k\|^2$ 主导——步长太大，每当梯度非零时我们就过冲。

**精讲内容**：

本页是对前几页技术推导的"元反思"——它不引入新的数学内容，而是帮助我们理解预备分析的结构和直觉。

**预备分析三步法的回顾**：

| 步骤 | 数学操作 | 直觉含义 |
|------|---------|---------|
| 1. 一阶刻画 | $f(x^k) - f(x^*) \leq (g^k)^T(x^k - x^*)$ | 函数值误差 $\leq$ 梯度与位移的内积 |
| 2. 范数恒等式 | 内积 $\to$ 范数差 | 把"梯度信息"转化为"距离信息" |
| 3. 伸缩和 | 累加消中间项 | 从单步分析推出全局累积上界 |

**两种极端情况的直觉解读**：

**极端一：$\gamma$ 很小**。当步长趋近于零时：
- 梯度项 $\frac{\gamma}{2}\sum\|g^k\|^2 \to 0$（走的步子极小，不会过冲）
- 距离项 $\frac{1}{2\gamma}\|x^0 - x^*\|^2 \to \infty$（走不动，被初始距离困住）

这对应"蜗牛模式"：步长太小，虽然不弹跳，但进展极其缓慢。如果你的初始点离最优点很远，你需要天文数字的迭代才能接近。

**极端二：$\gamma$ 很大**。当步长很大时：
- 梯度项 $\frac{\gamma}{2}\sum\|g^k\|^2$ 很大（每步走很远，严重弹跳）
- 距离项 $\frac{1}{2\gamma}\|x^0 - x^*\|^2$ 很小

这对应"醉汉模式"：步长太大，虽然每一步"走得远"，但方向不断反转，净前进效果很差。

最佳步长应该让两项**平衡**——即对 $\gamma$ 求导令其为零。这正是下一页要做的。

这种"两项 trade-off"的结构在优化理论中非常普遍。它本质上是 **bias-variance trade-off** 的优化版本：步长小 → 低"方差"（不弹跳）但高"偏差"（进展慢）；步长大 → 低"偏差"（进展快）但高"方差"（弹跳大）。

**教授口吻讲解**：

我来用一个比喻帮你理解这两种极端。

想象你在雾中下山。步长小 = 你每次只挪动一小步。你很安全，不会掉下悬崖。但你可能要走一百年才到山脚。步长大 = 你大跨步往前冲。你很快"接近"山脚，但可能冲过头，然后又要折返。

最优策略是什么？根据你离山脚的距离来调整步长。但我们现在用的是**固定**步长——所以最好的固定步长是一个折中：不太大也不太小。下一个定理告诉你这个最优折中是多少。

### 第 7 页：有界梯度下的梯度下降收敛定理

**原文翻译**：

- 假设 $f$ 是凸的，且 $\|\nabla f(x)\| \leq B$ 对所有 $x \in \mathbb{R}^n$ 成立。
- **定理**：设 $f$ 是凸函数且有全局最小值 $x^*$。假设 $\|\nabla f(x)\| \leq B$ 对所有 $x \in \mathbb{R}^n$ 成立，且 GD 从 $x^0$ 出发，$\|x^0 - x^*\| = R$。如果选择步长为

$$\gamma = \frac{R}{B\sqrt{K}}$$

则 GD 的 $K$ 步迭代满足：

$$\frac{1}{K}\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{RB}{\sqrt{K}}$$

所以 GD 的平均"误差"以 $1/\sqrt{K}$ 的速率递减。

**精讲内容**：

本页给出了梯度下降在最一般凸函数上的收敛速率定理。这是本讲的核心理论结果。

**假设分析**：

1. **$f$ 是凸函数**：这是基本假设，保证一阶刻画等工具可用。
2. **$\|\nabla f(x)\| \leq B$（梯度有界）**：对所有 $x$，梯度的模不超过常数 $B$。这是一个相当弱的假设——它意味着函数的"陡度"有上限。满足这个条件的典型例子包括定义在有界集上的光滑凸函数、Lipschitz 连续的凸函数等。注意：这个假设**不**要求梯度是 Lipschitz 连续的（即不要求光滑性），所以它适用于更广泛的函数类。
3. **$\|x^0 - x^*\| = R$（初始距离有界）**：初始点到最优点的距离为 $R$。这是一个合理的假设——如果初始点离最优点无限远，没有任何有限步算法能收敛。

**最优步长**：$\gamma = R/(B\sqrt{K})$。注意步长**依赖于总迭代次数 $K$**——你需要提前决定要跑多少步。$K$ 越大，步长 $\gamma$ 越小——这是合理的，因为步数越多，你需要越精细的步长来避免在最优点附近弹跳。

步长还依赖于 $R$ 和 $B$：
- $R$ 大（初始距离远）→ 步长大（需要走快一点）
- $B$ 大（梯度可以很大）→ 步长小（梯度大意味着更容易过冲，需要保守）

**收敛速率解读**：

$$\frac{1}{K}\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{RB}{\sqrt{K}}$$

左边是**平均误差**——$K$ 步迭代中函数值偏离最优值的平均量。右边以 $O(1/\sqrt{K})$ 衰减。这意味着：
- 要让平均误差减半，迭代次数需要变为 4 倍（$\sqrt{4} = 2$）
- 要让平均误差减小 10 倍，迭代次数需要变为 100 倍

这是一个比较慢的收敛速率。在 Lec 11 中，我们将看到光滑凸函数可以达到 $O(1/K)$ 速率——每增加一倍迭代次数，误差减半，快得多。

注意定理给出的是**平均误差**的上界，不是**最后一步误差**的上界。但由于 $\min_k (f(x^k) - f(x^*)) \leq \frac{1}{K}\sum_k (f(x^k) - f(x^*))$，平均误差的上界也是"最佳迭代"误差的上界。

**教授口吻讲解**：

这是本讲最重要的定理。让我帮你拆解几个关键点。

第一，步长 $\gamma = R/(B\sqrt{K})$ 需要你**提前知道 $K$**——你要决定跑多少步，然后据此设定步长。这在实际中有时不太方便，但理论上这是最优的选择。

第二，$1/\sqrt{K}$ 的收敛速率意味着**精度每提高一个数量级，计算量增加两个数量级**。想要误差从 0.1 降到 0.01？步数要从 $K$ 变成 $100K$。这在大规模 ML 中是昂贵的。所以下一讲我们会问：如果函数有更好的结构（光滑性），能不能做得更快？答案是可以——$O(1/K)$，快了一个量级。

第三，$RB$ 是"问题难度"的度量。$R$ 是你一开始离目标多远，$B$ 是函数有多"陡"。问题越难（$RB$ 越大），收敛越慢。这也很符合直觉。

### 第 8 页：收敛定理的证明

**原文翻译**：

**证明**：由预备分析：

$$\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{\gamma}{2}\sum_{k=0}^{K-1}\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2 \leq \frac{\gamma}{2}KB^2 + \frac{1}{2\gamma}R^2$$

累积误差的上界为 $(\gamma/2)KB^2 + (1/(2\gamma))R^2$。

选择 $\gamma$ 使上界最小：

$$H(\gamma) = \frac{1}{2}(KB^2\gamma + R^2/\gamma), \quad H'(\gamma) = \frac{1}{2}(KB^2 - R^2/\gamma^2) = 0$$

$$\implies \gamma^2 = R^2/(KB^2) \implies \gamma = R/(B\sqrt{K})$$

代入此步长：

$$\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{1}{2}\left(\frac{R}{B\sqrt{K}} \cdot K \cdot B^2 + \frac{B\sqrt{K}}{R} \cdot R^2\right) = RB\sqrt{K}$$

因此：

$$\frac{1}{K}\sum_{k=0}^{K-1}(f(x^k) - f(x^*)) \leq \frac{RB}{\sqrt{K}} \quad \text{证毕！}$$

**精讲内容**：

本页完成收敛定理的完整证明。证明分为三步：应用预备分析、最优化步长选择、代入计算。

**第一步：应用预备分析并代入假设**。

预备分析给了我们：$\sum(f(x^k) - f(x^*)) \leq \frac{\gamma}{2}\sum\|g^k\|^2 + \frac{1}{2\gamma}\|x^0 - x^*\|^2$

利用两个假设进行放缩：
- 梯度有界：$\|g^k\| = \|\nabla f(x^k)\| \leq B$，所以 $\sum_{k=0}^{K-1}\|g^k\|^2 \leq KB^2$
- 初始距离：$\|x^0 - x^*\| = R$，所以 $\|x^0 - x^*\|^2 = R^2$

代入得：$\sum(f(x^k) - f(x^*)) \leq \frac{\gamma}{2}KB^2 + \frac{1}{2\gamma}R^2$

**第二步：最优化步长**。

右边是关于 $\gamma$ 的函数 $H(\gamma) = \frac{1}{2}(KB^2 \gamma + R^2/\gamma)$。这是一个"线性项 + 倒数项"的形式——令导数为零求极值：

$$H'(\gamma) = \frac{1}{2}(KB^2 - R^2/\gamma^2) = 0$$
$$\gamma^2 = R^2/(KB^2)$$
$$\gamma = R/(B\sqrt{K})$$

（可以验证 $H''(\gamma) = R^2/\gamma^3 > 0$，所以这是最小值。）

这个优化过程展示了一个重要技巧：**通过最优化步长来最小化收敛上界**。不是任意选一个步长然后分析——而是先建立关于 $\gamma$ 的上界，然后找最好的 $\gamma$。这种方法论在优化理论中广泛使用。

**第三步：代入计算**。

将 $\gamma = R/(B\sqrt{K})$ 代入 $H(\gamma)$：

$$H\left(\frac{R}{B\sqrt{K}}\right) = \frac{1}{2}\left(\frac{R}{B\sqrt{K}} \cdot KB^2 + \frac{B\sqrt{K}}{R} \cdot R^2\right) = \frac{1}{2}(RB\sqrt{K} + RB\sqrt{K}) = RB\sqrt{K}$$

注意两项**恰好相等**——这不是巧合。当你最优化"$a\gamma + b/\gamma$"的形式时，最优点总是让两项相等（AM-GM 不等式的等号条件）。

除以 $K$ 得到平均误差上界 $RB/\sqrt{K}$。

**复杂度总结**：

| 精度 $\varepsilon$ | 所需迭代 $K$ | 增长关系 |
|---|---|---|
| 0.1 | $(RB/0.1)^2 = 100(RB)^2$ | — |
| 0.01 | $(RB/0.01)^2 = 10000(RB)^2$ | $100 \times$ |
| 0.001 | $(RB/0.001)^2 = 10^6(RB)^2$ | $100 \times$ |

每提高一个数量级的精度，计算量增加 100 倍——这是 $O(1/\varepsilon^2)$ 的复杂度。

**教授口吻讲解**：

证明非常干净——三步完成。第一步是代入假设到预备分析的结果中。第二步是对步长求导取最优。第三步是代入计算。

我想强调一个优美的结构：最优步长让上界中的两项**精确相等**。$\frac{\gamma}{2}KB^2 = \frac{1}{2\gamma}R^2 = \frac{1}{2}RB\sqrt{K}$。这是 AM-GM 不等式取等号的条件——$a\gamma = b/\gamma$ 当且仅当 $\gamma = \sqrt{b/a}$。

实际使用中的一个问题是：步长 $\gamma = R/(B\sqrt{K})$ 需要你知道 $R$（初始距离）和 $B$（梯度上界）。$R$ 通常可以估计（你知道你的初始点在哪里），但 $B$ 有时不容易估计。在实践中，人们常常用自适应步长方法（如 Adam）来绕过这个问题——但那是另一个话题了。

下一讲，我们会看到如果加上"光滑性"假设，收敛速率可以从 $O(1/\sqrt{K})$ 提升到 $O(1/K)$——这是一个数量级的改进。光滑性本质上让函数"更可预测"，GD 的每一步更有效率。

## 核心收获

**收获一：预备分析框架是凸 GD 收敛分析的通用模板**

"一阶刻画 → 范数恒等式 → 伸缩和"这个三步分析框架不仅适用于本讲的有界梯度情形，也适用于 Lec 11 的光滑凸函数情形。两个定理使用完全相同的预备分析，只是在最后一步利用不同的假设（有界梯度 vs 光滑性）来得到不同的收敛速率。掌握这个框架等于掌握了一大类收敛证明的核心结构。

**收获二：步长选择体现了 bias-variance trade-off**

上界 $\frac{\gamma}{2}KB^2 + \frac{1}{2\gamma}R^2$ 由两项组成——大步长增大梯度项（过冲），小步长增大距离项（进展慢）。最优步长 $\gamma = R/(B\sqrt{K})$ 让两项精确平衡。这种"两项 trade-off + 最优平衡"的模式在优化理论中极其常见，也是理解学习率调度的理论基础。

**收获三：$O(1/\sqrt{K})$ 是最一般凸函数的"基准线"速率**

在没有额外结构假设（光滑性、强凸性）的情况下，$O(1/\sqrt{K})$ 是凸 GD 能达到的最好速率。这意味着要精度 $\varepsilon$，需要 $O(1/\varepsilon^2)$ 步——对高精度需求来说非常昂贵。这激发了我们追问：能否利用函数的额外结构加速收敛？Lec 11 将展示光滑性如何将速率提升到 $O(1/K)$（$O(1/\varepsilon)$ 复杂度），实现数量级的改进。
